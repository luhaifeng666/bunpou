动词 ​

动词是描述动作、状态、变化、存在的一类词。

日语中，动词的词尾全部都是「う」段假名：う、く、ぐ、す、つ、ぬ、ぶ、む、る。动词均在词尾上进行变形。

🔖 动词按照性质可以分为：

  1. 持续动词：在一段时间内能持续动作。这类动词有明显的“开始→进行→结束”的持续过程。如「読むよむ、書くかく、話すはなす、降るふる」等

読んよんでいる。

正在读书。

  1. 瞬间动词：动作行为瞬间结束，不能持续进行。如「始まるはじまる、終わるおわる、立つたつ、座るすわる、卒業そつぎょうする、消えるきえる、死ぬしぬ」等。

橋はしが落ちおちている。

桥梁塌陷了。

  1. 状态动词：

    • 表示事物性质状态的动词。如「痩せるやせる、太るふとる、似るにる、優れるすぐれる、聳えるそびえる」等。

    • 表示恒常状态的动词。如「表示存在的ある、いる；(能动态)表示能力的できる、行けるいける、読めるよめる」等。这类动词没有ている的形式。

动词的分类 ​

日语中，动词主要分为三类，即一类动词（又称五段动词）、二类动词（又称一段动词） 以及 三类动词（又称か变【来る】、さ变【する】动词）。具体识别方式如下：

📚 不同类别的动词区分方式

  1. 一类动词: 词尾在「う」段：う、く、ぐ、す、つ、ぬ、ぶ、む、る;
  2. 二类动词:
    • 以「る」结尾；
    • 「る」前的假名是 い段 或 え段;
  3. 三类动词:

    • 来くる；

    • する；

    • 动作性名词 + する，例如：勉強する、買い物する。

关于动词的区分，有篇帖子写的不错，有兴趣的小伙伴可以戳这里。

⚠️ 注意

有几个看上去是二类动词，实际上却是 一类动词 的词需要特别注意：

  • 知しる；知道

  • 要いる；需要

  • 走はしる；跑

  • 帰かえる；回家

  • 入はいる；进入

  • 切きる；切

  • 喋しゃべる；聊天

速记口诀： 走、知、要、归、入、切、喋。

动词的体 ​

体是通过词形的变化表示运动（动作、变化、状态）处于何种状态的语法范畴。日语的体由完整体和持续体构成，完整体表示完整的动作、变化、状态，持续体则表示动作、状态的持续以及动作的结果或变化结果的持续。

持续体的表达方式： Ｖて +「いる／いない、いた／いなかった、います／いません、いました／いませんでした」。

动词的形态汇总 ​

1. 动词的非过去式 ​

  • 意义：表示将来的动作或者未来的状态。

🔖 说明

常与表示将来时点的时间名词呼应使用。如果对话语境比较明显地提示出“将来”这一信息时，时间名词可省略。

📚 例句

  1. 私わたし、明日あしたの授業じゅぎょうのあと、ここで宿題しゅくだいをします。

    我明天的课程结束后，在这里写作业。

  2. 来週らいしゅう、クラスで発表はっぴょうします。

    下周，在班上发表。

  3. 明日あした、宿題しゅくだいを提出ていしゅつします。

    明天交作业。

  4. 高橋たかはしさんは行いきますが、鈴木すずきさんは行いきません。

    高桥会去，但是，铃木不会去。

2. 动词第一连用形的常见用法 ​

动词的第一连用形，即将动词变成ます形后，去掉ます。常用用法主要分为以下几种：

📚 1. 表示动作的并列或者先后顺序

接续方式为 动词第一连用形 + 动词。这种通常只用于书面语。时态的变化体现在最后一个动词上。

  1. 本体ほんたいに単4電池たんよんでんちを2本ほん入いれ、「電源でんげん」キーを押おしてください。

    请在本体内放入2节7号电池，按下“电源”键。

  2. 100万円まんえんを箱はこに入いれ、鍵かぎをかけ、保管ほかんした。

    将100万日元放入箱中，上锁保管。

  3. ドアが開あき、男おとこが現あらわれた。

    门开了，一个男人进来了。

3. 动词第二连用形的常见用法 ​

动词的第二连用形，也叫做动词的て形， 其变化规律与动词的过去式(动词的た形)一致。常用用法主要分为以下几种：

📚 1. 表示动作的 相继发生 或者 句子的并列

多用于口语表达，相较于第一连用形表示并列的用法，此种用法较为随意自然。

  1. 家うちへ帰かえって、宿題しゅくだいをします。

    回家后开始写作业。(强调相继发生)

  2. お爺じいさんは[山.やま]へ行いって、お婆ばあさんは川かわへ行いきました。

    爷爷去了山那边，奶奶去了河边儿。(强调并列)

📚 2. 表示动作进行的 附带状况

  1. 眼鏡めがねをかけて本ほんを読よみます

    戴着眼镜读书。

  2. 手てを上あげて道路どうろを渡わたった。

    举着手过了马路。

📚 3. 表示 原因理由

風邪かぜを引ひいて学校がっこうを休やすみました。

我感冒了，所以没去上学。

📚 4. 表示 手段

バスに乗のって会社かいしゃに行いきます。

我坐公交上班儿。

4. 动词的连体形(简体)＜连体修饰语＞ ​

  • 意义：充当连体修饰语，对名词起修饰限定的作用。连体修饰语从句中的主语不能用「は」表示，而要用「が」或者「の」；
  • 接续：动词简体 + 名词。

📚 例句

  1. あそこは、李りさんがいつも行いく喫茶店きっさてんです。

    那儿是小李常去的咖啡店。

  2. 学校がっこうに行いかない日にちはうちで勉強べんきょうします。

    我放假的时候在家学习。

  3. 昨日きのう来くなかった人ひとは誰だれですか。

    昨天谁没来？

几个小技巧 ​

虽然动词形态上的变化比较多，但也算得上有迹可循。我整理了一些自己在记忆时候的技巧，希望对你有帮助。

  1. 简体否定词尾形式为ない，与之对应的过去简体否定词尾形式为なかった；敬体否定词尾形式为ません，与之相对应的敬体否定词尾形式为ませんでした；

  2. 不区分时态，一二类动词的变化都体现在词尾上：

    • 1类动词，在敬体的情况下，词尾由う段变为い段；在简体的情况下，词尾由う段变为あ段。

    • 2类动词，都是将词尾的る去掉后再进行变化。

  3. 3类动词主要分为くる与する的变化，词尾的る最终都会变成其他词缀，主要的区别在く与す上：

    • 不区分时态，く的变化规律为：简体时く变为こ，敬体时く变为き；

    • 不区分时态，す变为し。

如果上述的描述有不妥之处，烦请告知~

如果你有更好的记忆方式，也欢迎你分享出来，向你学习~

动词的能动态 ​

动词的能动态用来表示具有某种能力，或者动作、行为发生的可能性。 通常翻译为： 能......；会......；可以......。能动态是日语动词语态（ヴォイス）中的一种，动词变为能动态的规则如下：

动词类别能动态变形规律例子
一类动词

词尾う段变え段 + る

買かう -> 買かえる 、 行いく -> 行いける

二类动词

去词尾る + られる

起おきる -> 起おきられる 、 寝ねる -> 寝ねられる

三类动词

来くる -> 来こられる /　する -> できる

来くる -> 来くられる 、 勉強べんきょうする -> 勉強べんきょうできる

🔖 补充

1. 他动词涉及的对象，在能动态的句子中大多由「を」变为「が」;

  1. 300元げんぐらいでシルクのスカート が 買かえますよ。

    300块左右就可以买丝绸的裙子了。

  2. 私わたしはさしみ が 食たべられません。

    我吃不了刺身。

  3. 王おうさんは英語えいご が 話はなせます。

    小王能说英语。

2. 非生物名词做主语的句子，一般不能使用动词的能动态（表示主语性质、特征的除外）;

  1. ３日みっか間かんで荷物にもつが 届とどける ❌／届とどく。

    货物3日内能到。

  2. 会議かいぎは９時じに 始はじまれる ❌／始はじまる。

    会议9点能开始。

3. 本身已包含“可能”含义的词没有能动态。如：「見みえる」「聞きこえる（听见）」「わかる」等。

动词的意志形 ​

用来表示说话人的意志或者建议, 译作要……；……吧。变形规则如下：

动词类别能动态变形规律例子
一类动词

词尾う段变お段 + う

買かう -> 買かおう 、 死しね -> 死しのう

二类动词

去词尾る + よう

見みる -> 見みよう 、 集あめる -> 集あめよう

三类动词

来くる -> 来こよう/　する -> しよう

勉強べんきょうする -> 勉強べんきょうしよう

🔖 说明&例句

A. 动词的意志形用于第一人称句时，表示说话人(第一人称)要进行该动作的意志；用于第一、二人称句时，表示说话人(第一人称)建议对方(第二人称)与自己共同进行该动作，它是「Vましょう」的简体形式；

  1. 遣使けんしの会かいで忘年会ぼうねんかいを開こうひらこうと思っておもって、計画けいかくを立てたてているんです。

    我想在遣使会上开忘年会，正在制定计划。

  2. 明日あした朝あさ早いはやいから、今晩こんばん早くはやく寝ようねよう。

    明晚要早起，今晚早点睡吧。

  3. みんなで頑張ろうがんばろう。

    大家一起加油吧。

  4. 人数にんずうが足りたりないから、王おうさんも一緒いっしょに行こういこう。

    人数不够，小王也一起去吧。

B. 「Ｖ(よ)う」后接「と思う」构成「Ｖ(よ)うと思う」这个句式，表示说话人(第一人称)的意志，即决定要做某事，相当于汉语的“我想要……；我决心……”等。

  1. 今日きょうは早くはやく帰ろうかえろうと思おもいます。

    今天想早点回家。

  2. 運動うんどうしようと思おもいますけど、時間じかんがありません。

    我想运动，但没有时间。
