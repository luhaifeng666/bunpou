# 动词谓语句

指代的是 `动词` 做 `谓语` 的句子。与中文的语序不同，在日语中，动词谓语句的语序为：

`主语+宾语(动作作用的人或事物)+谓语`。

<grammer-content sentence="例如： **[王/おう]さんは(主语) + リンゴを(宾语) + [食/た]べる(谓语)。** 吃苹果。" />
