# Nのもとで＜影響・支配＞ <Badge type="danger" text="N1" />

* 意义：表示在某人或事物的影响、支配下活动。
* 接续：名词＋のもとで
* 译文：在……之下

::: info :books: 例句

1. <grammer-content sentence='ゼミの[指導教員/しどうきょういん]**のもとで**、[同/おな]じ[関心/かんしん]を[持/も]つ[先輩/せんぱい]や[学友/がくゆう]とともにさまざまな[角度/かくど]から[学/まな]んでいくのである。' trans='在研讨课任课教师的指导下，与对同一课题感兴趣的前辈和学友一起从各个角度进行学习。' />
2. <grammer-content sentence='[吉田/よしだ][先生/せんせい]のご[指導/しどう]**のもとで**[研究/けんきゅう]を[行/おこな]ってきた。' trans='在吉田老师的指导下进行了研究。' />
3. <grammer-content sentence='どうしてこの[人/ひと]**のもとで**[働/はたら]きたいと[思/おも]っているんですか。' trans='你为什么想在这个人手下工作？' />
4. <grammer-content sentence='[今日/きょう]は、[暖/あたた]かい[太陽/たいよう]**のもとで**のんびりと[楽/たの]しい[時間/じかん]を[過/す]ごしましょう。' trans='今天让我们在温暖的阳光下悠闲地度过愉快的时光吧。' />

:::
