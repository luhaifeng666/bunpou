# すなわち＜換言＞ <Badge type="danger" text="N1" />

* 意义：表示换一种说法，用于进一步解释、说明前面提到过的事物。
* 接续：词语、句子＋すなわち
* 译文：换句话说……；亦即……

::: tip :bookmark: 説明

「つまり」也可以表示换言，但「つまり」还有"总而言之"的意思。

:::

::: info :books: 例句

1. <grammer-content sentence='**すなわち**、[社会/しゃかい]的な[空間/くうかん]の[中/なか]に[勝手/かって]に[個人/こじん]的な[空間/くうかん]を[作/つく]り、[周囲/しゅうい]の[人/ひと]を[風景/ふうけい]や[物/もの]のようにしか[見/み]なくなった[人々/ひとびと]が[増/ふ]えたのだという[見方/みかた]である。' trans='换句话说，这是一种看法，认为在社会空间中随意创造个人空间，把周围的人只当作风景和物品来看的人增加了。' />
2. <grammer-content sentence='[日本/にほん]の[気候/きこう]は、[四季/しき]**すなわち**[春夏秋冬/しゅんかしゅうとう]に[分/わ]けられる。' trans='日本的气候分为四季，即春夏秋冬。' />
3. <grammer-content sentence='[人間/にんげん]が[健康/けんこう]を[維持/いじ]するためには、[毎日/まいにち][適度/てきど]な[食事/しょくじ]と[睡眠/すいみん]をとることが[重要/じゅうよう]です。**すなわち**、それが[健康/けんこう]の[元/もと]だと[思/おも]います。' trans='人类为了维持健康，每天适度的饮食和睡眠很重要。换句话说，我认为那是健康的基础。' />
