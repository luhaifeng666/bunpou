# V（よ）うではないか＜呼びかけ＞ <Badge type="danger" text="N1" />

* 意义：表示号召，多用于书面语或者演讲中。
* 接续：动词的意志形＋ではないか／じゃないか
* 译文：（让我们）……吧

::: info :books: 例句

1. <grammer-content sentence='ともに[大学生活/だいがくせいかつ]を[充実/じゅうじつ]したものに**しようではないか**。' trans='让我们一起充实的大学生活吧！' />
2. <grammer-content sentence='みんなの[力/ちから]を[合/あ]わせて[頑張/がんば]って**みようではないか**。' trans='让我们一起努力吧！' />
3. <grammer-content sentence='これから[何/なに]をすべきか[考/かんが]えようではありませんか。' trans='让我们想想今后该做什么吧。' />
4. <grammer-content sentence='さあ、そろそろ[出/で]かけようじゃないか。' trans='来吧，差不多该出门了。' />
5. <grammer-content sentence='われわれは、[心/こころ]を[一/ひと]つにし、[中華民族/ちゅうかみんぞく]の[偉大/いだい]な[復興/ふっこう]を[全面的/ぜんめんてき]に[推進/すいしん]するために、[団結奮闘/だんけつふんとう]しようではないか。' trans='让我们团结奋斗，为全面推进中华民族的伟大复兴而同心协力吧！' />

:::
