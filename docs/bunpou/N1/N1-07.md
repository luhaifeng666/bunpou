# ～末（に）＜結果＞ <Badge type="danger" text="N1" />

* 意义：用于表示经过一段时间比较艰难的过程后，最终出现了后句所示的结果。
* 接续：Vた＋末（に）；动作性名词＋の＋末（に）
* 译文：经过了……，最终……

::: tip :bookmark: 説明

多用于书面语。经常与「いろいろ／さんざん／長い時間」等副词搭配使用。

:::

::: info :books: 例句

1. <grammer-content sentence='また、[彼/かれ]はいろいろ[考/かんが]え[抜/ぬ]いた**末に**[日本/にほん][帰化/きか]して[親方/おやかた]となった。' trans='而且，他经过深思熟虑后，最终归化日本成为了师傅。' />
2. <grammer-content sentence='いろんな[人/ひと]に[相談/そうだん]し、よく[考/かんが]えた**末**、[会社/かいしゃ]を[止/や]めることにした。' trans='和很多人商量，深思熟虑后，决定辞职。' />
3. <grammer-content sentence='さんざん[迷/まよ]った**末に**[留学/りゅうがく]を[諦/あきら]めた。' trans='犹豫了很久，最终放弃了留学。' />
4. <grammer-content sentence='[長年/ながねん]の[努力/どりょく]の**末**、やっと[自分/じぶん]の[会社/かいしゃ]を[立/た]ち[上/あ]げた。' trans='经过多年的努力，终于创立了自己的公司。' />

:::
