# こそ＜強調＞ <Badge type="danger" text="N1" />

* 意义：用于凸显、强调名词所指的事物，以区别于其他同类事物。
* 接续：名词＋（に／で／と）＋こそ
* 译文：才（是）……；正是……

::: tip :bookmark: 説明

前接名词做主语时，往往用「Ｎこそが」。

:::

::: info :books: 例句

1. <grammer-content sentence='その[意味/いみ]で、ゼミ**こそ**が、[大学生活/だいがくせいかつ]の[締/し]めくくりに[最/もっと]もふさわしい[時間/じかん]と[言/い]えるだろう。' trans='从这个意义上说，研讨课才是大学生活结束时最适合的时间吧。' />
2. <grammer-content sentence='[健康/けんこう]**こそ**がこの[上/うえ]ない[幸/しあわ]せだ。' trans='健康才是最大的幸福。' />
3. <grammer-content sentence='こちら**こそ**、[失礼/しつれい]いたしました。' trans='我才要请您多包涵。' />
4. <grammer-content sentence='[姉/あね]は、[今年/ことし]**こそ**[結婚/けっこん]したいといっていた。' trans='姐姐说今年一定要结婚。' />
5. <grammer-content sentence='こういう[時/とき]に**こそ**、[友達/ともだち]の[大切/たいせつ]さがわかる。' trans='正是这种时候，才能明白朋友的重要性。' />
6. <grammer-content sentence='この[作品/さくひん]は、[今/いま]で**こそ**知られているが、[当時/とうじ]は[無名/むめい]だった。' trans='这个作品虽然现在为人所知，但当时却默默无闻。' />

:::
