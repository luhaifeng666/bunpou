# Nにおける／において＜空間・時間＞ <Badge type="danger" text="N1" />

* 意义：表示时间、空间、范围、领域等。
* 接续：名词＋「における」＋体词；名词＋「において」＋用词
* 译文：在……；于……

::: tip :bookmark: 説明

一般为书面语。

Ｎにおける（Ｎ）＝での（例：大学における勉強＝大学での勉強）

Ｎにおいて＝で（放在句子中间连接两个句子）

:::

::: info :books: 例句

1. <grammer-content sentence='[大学/だいがく]**における**（での）[勉強/べんきょう]は[与/あた]えられるものではない。' trans='大学的学习不是被动接受的。' />
2. <grammer-content sentence='[現代社会/げんだいしゃかい]**における**（での）[環境問題/かんきょうもんだい]について[研究/けんきゅう]したい。' trans='想研究现代社会中的环境问题。' />
3. <grammer-content sentence='[今度/こんど]の[国際大会/こくさいたいかい]は[北京/ぺきん]**において**（で）[開催/かいさい]される[予定/よてい]だ。' trans='这次国际大会计划在北京举办。' />
4. <grammer-content sentence='[就職活動/しゅうしょくかつどう]**において**（で）、[自己分析/じこぶんせき]はとても[大切/たいせつ]である。' trans='在求职活动中，自我分析非常重要。' />
5. <grammer-content sentence='これらの[作品/さくひん]のテーマは、[現代/げんだい]**において**はあまり[珍/めずら]しくない。' trans='这些作品的主题在现代并不罕见。' />

:::
