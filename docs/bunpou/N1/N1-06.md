# Nを抜きにして＜排除＞ <Badge type="danger" text="N1" />

* 意义：表示如果将该名词所指的事物排除在外或无视其存在的话，主句所示的动作将无法实现。
* 接续：名词＋を抜きにしては
* 译文：抛开……（就不能……）；不提……（就不能……）

::: tip :bookmark: 説明

句尾一般为否定性表达方式。

当句尾的词语是肯定形式时，「Nを抜きにして」则构成连用修饰语从句，表示在该状态下进行某一动作。

:::
