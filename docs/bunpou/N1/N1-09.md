# Nじゃあるまいし＜否定性原因＞ <Badge type="danger" text="N1" />

* 意义：表示否定性的原因、理由，主句一般为说话人的判断、反问、祈使、责备、推测等。
* 接续：名词＋じゃ（では）あるまいし
* 译文：又不是……，（所以）

::: info :books: 例句

1. <grammer-content sentence='アニメ**じゃあるまいし**、[大金持/おおがねも]ちでかわいくて[性格/せいかく]もいい[女/おんな]の[子/こ]がいるわけがない。' trans='又不是动画，不可能有又有钱又可爱性格又好的女孩子。' />
2. <grammer-content sentence='[子供/こども]**じゃあるまいし**、そんなことで[泣/な]くな。' trans='又不是孩子，别因为那种事哭。' />
3. <grammer-content sentence='[外国人/がいこくじん]**じゃあるまいし**、[箸/はし]の[使/つか]い[方/かた]くらい[知/し]ってるよ。' trans='又不是外国人，筷子的用法还是知道的。' />
4. <grammer-content sentence='[夏/なつ]**じゃあるまいし**、そんなに[暑/あつ]くないよ。' trans='又不是夏天，没那么热。' />

:::
