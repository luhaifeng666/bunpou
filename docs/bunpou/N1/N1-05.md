# にあたり／にあたって＜動作行為の時間＞ <Badge type="danger" text="N1" />

* 意义：表示进行动作或发生变化的时间。此句式语气郑重，主要用于书面语或正式场合。
* 接续：动作性或变化性名词／动词的词典形＋にあたり／あたって
* 译文：值此……之际

::: info :books: 例句

1. <grammer-content sentence='[後期/こうき]の[大学生活/だいがくせいかつ]を[始/はじ]める**にあたり**、[何/なに]か[新/あたら]しいことにチャレンジしてみたいと[思/おも]っている[人/ひと]も[多/おお]いのではないかと[思/おも]います。' trans='值此开始后期大学生活之际，我想有很多人想尝试一些新的事情。' />
2. <grammer-content sentence='[開会/かいかい]**にあたって**、[一言/いちごん]ご[挨拶/あいさつ]申し上げます。' trans='值此开幕之际，请允许我说几句话。' />
3. <grammer-content sentence='[面接/めんせつ]を[受/う]ける**にあたって**、まず[履歴書/りれきしょ]に[書/か]いた[内容/ないよう]を、しっかりと[頭/あたま]に[入/い]れておかなければならない。' trans='在接受面试之际，首先必须把简历上写的内容牢牢记住。' />
4. <grammer-content sentence='[本書/ほんしょ]の[出版/しゅっぱん]**にあたって**、いろいろとお[世話/せわ]くださった[阪田/さかた][先生/せんせい]に[心/こころ]から[御礼/おれい]申し上げます。' trans='值此本书出版之际，向给予诸多帮助的阪田老师表示衷心的感谢。' />
