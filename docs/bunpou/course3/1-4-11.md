# だから・ですから<因果关系>

* 意义：表示因果关系，前因后果，即前面的句子表示原因，后面的句子表示由该原因引起的结果。
* 接续：用于两个句子之间。`ですから` 用于敬体句子中。
* 译文：`因此······；所以······`
