# だけ＜限定＞

* 意义：表示限定；
* 接续：名词 + だけ；
* 译文：只有......；仅仅......,接否定表达时，意为“唯独（只有）...不（没）...”

::: tip :bookmark: 说明

「だけ」用于格助词「が」「を」前面时，`「が」「を」可以省略`。

:::
