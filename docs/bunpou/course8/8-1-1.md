# Vることになる〈事态发展的结果〉 <Badge type="tip" text="N4" />

* 意义：表示客观存在的规定或事态自然发展、变化所产生的结果。
* 译文： `······的结果`。

::: tip :bookmark: 说明

1. 接续：动词的词典形 / 可能态动词的连体形 / 动词的否定形 + ことになる
2. 强调客观的规定或自然而然出现的结果，与「Vることにする」强调主观决定不同。

:::

::: info :books: 例句

1. <grammer-content sentence="今年は[2年生/にねんせい]から[出/で]られる[こと/こと]になったんですか。" trans='从今年起二年级学生也能参加了吗。' />
2. <grammer-content sentence="父は、[春/はる]から[名古屋/なごや]に[転勤/てんきん]する[こと/こと]になった。" trans='父亲从春天开始被调到名古屋了。' />
3. <grammer-content sentence="[日本語/にほんご][学科/がっか]の[学生/がくせい]は[シンポジウム/しんぽじうむ]を[手伝/てつだ]う[こと/こと]になった。" trans='日语系的学生被指派去协助举办研讨会。' />
