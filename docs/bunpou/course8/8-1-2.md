# Vることにする〈决定〉 <Badge type="tip" text="N4" />

* 意义：表示动作主体决定做(或不做)某事。
* 译文： `决定······`。

::: tip :bookmark: 说明

1. 接续：动词的词典形 / 动词的否定形 + ことにする
2. 强调动作主体主观上做出的决定，与「Vることになる」强调客观结果不同。
3. 和「する」相比，由于「なる」弱化了说话人的意志，语气显得委婉客气。

:::

::: info :books: 例句

1. <grammer-content sentence="僕、[参加/さんか]する[こと/こと]にします。" trans='我决定参加。' />
2. <grammer-content sentence="[頭/あたま]が[痛/いた]いので、[今日/きょう]は[学校/がっこう]を[休/やす]む[こと/こと]にします。" trans='因为头疼，我今天决定不上学了。' />
3. <grammer-content sentence="私は[今日/きょう]から[たばこ/たばこ]を[吸/す]わない[こと/こと]にした。" trans='我决定从今天开始不抽烟。' />
4. <grammer-content sentence="[卒業/そつぎょう]したら[就職/しゅうしょく]しないで、[会社/かいしゃ]を[立/た]ち上げる[こと/こと]にした。" trans='毕业后我决定不就业，而是创业。' />

:::

::: tip :bookmark: 比较

「Vることにする」强调动作主体主观决定，而「Vることになる」表示客观规定或自然产生结果。

:::

::: info :books: 例句

1. <grammer-content sentence="後輩：来月、[結婚/けっこん]する[こと/こと]になりました。" trans='学弟：下个月我要结婚了。' />
2. <grammer-content sentence="先輩：それは、[おめでとう]！" trans='学长：那恭喜你！' />

:::
