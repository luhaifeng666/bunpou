# Nなら（ば）〈话题、条件〉 <Badge type="tip" text="N4" />

* 意义：假设在该话题或条件之下的情况，后句表达说话人的意见、建议、判断、意图等。
* 译文： `······的话;如果是······`。

::: tip :bookmark: 说明

1. 接续：名词 + なら（ば） / 形容词 + なら（ば） / 简体句子 + なら（ば）
2. 承接对方说过的话，后面陈述说话人的建议、意志、意见、回答等。

:::

::: info :books: 例句

1. <grammer-content sentence="[李/り]さんなら、[優勝/ゆうしょう]できるかも。" trans='如果是小李的话，说不定能赢。' />
2. <grammer-content sentence="[明日/あした]の[午後/ごご]なら（ば）、[時間/じかん]があります。" trans='如果是明下午的话，我有时间。' />
3. <grammer-content sentence="[本当/ほんとう]に[風邪/かぜ]なら（ば）、[早/はや]く[帰/かえ]って[寝/ね]たほうがいいよ。" trans='如果真的感冒的话，早点回家睡觉比较好。' />
4. <grammer-content sentence="[今度/こんど]の[試験/しけん]で[成績/せいせき]が[一番/いちばん]なら（ば）、[留学/りゅうがく][試験/しけん]を[受/う]けようと[思/おも]います。" trans='如果这次考试成绩是第一名的话，我想参加留学考试。' />

:::

::: tip :bookmark: 用法

Nなら（ば）表示话题的凸显，假设在特定话题或条件之下。

:::

::: info :books: 例句

1. <grammer-content sentence="A:今から[図書館/としょかん]へ[行/い]きます。" trans='A：现在去图书馆。' />
2. <grammer-content sentence="B:あ、[図書館/としょかん]に[行/い]くなら、私も[返/かえ]したい[本/ほん]があるんですが…" trans='B：啊，如果去图书馆的话，我也有想还的书……' />
3. <grammer-content sentence="A:田中/やまだ]さん、いませんか。" trans='A：田中不在吗？' />
4. <grammer-content sentence="B:田中/やまだ]さんなら、さっき[出/で]かけましたよ。" trans='B：如果是田中的话，刚才出门了。' />
5. <grammer-content sentence="A:日本語/にほんご]を[勉強/べんきょう]したいです。" trans='A：我想学日语。' />
6. <grammer-content sentence="B:日本語/にほんご]を[勉強/べんきょう]したいなら、[授業/じゅぎょう]を[受/う]けたほうがいいですよ。" trans='B：如果学日语的话，建议你上课。' />
7. <grammer-content sentence="王/おう]さん、[納豆/なっとう]が[きらい/きらい]なら、[食/た]べなくてもいいんですよ。" trans='小王，如果你讨厌纳豆的话，不吃也可以哦。' />
