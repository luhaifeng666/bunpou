# Vてある〈客体存续的状态〉 <Badge type="tip" text="N4" />

* 意义：表示动作、行为完成之后客体存续的状态。
* 译文： `······着`。

::: tip :bookmark: 说明

1. 接续：他动词 + てある
2. 客体的状态实际上是为了某种目的而存续的，因此动词V多为自主的他动词。
3. 他动词的"を格"补足语，在「てある」的句子中一般要改成"が格"的形式。

:::

::: info :books: 例句

1. <grammer-content sentence="[参加者/さんかしゃ]は[2年生/にねんせい]以上って[書/か]いてあります。ほら。" trans='写着参加者是二年级以上。你看。' />
2. <grammer-content sentence="部屋に[鍵/かぎ]が[かけて/かけて]あるから、[安心/あんしん]です。" trans='门锁着，所以很放心。' />
3. <grammer-content sentence="[壁/かべ]に[ポスター/ポスター]が[貼/は]ってありました。" trans='墙上贴着海报。' />

:::

::: tip :bookmark: 说明

存在自他对应的动词，「他动词+てある」和「自动词+ている」都可以表达结果状态的存续，其中「Vてある」表示的是动作、行为的主体出于某种意图或理由实施该动作后，客体的存续状态，「Vている」则不具有这层含义，只是表示结果状态。

:::

::: info :books: 例句

1. <grammer-content sentence="[節電/せつでん]のために、[電気/でんき]が｛〇[消/け]してある /X[消/け]えてある/ X[消/け]えている｝。" trans='为了节电，电是关着（主动关的）。' />
2. <grammer-content sentence="[電気/でんき]が[消/け]えている。" trans='电是关着（只是描述状态）。' />

:::

::: tip :bookmark: 说明

「てある」还可以表示做好某种准备之后的状态，一般使用「〜は /を (もう)V てある」的句式。

:::

::: info :books: 例句

1. <grammer-content sentence="新しい[単語/たんご]はもう[調/しら]べてある。" trans='新单词已经查过了。' />
2. <grammer-content sentence="[高橋/たかはし]:[遠藤/えんどう]先生には[遠足/えんそく]の[こと/こと]を[話/はな]してありますか。" trans='高桥：跟远藤老师说明天去春游的事了吗？' />
3. <grammer-content sentence="王/おう:いいえ 、まだです。" trans='王：还没有。' />

:::
