# Vなさい＜命令・敬体＞ <Badge type="info" text="N5" />

* 意义：用于表示要求和命令。
* 接续：动词的第一连用形＋なさい（勉強する⇒勉強しなさい／勉強なさい）
* 译文：（请）……

::: tip :bookmark: 说明

`Vなさい`所表示的命令的语气比动词简体命令形所表示的命令的语气要缓和一些，但一般只用于大人对小孩，老师对学生，上级对下级或是关系亲密者之间。身份同等的人之间也不太使用。

`Vなさい`还可以构成`お/ご～なさい`这种句式，用于表示略为客气的命令。`おやすみなさい/ごめんなさい`已经成为寒暄语，其使用的场合比较固定。

`Vなさい`在口语中还可以简化为`Vな`，这种表达方式不太客气，仅用于关系非常密切的人（如家里人）之间。

:::
