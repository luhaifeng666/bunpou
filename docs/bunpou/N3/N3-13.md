# Vていらっしゃる＜尊他＞ <Badge type="warning" text="N3" />

* 意义：表示对动作主体的敬意。
* 接续：动词て形＋いらっしゃる
* 译文：（表示尊敬）正在……；……着

::: tip :bookmark: 説明

`Vていらっしゃる`是`Vている`的敬语（尊他）表现形式。`いらっしゃる`是`行く・来る・いる`的尊敬语。

:::

::: info :books: 例句

1. <grammer-content sentence='[実/じつ]は、[山田/やまだ]さんからそちらの[会社/かいしゃ]で[翻訳/ほんやく]ができる[人/ひと]を[探/さが]し**ていらっしゃる**とお[聞/き]きしたんですが……。' trans='其实，我听山田说贵公司正在找能翻译的人……' />
2. <grammer-content sentence='お[仕事/しごと]は[何/なに]をし**ていらっしゃいます**か。' trans='您做什么工作？' />
3. <grammer-content sentence='[佐藤/さとう][先生/せんせい]は[神戸/こうべ]に[住/す]ん**でいらっしゃる**そうです。' trans='听说佐藤老师住在神户。' />
4. <grammer-content sentence='みなさんはスマホで[動画/どうが]を[撮/と]っ**ていらっしゃいます**か。' trans='大家用智能手机拍视频吗？' />
