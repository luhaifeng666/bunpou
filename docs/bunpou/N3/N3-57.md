# とても＋V能动ない＜否定可能性＞ <Badge type="warning" text="N3" />

* 意义：表示无论用什么样的方法都无法实现该动作或行为。外在限制导致。
* 接续：とても＋能动态的否定形
* 译文：无论如何都不（能）……；根本就无法……

::: info :books: 例句

1. <grammer-content sentence='かなり[早口/はやくち]で、**とても**[私/わたし]には[内容/ないよう]を**聞ききとることはできなかった**。' trans='语速很快，我无论如何也听不懂内容。' />
2. <grammer-content sentence='[彼/かれ]がなぜそんなことをしたのか、**とても**[私/わたし]には**理解りかいできない**。' trans='他为什么做那样的事，我怎么也无法理解。' />
3. <grammer-content sentence='[目的地/もくてきち]までは3～4キロあって、**とても**[歩/ある]いて**行いけそうにない**。' trans='目的地有3-4公里，根本不可能走过去。' />
4. <grammer-content sentence='[出/だ]された[料理/りょうり]はどれもおいしかったが、[二人/ふたり]では**とても** **食べきれなかった**。' trans='端上来的菜都很好吃，但两个人根本吃不完。' />
:::
