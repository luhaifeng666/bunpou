# Vたばかりだ〈动作刚刚结束〉 <Badge type="warning" text="N3" />

- 意义：表示动作行为完成或某件事情发生不久。
- 译文：刚......；刚......不久

::: info :books: 例句

1. <grammer-content sentence='あのころは、[私/わたし]は[北京/ペキン]に[来/き]たばかりで、[中国語/ちゅうごくご]もあまりわからなくて……。' trans='那时候，我刚来北京，还不太懂中文……。' />
2. <grammer-content sentence='[先月/せんげつ]張さんは[日本/にほん]から[帰/き]って来たばかりです。' trans='上个月张先生刚从日本回来。' />
3. <grammer-content sentence='その[言葉/ことば]は[習/なら]ったばかりで、まだ[上手/じょうず]に[使/つか]えません。' trans='那个单词刚刚学会，还不熟练。' />
4. <grammer-content sentence='このパソコンは[去年/きょねん][買/か]ったばかりです。' trans='这台电脑去年刚买。' />

:::
