# ～さえ～ば<充分条件> <Badge type="warning" text="N3" />

* 意义：表示只要满足该条件，就可以得出后面的结论。
* 接续：名词+さえ+动词、形容词的条件形
* 译文：只要……就……

::: tip :bookmark: 説明

接续：①Nさえ+V/Aば ②V第一连用+さえ+すれば ③Vて（状态性）+さえ+いれば ④A1く/A2で+さえ+あれば

:::

::: info :books: 例句

1. <grammer-content sentence='[日本語/にほんご]**さえ[使/つか]えれば**[バイト代/ばいとだい]は[安/やす]なくてもいいんです。' trans='只要会日语，兼职费便宜点也没关系。' />
2. <grammer-content sentence='[雨/あめ]**さえ[降/ふ]らなければ**[少/すこ]し[寒/さむ]くても[出/で]かけよう。' trans='只要不下雨，就算有点冷也出门吧。' />
3. <grammer-content sentence='**[ Yel /ゆめ]、[欲/ほ]しくなる**さえ[あれ][ば/れば][国籍/こくせき]には**[拘/こだ]わりません**[よ/よ]。' trans='反正，只要有干劲，我不在乎是哪个国籍的。' />
4. <grammer-content sentence='**[一所懸命/いっしょうけんめい][勉強/べんきょう]さえすれば**[試験/しけん]に[合格/ごうかく]できる[だろう/だろう]**。' trans='只要拼命学习，就能通过考试吧。' />
:::
