# ～ないことはない<双重否定> <Badge type="warning" text="N3" />

- <grammer-content sentence="意义：通过双重否定的形式，表达肯定的意义。" />
- <grammer-content sentence="接续：**动词、形容词的否定形**＋ことはない" />
- 译文：也不是不……；也不是不……

::: tip :bookmark: 説明

也可以用「ないこともない」的形式，语气更为委婉。

:::

::: info :books: 例句

1. <grammer-content sentence="A：これ、まだ食べられる？" trans="A：这个，还能吃吗？" />
   <grammer-content sentence="B：う〜ん、[食/た]べられ**ないことはない**[けど]、[賞味期限/しょうみきげん]が[切/き]れているのでやめておいたほうが[良/い]いと[思/おもう]よ。" trans="B：嗯，也不是不能吃，只是保质期已经过了，我觉得还是不吃为好。" />

2. <grammer-content sentence="[出来/でき]**ないことはない**[けど]。" trans="也不是做不到，只是……" />

3. <grammer-content sentence="[納豆/なっとう]は[食/た]べ**ないことはない**[んですけど]、[余/あま]り[好/す]きではありません。" trans="纳豆也不是不吃，只是不太喜欢。" />

4. <grammer-content sentence="[提出/ていしゅつ]は[明日/あした]は[無理/むり]です[が]、[明後日/あさって]までなら、[出来/でき]**ないことはない**。" trans="明天交不行，但后天之前也不是做不到。" />

5. <grammer-content sentence="A：[林/りん]さん、[彼女/かのじょ]のこと、ほんとに[好/す]きじゃないんじゃないの？" trans="A：林同学，你确实不是不喜欢她吧？" />
   <grammer-content sentence="B：う〜ん、[好/す]きじゃ**ないことはない**んだけど……。" trans="B：嗯，也不是不喜欢……" />

:::
