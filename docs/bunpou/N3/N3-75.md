# ~ないことはない<双重否定> <Badge type="warning" text="N3" />

* 意义：通过双重否定的形式，表达肯定的意义。
* 接续：动词、形容词的否定形+ことはない
* 译文：也不是不……；也不是不……

::: tip :bookmark: 説明

也可以用「ないこともない」的形式，语气更为委婉。
:::


::: info :books: 例句

1. <grammer-content sentence="**[食/た]べない**んだけど although[好/す]きではありません。" trans="纳豆也不是不吃，只是不太喜欢。" />
2. <grammer-content sentence="**[できない**[かもしれない/かもしれない]**けど。" trans="也不是做不到，只是……" />
3. <grammer-content sentence="**[おいしくない**んだけど，[少/すこ]しがい[気/き]が**。" trans="也不是不好吃，只是觉得味道有点淡。" />
:::
