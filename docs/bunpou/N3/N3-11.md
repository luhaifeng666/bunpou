# N／AⅡでいらっしゃる＜尊他＞ <Badge type="warning" text="N3" />

* 意义：用于表示对他人的尊敬。
* 接续：名词／Ⅱ类形容词词干＋でいらっしゃる
* 译文：（表示尊敬）是……

::: tip :bookmark: 説明

`N／AⅡでいらっしゃる`是`N／AⅡです`的敬语（尊他）表现形式。

:::

::: info :books: 例句

1. <grammer-content sentence='あ、[丸井/まるい]さん**でいらっしゃいます**か。' trans='啊，您是丸井先生吗？' />
2. <grammer-content sentence='[失礼/しつれい]ですが、どちらさま**でいらっしゃいます**か。' trans='不好意思，请问您是哪位？' />
3. <grammer-content sentence='こちらは[鈴木/すずき]さんのお[母様/かあさま]**でいらっしゃいます**。' trans='这位是铃木先生的母亲。' />
4. <grammer-content sentence='お[久/ひさ]しぶりです。お[元気/げんき]**でいらっしゃいます**か。' trans='好久不见。您身体好吗？' />

:::
