# さえ＜凸顕代表性的事物＞ <Badge type="warning" text="N3" />

* 意义：用于凸显有代表性的事物，表示该事物尚且如此，其他自不用说之意。
* 接续：名词＋（に／で／と＋）さえ
* 译文：连……都……；甚至……

::: info :books: 例句

1. <grammer-content sentence='あの[金閣寺/きんかくじ]**でさえ**[当時/とうじ]の[人/ひと]の[目/め]には[驚/おどろ]きだった、ということを[考/かんが]えれば[新/あたら]しさと[伝統/でんとう]は[紙/かみ]の[裏表/うらおもて]ということになる。' trans='就连那个金阁寺对当时的人来说都是令人惊讶的，考虑到这一点，新事物和传统就是一张纸的正反面。' />
2. <grammer-content sentence='[自分/じぶん]の[心/こころ]は[自分/じぶん]**にさえ**わからないこともある。' trans='自己的心连自己都不明白的情况也有。' />
3. <grammer-content sentence='それは[小学生/しょうがくせい]**さえ**[知/し]っている[常識/じょうしき]だ。' trans='那是连小学生都知道的常识。' />
4. <grammer-content sentence='[彼/かれ]は[病気/びょうき]のことを、[自分/じぶん]の[妻/つま]**にさえ**[話/はな]していないらしい。' trans='他好像连自己的妻子都没有告诉生病的事。' />
5. <grammer-content sentence='[卒業以来/そつぎょういらい]、[彼/かれ]からはまったく[連絡/れんらく]がなく、[今/いま]どこで[何/なに]をしているの**かさえ**わからない。' trans='毕业以来，他完全没有联系，连现在在哪里做什么都不知道。' />

:::
