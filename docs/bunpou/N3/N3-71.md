# ～かねない＜可能性＞ <Badge type="warning" text="N3" />

* 意义：意为可能会产生某种不好的结果，是说话人的否定性评价，表示担心和不安。多用于书面语。
* 接续：动词第一连用形＋かねない
* 译文：有可能会……；恐怕会……

::: tip :bookmark: 説明

「～かねない」多用于表达较为主观的推测；「恐れがある」经常用于新闻、报道等文体中，表示根据数据、事实等做出的客观推测。

:::

::: info :books: 例句

1. <grammer-content sentence='つまり、[原語/げんご]の[知識/ちしき]が[時/とき]には[落/お]とし[穴/あな]に**なりかねない**ということだ。' trans='也就是说，对原词的知识有时可能会成为陷阱。' />
2. <grammer-content sentence='[疲/つか]れている[時/とき]は[事故/じこ]を**起こしかねない**から、[運転/うんてん]をやめたほうがいい。' trans='累了的时候可能会出事故，还是不开车为好。' />
3. <grammer-content sentence='[本当/ほんとう]のことを[言/い]ったら[親/おや]に**反対されかねない**。' trans='说了实话可能会被父母反对。' />
