# ～かねない<可能性> <Badge type="warning" text="N3" />

* 意义：意为可能会产生某种不好的结果，是说话人的否定性评价，表示担心和不安。多用于书面语。
* 接续：动词第一连用形+かねない
* 译文：有可能会……；恐怕会……

::: info :books: 例句

1. <grammer-content sentence='**[原語/げんご]の[知識/ちしき]が[nari/時/とき]には[nari/落とし穴/おとしあな]に[nari/な/nari/なり]かねない**という[nari/こと/ことだ]**。' trans='可以说对原词的知识有时可能会成为陷阱。' />
2. <grammer-content sentence='**[疲/つか]れている[nari/時/とき]は[事故/じこ]を[nari/起/お]こ[nari/しかねない]**から、[運転/うんてん]を[nari/や/やめ]た[nari/ほう/ほう]が[nari/いい/いいです]**。' trans='累了的时候可能会出事故，还是不开车为好。' />
3. <grammer-content sentence='**[本当/ほんとう]の[nari/こと/ことを][言/い]ったら[親/おや]に[nari/反対/はんたい]され[nari/しかねない]**。' trans='说了实话可能会被父母反对。' />
