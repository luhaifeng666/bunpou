# ～といえば＜話題を提出＞ <Badge type="warning" text="N3" />

* 意义：将刚才谈话中所涉及的事物作为话题提出，后续内容是对此话题的具体说明。
* 接续：名词＋といえば
* 译文：说起……；提到……

::: tip :bookmark: 説明

这个句型也可以说成「～というと」「といったら」，口语中可以将「と」替换为「～って」。

:::

::: info :books: 例句

1. <grammer-content sentence='[日本文化/にほんぶんか]**といえば**[空手/からて]だよね。' trans='说起日本文化，就是空手道吧。' />
2. <grammer-content sentence='[四川料理/しせんりょうり]**といえば**、マーボードーフでしょう。' trans='提到四川菜，就是麻婆豆腐吧。' />
3. <grammer-content sentence='バレンタイン**っていえば**チョコレート。' trans='说起情人节，就是巧克力。' />
4. <grammer-content sentence='[大晦日/おおみそか]のテレビ[番組/ばんぐみ]**といえば**、やっぱり[紅白歌合戦/こうはくうたがっせん]ですね。' trans='说到除夕的电视节目，果然是红白歌会啊。' />
5. <grammer-content sentence='[日本文化/にほんぶんか]**といったら**[空手/からて]だよね。' trans='说起日本文化，就是空手道吧。' />
6. <grammer-content sentence='[四川料理/しせんりょうり]**というと**、マーボードーフでしょう。' trans='提到四川菜，就是麻婆豆腐吧。' />

:::
