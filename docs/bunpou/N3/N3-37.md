# ところが＜転折＞ <Badge type="warning" text="N3" />

* 意义：表示转折，即后面的内容与根据前文内容进行的预测、期待相反。有时也用于表示两项事物的对比。后面多为消极的内容。
* 译文：可是；然而；不过

::: tip :bookmark: 説明

后句不能使用意志、愿望、命令、推测等表达方式。

:::

2. <grammer-content sentence='[遅/おく]れないように[朝早/あさはや]く[出/で]かけた。**ところが**[途中/とちゅう]で[渋滞/じゅうたい]に[合/あ]い、[結局/けっきょく][遅刻/ちこく]してしまった。' trans='为了不迟到早上很早就出门了。可是途中遇到堵车，结果还是迟到了。' />
