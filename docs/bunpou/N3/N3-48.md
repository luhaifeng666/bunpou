# ～もの＜強調原因・理由＞ <Badge type="warning" text="N3" />

* 意义：表示原因、理由。
* 接续：简体句子＋もの
* 译文：因为……嘛

::: tip :bookmark: 説明

多用于为自己的动作行为辩解，有时也用来表示强调。多为女性或儿童在非正式的会话中使用。口语中常说「もん」。

:::

::: info :books: 例句

1. <grammer-content sentence='A：[遅/おそ]かったですね。B：[道/みち]が[込/こ]んでた**もの**。' trans='A：好晚啊。B：因为路上很堵嘛。' />
2. <grammer-content sentence='[子供/こども]は[寝/ね]る[時間/じかん]だ**もの**、[早/はや]く[寝/ね]なさい。' trans='因为孩子该睡觉了嘛，快睡吧。' />
3. <grammer-content sentence='[私/わたし]は[子供/こども]じゃない**もの**。' trans='因为人家不是孩子嘛。' />

:::
