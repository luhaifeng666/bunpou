# ～に比べて＜比較＞ <Badge type="warning" text="N3" />

* 意义：表示比较。被比较的如果是不同的事物，表示二者之间的差异；如果为同一事物，通常表示变化。
* 接续：名词＋に比べて
* 译文：比起……

::: info :books: 例句

1. <grammer-content sentence='そういえば、[前/まえ]**に比べて**[逞/たくま]しくなったわね。' trans='说起来，比以前强壮了呢。' />
2. <grammer-content sentence='[一般的/いっぱんてき]に、[女性/じょせい]**に比べて**、[男性/だんせい]は[体力/たいりょく]があると[言/い]われている。' trans='一般来说，比起女性，男性被认为更有体力。' />
3. <grammer-content sentence='[東京/とうきょう]**に比べて**[鎌倉/かまくら]には[緑/みどり]や[花/はな]がたくさんあります。' trans='比起东京，镰仓有很多绿色和花。' />
4. <grammer-content sentence='[豆腐/とうふ]は[肉/にく]**に比べて**[低/てい]カロリーだから、[人気/にんき]がある。' trans='豆腐比肉热量低，所以受欢迎。' />
5. <grammer-content sentence='3[年間/ねんかん][留学/りゅうがく]した[彼女/かのじょ]の[語学力/ごがくりょく]は[同級生/どうきゅうせい]**に比べて**[飛躍的/ひやくてき]に[伸/の]びた。' trans='留学三年的她的语言能力比同学飞跃性地提高了。' />

:::
