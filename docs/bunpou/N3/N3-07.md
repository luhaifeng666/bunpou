# 非自発動詞が「可能」の意味を表す <Badge type="warning" text="N3" />

* 意义：一部分非自主动词，其本身就带有可能的意义（一般称为「無標可能」），不使用`Vることができる``V(ら)れる`的形式。
* 译文：能够……；可以……

::: tip :bookmark: 説明

我们在第一册第9课学习过句式`Vることができる``V(ら)れる`表示"可能"意义的用法，此时出现的动词一般为自主动词。也有一部分非自主动词，其本身就带有可能的意义，在学习时需要注意。

这部分非自主动词多为有与之对应的他动词的自动词，如`見つかる``開く``上がる``入る``続く``決まる``つく`等，也包括`見える``聞こえる``わかる``慣れる`等自动词。

:::
