# N向け＜対象＞ <Badge type="warning" text="N3" />

* 意义：表示以某人或群体为对象。
* 接续：名词＋向け
* 译文：面向……

::: tip :bookmark: 説明

「N向け」表示以某人或群体为对象，「N向き」表示本身就拥有的特质，适于某方面。

:::

::: info :books: 例句

1. <grammer-content sentence='この[番組/ばんぐみ]は[子供/こども]**向け**です。' trans='这个节目是面向儿童的。' />
2. <grammer-content sentence='[外国人/がいこくじん]**向け**の[日本語/にほんご][教科書/きょうかしょ]を[買/か]った。' trans='买了面向外国人的日语教科书。' />
3. <grammer-content sentence='この[商品/しょうひん]は[女性/じょせい]**向け**に[開発/かいはつ]された。' trans='这个商品是面向女性开发的。' />

:::
