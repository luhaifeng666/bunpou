# ～向け<对象> <Badge type="warning" text="N3" />

* 意义：表示以某人或群体为对象。
* 接续：名词（多为指称人、组织的名词）+向け
* 译文：面向……；适合于……

::: tip :bookmark: 説明

「N向け」可以用「向けの」的形式做连体修饰语，也可以用「N向けに」的形式做连用修饰语。

:::

::: info :books: 例句

1. <grammer-content sentence='[次/つぎ]は[世界/せかい][平和/へいわ]を[メッセージ/めっせーじ]とした**[子供/こども]向けの**[作品/さくひん]**にする[つもり/つもり]**です。' trans='接下来打算以世界和平为主题，制作面向孩子的作品。' />
2. <grammer-content sentence='[最近/さいきん][男性/だんせい]**向けの**[料理教室/りょうりきょうしつ]**が[流行/りゅうこう]っている[ようだ/ようです]**。' trans='最近男性专用的料理教室似乎很流行。' />
3. <grammer-content sentence='[中村/なかむら]さんは[北京/ペキン]に[住/す]む**[日本人/にほんじん]向けに**[情報雑誌/じょうほうざっし]**を[作/つく]っている**。' trans='中村先生面向居住在北京的日本人制作信息杂志。' />
4. <grammer-content sentence='[会社/かいしゃ]の[輸出/ゆしゅつ]の8[割/わり]は**[日本/にほん]向け**です。' trans='公司出口的八成是面向日本的。' />
:::
