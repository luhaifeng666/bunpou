# Nさえ～ば＜充分条件＞ <Badge type="warning" text="N3" />

* 意义：表示只要满足该条件，就可以得出后面的结论。
* 接续：名词＋さえ＋动词、形容词的条件形
* 译文：只要……就……

::: info :books: 例句

1. <grammer-content sentence='[日本語/にほんご]**さえ**[使/つか]えれば、バイト[代/だい]は[安/やす]くてもいいんです。' trans='只要能用日语，报酬低一点也没关系。' />
2. <grammer-content sentence='[雨/あめ]**さえ**[降/ふ]らなければ、[少/すこ]し[寒/さむ]くても[出/で]かけよう。' trans='只要不下雨，即使有点冷也出门吧。' />
3. <grammer-content sentence='スマホ**さえ**あれば、ほかは[何/なに]も[要/い]らない。' trans='只要有智能手机，其他什么都不需要。' />
4. <grammer-content sentence='[自分/じぶん]**さえ**よければいいと[考/かんが]える[人/ひと]がいる。' trans='有人认为只要自己好就行。' />
5. <grammer-content sentence='[体/からだ]**さえ**[健康/けんこう]なら（ば）、それで[充分/じゅうぶん]に[幸/しあわ]せだ。' trans='只要身体健康，那样就很幸福了。' />

:::
