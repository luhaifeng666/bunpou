# ～ものだ＜回顾、感叹＞ <Badge type="warning" text="N3" />

* 意义：用于感慨地回顾过去的某一习惯或经常做的事；或表示说话人的吃惊、感叹。
* 接续：Vた＋ものだ（回顾）；动词、形容词的连体形＋ものだ（感叹）
* 译文：（那时候）经常……

::: info :books: 例句

1. <grammer-content sentence='よく[元/もと]カノと[一緒/いっしょ]にそのレストランに**行ったものだ**。' trans='以前经常和前女友一起去那家餐厅。' />
2. <grammer-content sentence='[学生/がくせい][時代/じだい]はよく[友達/ともだち]とあの[店/みせ]で[食事/しょくじ]を**したものだ**。' trans='学生时代经常和朋友在那家店吃饭。' />
3. <grammer-content sentence='[子供/こども]の[頃/ころ]、[放課後/ほうかご][遅/おそ]くまで[外/そと]で[遊/あそ]んでよく[親/おや]に**怒おこられたものだ**。' trans='小时候放学后在外面玩到很晚，经常被父母骂。' />
4. <grammer-content sentence='[時/とき]の[流/なが]れは**速いものだ**な。' trans='时光流逝真快啊。' />
5. <grammer-content sentence='デジタル[機器/きき]のおかげで[本当/ほんとう]に[便利/べんり]な[時代/じだい]に**なったものだ**。' trans='多亏了数码设备，真的变成了一个便利的时代。' />
:::
