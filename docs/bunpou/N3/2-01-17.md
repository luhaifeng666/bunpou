# Nみたいだ〈比喻、举例〉 <Badge type="warning" text="N3" />

- 意义：在本课中用于比喻和举例。
- 接续：名词 + みたいだ
- 译文：好像......；跟......一样；像......那样

::: info :books: 说明

「みたいだ」接在名词的后面时，与「ようだ」的用法相同，它是「ようだ」的口语表现形式，多用于口语中。「みたいだ」的活用形式与II类形容词相同。

:::

### ① 表示比喻

这是明喻的用法，「みたいだ」前面的名词表示喻体。用于比喻时经常与副词「まるで」呼应。

::: info :books: 例句

1. <grammer-content sentence='まるで、このあいだみんなで[見/み]た[映画/えいが]の、あの[主人公/しゅじんこう]みたいですよ。' trans='简直就像之前大家一起看的电影里面的主人公一样呢。' />
2. <grammer-content sentence='[昨日/きのう]、大好きな[歌手/かしゅ]に[会/あ]った。まるで[夢/ゆめ]みたいだ。' trans='昨天见到了最喜欢的歌手。简直像做梦一样。' />
3. <grammer-content sentence='まるで[台風/たいふう]みたいにすごい[風/かぜ]が[吹/ふ]いている。' trans='像台风一样刮着大风。' />

:::

### ② 表示举例

表示某一事物（多用主语表示）与作为例子举出的另一事物具有某一相同特征。用于举例时通常不用「みたいだ」的形式，而采用连用修饰语「みたいに」或连体修饰语「みたいな」的形式。

::: info :books: 例句

1. <grammer-content sentence='[私/わたし]も[鈴木/すずき]さんみたいにライオンヘアにしようかな。' trans='我也想做一个跟铃木一样的狮子头吧。' />
2. <grammer-content sentence='[王/おう]さんみたいにやさしい[人/ひと]になりたいです。' trans='我想成为像小王那样温柔的人。' />
3. <grammer-content sentence='[山田/やまだ]さんみたいな[日本人/にほんじん]は[珍/めずら]しいと[思/おも]います。' trans='我觉得像山田那样的日本人很少见。' />

:::
