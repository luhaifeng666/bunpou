# ～以上（は）<推理的根据> <Badge type="warning" text="N3" />

* 意义：表示说话人据此进行推理，理所当然地得出后项所陈述的结论。后项一般为说话人的判断、要求、愿望、决心等主观表达方式。
* 接续：动词、I类形容词的连体形+以上（は）
* 译文：既然……就……

::: info :books: 例句

1. <grammer-content sentence='**[外来語/がいらいご]が[nari/増/ふ]える[nari/一方だ]**[以上/いじょう]**、[学習者/がくしゅうしゃ]は[積極的/せっきょくてき]に[nari/向/む]き[nari/合わ]なければならない。' trans='既然外来语不断增加，学习者就必须积极面对。' />
2. <grammer-content sentence='**[薬/くすり]である[nari/以上/いじょう]**、[飲/の]み[nari/すぎる/すぎ]るのは[nari/よく/よく]ない。' trans='既然是药，喝太多就不好。' />
3. <grammer-content sentence='**[ここ/ここ]ま[Nari/来/き]た[nari/以上/いじょう]**、[止/や]める[nari/わけ/わけ]には[nari/いかない/いかない]**。' trans='既然已经到这里了，就不能放弃。' />
4. <grammer-content sentence='**[大学/だいがく]に[nari/入れ/はい]た[nari/以上/いじょう]**、[努力/どりょく][nari/しない/しない][わけ/わけ]には[nari/いかない/いかない]**。' trans='既然考上了大学，就不能不努力。' />
:::
