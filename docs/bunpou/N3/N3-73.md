# ～以上（は）＜推理的根拠＞ <Badge type="warning" text="N3" />

* 意义：表示说话人据此进行推理，理所当然地得出后项所陈述的结论。后项一般为说话人的判断、要求、愿望、决心等主观表达方式。
* 接续：动词、Ⅰ类形容词的连体形＋以上（は）；名词／Ⅱ类形容词的词干＋である＋以上（は）
* 译文：既然……就……

::: info :books: 例句

1. <grammer-content sentence='[外来語/がいらいご]が[増/ふ]える[一方/いっぽう]である**以上**、[学習者/がくしゅうしゃ]は[積極的/せっきょくてき]に[向/む]き[合/あ]わなければならない。' trans='既然外来语不断增加，学习者就必须积极面对。' />
2. <grammer-content sentence='[薬/くすり]である**以上**、[飲/の]みすぎるのはよくない。' trans='既然是药，喝太多就不好。' />
3. <grammer-content sentence='ここまで[来/き]た**以上**、やめるわけにはいかない。' trans='既然已经到这里了，就不能放弃。' />
4. <grammer-content sentence='あらゆる[製品/せいひん]は[人間/にんげん]が[作/つく]る**以上**、[完全/かんぜん]であるとは[言/い]えない。' trans='既然所有产品都是人类制造的，就不能说是完美的。' />

:::
