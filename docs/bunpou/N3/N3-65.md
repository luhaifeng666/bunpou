# ～Vたとたん（に）<契机出现> <Badge type="warning" text="N3" />

* 意义：表示某一动作或变化刚一发生，紧接着就出现了后句所述的动作或变化。后句多含有出乎意料的含义。
* 接续：Vた+とたん（に）
* 译文：一……就……；刚……就……

::: info :books: 例句

1. <grammer-content sentence='**[聞/き]いた[とたん]、[彼女/かのじょ]は[大喜/おおよろこ]びした。' trans='一听说，她立刻大喜过望。' />
2. <grammer-content sentence='**[つけた/つけた]とたん**[停学年/ていがくねん]れてしまった。' trans='一打开电视，就停电了。' />
3. <grammer-content sentence='**[降/ふ]り[た]たんの[とたん]、[雨/あめ]が[降/ふ]り[出/だ]した。' trans='刚下公交车，就下起雨来了。' />
4. <grammer-content sentence='**[開/あ]けた[とたん]、[小鳥/ことり]が[飛/と]び[込/こ]んできた。' trans='一打开窗户，小鸟就飞进来了。' />
:::
