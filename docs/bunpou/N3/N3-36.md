# N以来／Vて以来＜時間状語＞ <Badge type="warning" text="N3" />

* 意义：表示自那时起至今一直持续某种状态。
* 接续：时间名词或动作性名词／Vて＋以来
* 译文：~以来一直~

::: info :books: 例句

1. <grammer-content sentence='わあ、[久/ひさ]しぶり！[中学校/ちゅうがっこう]の[修学旅行/しゅうがくりょこう]**以来**だなあ。' trans='哇，好久不见！自中学修学旅行以来（一直没见）。' />
2. <grammer-content sentence='あのときの[大失敗/だいしっぱい]**以来**、[自信/じしん]がなくなっちゃったよ。' trans='自那次大失败以来，就没自信了。' />
3. <grammer-content sentence='お[正月/しょうがつ]**以来**、ずっと[寒/さむ]い[日/ひ]が[続/つづ]いている。' trans='自正月以来，寒冷的日子一直持续着。' />
4. <grammer-content sentence='ダイエットを[始/はじ]めて**以来**、[大好/だいす]きなチョコレートは[食/た]べていない。' trans='自开始减肥以来，一直没吃最喜欢的巧克力。' />
5. <grammer-content sentence='[大学/だいがく]を[卒業/そつぎょう]して**以来**、8[年/ねん]ぶりでしょうか。' trans='自大学毕业以来，有8年了吧。' />

:::
