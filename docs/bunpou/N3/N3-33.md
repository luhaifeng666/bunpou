# ～とか＜不確切な間接引語＞ <Badge type="warning" text="N3" />

* 意义：表示内容不确切的间接引语。通常用「Ｎによると」或「Ｎの話では」等表达信息的来源。
* 接续：简体句子＋とか
* 译文：据说（好像）……

::: info :books: 例句

1. <grammer-content sentence='[彼女/かのじょ]によると、[周/まわ]りにいる[乗客/じょうきゃく]が[人/ひと]として[意識/いしき]されず[風景/ふうけい]のようなものになっているから、[化粧/けしょう]ができるのだ**とか**。' trans='据她说，周围的乘客不被当作人看待，而像风景一样，所以能化妆。' />
2. <grammer-content sentence='ニュースによると、[今年/ことし]は9[月/がつ]に[入/はい]ってもまだまだ[暑/あつ]い**とか**。' trans='据新闻说，今年进入9月还是很热。' />
3. <grammer-content sentence='ガイドの[話/はなし]では、そのお[寺/てら]には[日本最古/にほんさいこ]の[桜/さくら]の[木/き]がある**とか**。' trans='据导游说，那座寺庙有日本最古老的樱花树。' />
4. <grammer-content sentence='ガイドブックによると、このお[酒/さけ]は[全国/ぜんこく]にファンが[多/おお]い**とか**。' trans='据旅游指南说，这种酒在全国有很多粉丝。' />

:::
