# さすが＜評価＞ <Badge type="warning" text="N3" />

* 意义：用于实际情况与预测或期待的事情一致时表达感慨、感叹。
* 译文：真不愧是……；到底是……；的确……；果然……

::: tip :bookmark: 説明

常用的形式还有「さすがは」和「さすがに」。「さすが」和「さすがは」一般用于积极、褒义的评价。「さすがに」既用于褒义，也用于贬义。

:::

::: info :books: 例句

1. <grammer-content sentence='**さすが**[王/おう]さん、[詳/くわ]しいね。' trans='真不愧是王，懂得真多。' />
2. <grammer-content sentence='**さすが**[万里/ばんり]の[長城/ちょうじょう]だ。' trans='真不愧是万里长城。' />
3. <grammer-content sentence='Ａ：おかげさまで[優勝/ゆうしょう]しました。Ｂ：**さすが**！' trans='A：托您的福，我赢了。B：真厉害！' />
4. <grammer-content sentence='[ハルピン/はるぴん]の[冬/ふゆ]は**さすがに**[寒/さむ]い。' trans='哈尔滨的冬天果然很冷。' />

:::
