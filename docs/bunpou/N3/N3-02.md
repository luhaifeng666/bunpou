# Vてしまう＜強行做某事的意志＞ <Badge type="warning" text="N3" />

* 意义：表示不顾困难、障碍或别人的意愿，强行进行该动作。
* 接续：自主动词Vて＋しまう
* 译文：干脆……；……算了

::: tip :bookmark: 説明

该用法在使用时常采用意志形`Vてしまおう/Vちゃおう`的形式。在口语中`Vて(で)しまう`一般说成`Vちゃ(じゃ)う`。

接在非自主动词后面时，除可表示消极的结果（第5课第2单元）外，还可表示未预料的事情的发生。

:::
