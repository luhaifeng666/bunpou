# Nとも〈相同〉 <Badge type="warning" text="N3" />

- 意义：表示复数的人和事物都具有谓语所表达的特征。
- 接续：指称两个或两个以上的人或事物的名词（或数量词）+ とも
- 译文：全部......；都......

::: info :books: 例句

1. <grammer-content sentence='きっとお[二人/ふたり]とも寂しかったんでしょうね。' trans='他们二人肯定觉得很寂寞吧。' />
2. <grammer-content sentence='[孫/そん]さんは[大学/だいがく]で[英語/えいご]と[日本語/にほんご]を[勉強/べんきょう]しました。[両方/りょうほう]ともたいへん[上手/じょうず]です。' trans='孙先生在大学学了英语和日语，两者都很擅长。' />
3. <grammer-content sentence='[兄弟/きょうだい]4[人/にん]とも同じ[高校/こうこう]に[通/かよ]っていました。' trans='兄弟四人都在同一所高中上学。' />
4. <grammer-content sentence='[男女/だんじょ]とも好きな[野菜/やさい]の[一位/いちい]はトマトだそうです。' trans='男女都喜欢的第一位的蔬菜是西红柿。' />

:::
