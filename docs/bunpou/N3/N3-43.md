# ものだ＜事物の本質＞ <Badge type="warning" text="N3" />

* 意义：表示说话人对事物的本质、真理、客观规律等的认识。
* 接续：动词、形容词的连体形＋ものだ
* 译文：本来就是……；应该……

::: info :books: 例句

1. <grammer-content sentence='[人/ひと]はだれでも[間違/まちが]える**ものだ**。' trans='人都会犯错。' />
2. <grammer-content sentence='[子供/こども]は[親/おや]の[言/い]うことを[聞/き]かない**ものだ**。' trans='孩子不听父母的话是正常的。' />
3. <grammer-content sentence='[時間/じかん]が[経/た]つのは[早/はや]い**ものだ**。' trans='时间过得真快。' />
4. <grammer-content sentence='[若者/わかもの]は[理想/りそう]を[持/も]つ**ものだ**。' trans='年轻人应该有理想。' />

:::
