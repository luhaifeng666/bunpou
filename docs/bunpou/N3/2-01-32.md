# Vながら〈同时〉 <Badge type="warning" text="N3" />

- 意义：表示两个动作同时进行或持续交替进行。
- 接续：动词第一连用形 + ながら
- 译文：一边......一边......；边......边......

::: info :books: 说明

两个动作必须是同一个动作主体。

:::

::: info :books: 例句

1. <grammer-content sentence='父も母も、ときどき[涙/なみだ]を[拭/ふ]きながら、お[祝/いわい]いのスピーチを[聞/き]いていました。' trans='爸爸妈妈时不时地边擦眼泪边听贺词。' />
2. <grammer-content sentence='[弟/おとうと]はいつもテレビを[見/み]ながら、[宿題/しゅくだい]をしています。' trans='弟弟总是边看电视边做作业。' />
3. <grammer-content sentence='李さんはアルバイトをしながら、[日本語/にほんご]の[勉強/べんきょう]をしている。' trans='小李一边打工一边学习日语。' />

:::
