# Vきる／きれる／きれない＜动作彻底与否＞ <Badge type="warning" text="N3" />

* 意义：表示彻底进行该动作。
* 接续：动词第一连用形＋きる／きれる／きれない
* 译文：Vきる：全部……；……光／Vきれる：能全部……；能都……／Vきれない：……不了；无法全部……

::: tip :bookmark: 説明

「Vきれない」表示无法彻底地进行该动作。如果动作所涉及的客体为复数，表示该动作无法作用于全部客体。

:::
