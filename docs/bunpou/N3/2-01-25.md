# 被动态与被动句(2)〈物主被动句・间接被动句〉 <Badge type="warning" text="N3" />

## 物主被动句

**N1（所有者）が N2（人）に N3（所有物）を V（ら）れる**

物主被动句与主动句的关系：人が 所有者の所有物を Vる → 所有者が 人に 所有物を V（ら）れる

"所有物"可以是所有者身体的一部分或其拥有的物品，也可以是其某一侧面（如动作、姿势、状态等）。

::: info :books: 例句

1. <grammer-content sentence='[私/わたし]は[弟/おとうと]にパソコンを[壊/こわ]されました。' trans='我的电脑被弟弟弄坏了。' />
2. <grammer-content sentence='[私/わたし]は[電車/でんしゃ]の[中/なか]で[男/おとこ]の[人/ひと]に[足/あし]を[踏/ふ]まれました。' trans='我在电车上被男人踩了脚。' />
3. <grammer-content sentence='[姉/あね]は[妹/いもうと]に[大好/だいす]きなケーキを[食/た]べられて、[怒/おこ]っている。' trans='姐姐最喜欢的蛋糕被妹妹吃了，正在生气。' />

:::

## 间接被动句

**N1（人）が N2に（N3を）V（ら）れる**

间接被动句与主动句之间没有完全的对应关系，其主语名词在主动句中并未出现。间接被动句多表示某一事态的发生间接地给另一方（多为说话人）带来了不良的影响或损害。

### a. 自动词做谓语的间接被动句

::: info :books: 例句

1. <grammer-content sentence='[彼/かれ]は3[年前/ねんまえ]に[母親/ははおや]に[死/し]なれた。' trans='他3年前母亲去世了（对他造成了影响）。' />
2. <grammer-content sentence='[前/まえ]の[日/ひ]に、[隣/となり]の[部屋/へや]の[人/ひと]に[騒/さわ]がれて、よく[眠/ねむ]れなかった。' trans='前一天晚上被隔壁房间的人吵得没睡好。' />
3. <grammer-content sentence='[遠足/えんそく]の[日/ひ]に、[雨/あめ]に[降/ふ]られて[困/こま]りました。' trans='郊游那天被雨淋了，很困扰。' />
4. <grammer-content sentence='[昨日/きのう]の[夜/よる]、[子供/こども]に[泣/な]かれて、[目/め]が[覚/さ]めてしまいました。' trans='昨晚被孩子哭醒了。' />

:::

### b. 他动词做谓语的间接被动句

::: info :books: 例句

1. <grammer-content sentence='[私/わたし]たちは1[年生/ねんせい]に[運動場/うんどうじょう]を[占領/せんりょう]された。' trans='我们的操场被一年级学生占领了。' />
2. <grammer-content sentence='レストランで、[隣/となり]のテーブルの[人/ひと]にたばこを[吸/す]われて、[気分/きぶん]が[悪/わる]くなりました。' trans='在餐厅被邻桌的人抽烟弄得心情不好。' />

:::
