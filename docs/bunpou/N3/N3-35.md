# Nとなっている＜既定＞ <Badge type="warning" text="N3" />

* 意义：用于说明规定、计划、现状等。
* 接续：名词＋となっている
* 译文：规定（是）……；计划（是）……

::: info :books: 例句

1. <grammer-content sentence='[東大寺/とうだいじ]での[見学時間/けんがくじかん]は1[時間/じかん]と**なっております**。' trans='东大寺的参观时间规定是1小时。' />
2. <grammer-content sentence='[当駅/とうえき]のホームは[禁煙/きんえん]と**なっております**。' trans='本站的站台规定是禁烟的。' />
3. <grammer-content sentence='[国慶節/こっけいせつ]の[休/やす]みは、その[前後/ぜんご]の[約/やく]1[週間/しゅうかん]と**なっています**。' trans='国庆节的假期规定是前后约一周。' />
4. <grammer-content sentence='[辞書/じしょ]は[貸/か]し[出/だ]し[不可/ふか]と**なっております**ので、[館内/かんない]でご[覧/らん]ください。' trans='字典规定不能外借，请在馆内阅览。' />

:::
