# 〜ようだ〈推测〉 <Badge type="warning" text="N3" />

- 意义：表示说话人根据自己的印象、感觉、经验做出推测。
- 接续：动词、形容词的连体形 + ようだ；名词 + の + ようだ
- 译文：大概......吧；可能......吧；好像......

::: info :books: 说明

「〜ようだ」是根据自己的感觉或经验进行综合判断得出推测，「〜そうだ」是根据眼前观察到的状态或动向做出直观推测。

:::

::: info :books: 例句

1. <grammer-content sentence='[北京/ペキン]での[生活/せいかつ]を[話/はな]したら、[安心/あんしん]したようでした。' trans='说了北京的生活，他们好像挺放心的。' />
2. <grammer-content sentence='[体/たい]がだるい。[風邪/かぜ]を[引/ひ]いたようだ。' trans='身体乏力，好像感冒了。' />
3. <grammer-content sentence='[田中/たなか][先生/せんせい]はお[酒/さけ]がお好きなようですね。' trans='田中老师好像喜欢喝酒。' />
4. <grammer-content sentence='あの[歌手/かしゅ]は[最近/さいきん][人気/にんき]が[高/たか]いようですよ。' trans='那个歌手最近好像人气很高。' />
5. <grammer-content sentence='[通行/つうこう][止/ど]めになっている。[事故/じこ]のようだ。' trans='禁止通行了，好像出了事故。' />

:::
