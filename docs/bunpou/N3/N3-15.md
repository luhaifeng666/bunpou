# V（よ）うとする＜意図＞ <Badge type="warning" text="N3" />

* 意义：表示为了某一目的，努力尝试进行某动作。
* 接续：自主动词的意志形V（よ）う＋とする
* 译文：试图……；想要……

::: tip :bookmark: 説明

其否定形式为「V（よ）うとしない」。

通常「V（よ）うとしたとき」「V（よ）うとしたところ」的形式，表示正要实施该动作的时候。

当动词为非自主动词时，表示变化即将发生或结束。

:::
