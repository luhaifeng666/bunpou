# 疑问词 + Vたらいいか〈询问〉 <Badge type="warning" text="N3" />

- 意义：用于说话人向对方询问做某事的最佳方式、方法或时间、地点等。
- 接续：疑问词 + Vたらいいか
- 译文：怎样......オ好呢；如何......好呢

::: info :books: 说明

这个句式可以用于直接提问，也可用于陈述句的从句。

:::

::: info :books: 例句

1. <grammer-content sentence='[美容室/びようしつ]でどう[説明/せつめい]したらいいかわからなくて、アンダーソンの[写真/しゃしん]を[見/み]せたんですが……。' trans='不知道该怎么跟美容店的人说，就把安德森的照片给他看了……。' />
2. <grammer-content sentence='[電車/でんしゃ]の[中/なか]にかばんを[忘/わす]れてしまったんですが、どうしたらいいですか。' trans='我把包忘在电车上了，该怎么办好呢？' />
3. <grammer-content sentence='パソコンを[買/か]いたいんですが、どこで[買/か]ったらいいですか。' trans='我想买电脑，在哪里买好呢？' />
4. <grammer-content sentence='[明日/あした]、[何時/なんじ]に[出発/しゅっぱつ]したらいいか、[後/あと]で[連絡/れんらく]をください。' trans='明天几点出发好，之后请联系我。' />

:::
