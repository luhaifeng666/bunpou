# ものだ＜回顧＞ <Badge type="warning" text="N3" />

* 意义：用于感慨地回顾过去的某一习惯或经常做的事。
* 接续：Vた＋ものだ
* 译文：（那时候）经常……

::: info :books: 例句

1. <grammer-content sentence='[若/わか]いときは、よく[友達/ともだち]と[一緒/いっしょ]に[旅行/りょこう]に[行/い]った**ものだ**。' trans='年轻的时候，经常和朋友一起去旅行。' />
2. <grammer-content sentence='[子供/こども]の[頃/ころ]、よくこの[川/かわ]で[遊/あそ]んだ**ものだ**。' trans='小时候，经常在这条河里玩。' />
3. <grammer-content sentence='[昔/むかし]は、よく[夜遅/よるおそ]くまで[勉強/べんきょう]した**ものだ**。' trans='以前经常学习到很晚。' />
4. <grammer-content sentence='よく[元彼女/もとかのじょ]と[一緒/いっしょ]にそのレストランに[行/い]った**ものだ**。' trans='经常和前女友一起去那家餐厅。' />

:::
