# Vきる＜状态＞ <Badge type="warning" text="N3" />

* 意义：表示某种状态达到极限。
* 接续：动词第一连用形＋きる
* 译文：十分的……；非常……

::: tip :bookmark: 説明

部分动词搭配意思比较固定：わかりきる／十分了解；弱りきる／极度衰弱。

:::

::: info :books: 例句

1. <grammer-content sentence='[彼/かれ]のことを**信しんじきっています**。' trans='我十分信任他。' />
2. <grammer-content sentence='[今日/きょう]は[忙/いそが]しくて、もう**疲つかれきってしまった**。' trans='今天很忙，已经累得不行了。' />
3. <grammer-content sentence='[彼女/かのじょ]のわがままに**困こまりきっています**。' trans='她的任性让我非常为难。' />
:::
