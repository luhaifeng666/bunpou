# 〜ば〜ほど〈程度递进〉 <Badge type="warning" text="N3" />

- 意义：表示随着前句所述的动作、行为或性质状态的发展变化，主句所述的性质状态就越典型或随之发生相应的变化。
- 接续：
  - 动词：VばVるほど；サ变动词：词干すればするほど
  - 形容词：イAければイAいほど；ナAならナAなほど / ナAであればあるほど
  - 名词：NであればNであるほど / Nであればあるほど
- 译文：越......越......

::: info :books: 例句

1. <grammer-content sentence='[毎年/まいとし]この[時期/じき]、チケットの[予約/よやく]は[早/はや]ければ[早/はや]いほどいいんですよ。' trans='每年这个时候的机票都是越早订越好。' />
2. <grammer-content sentence='[機械/きかい]の[操作/そうさ]は、[簡単/かんたん]なら[簡単/かんたん]なほどいいと[思/おも]います。' trans='机器的操作，越简单越好。' />
3. <grammer-content sentence='[日本語/にほんご]は、[勉強/べんきょう]すればするほど[難/むずか]しくなります。' trans='日语，越学越难。' />
4. <grammer-content sentence='[聞/き]けば[聞/き]くほどおもしろい[話/はなし]です。' trans='越听越有趣的故事。' />
5. <grammer-content sentence='[厳/きび]しい[時代/じだい]であればあるほど、[自分/じぶん]を[磨/みが]けるのだ。' trans='时代越严峻，越能磨练自己。' />

:::
