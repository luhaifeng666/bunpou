# ～向き<本身的属性> <Badge type="warning" text="N3" />

* 意义：表示本来就拥有的特质，适于某方面。
* 接续：名词+向きのN/N向きだ
* 译文：适合……的

::: info :books: 例句

1. <grammer-content sentence='[彼女/かのじょ]は**[アニメ/あにめ]の[声優/せいゆう]向きの**[声/こえ]**をしている**。' trans='她有一副适合当动画配音演员的嗓音。' />
2. <grammer-content sentence='この[山/やま]は、[道/みち]が[狭/せま]くて[険/けわ]しいので**[経験/けいけん]者向きです**。' trans='这座山因为路窄且险，所以适合有经验的人。' />
3. <grammer-content sentence='この[服/ふく]は[かっこ/かっこと]いいけれど**[仕事/しごと]向きじゃない**[から/から][買/か]わないことにした**。' trans='这件衣服虽然很帅气但不适合工作穿，所以决定不买了。' />
4. <grammer-content sentence='女性は**[甘口/あまぐち]で**[日本人/にほんじん]向きです**。' trans='女性是甜口的，适合日本人。' />
:::
