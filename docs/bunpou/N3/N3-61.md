# ～っぽい<性质、倾向> <Badge type="warning" text="N3" />

* 意义：表示具有名词所示意义的那种感觉。接动词后面时，表示有某种倾向。易怒的。
* 接续：名词/动词第一连用形+っぽい
* 译文：有……倾向的；动不动就……；容易……

::: info :books: 例句

1. <grammer-content sentence='**[子供/こども]っぽい**[映画/えいが]**は[面白/おもしろ]くない。' trans='孩子气的电影没意思。' />
2. <grammer-content sentence='[一/いち][ヶ月/かげつ]も[留守/るす]にしていたので、[家/いえ]の[中/なか]が**[埃/ほこ]っぽい**。' trans='因为离开了一个月，家里都是灰。' />
3. <grammer-content sentence='**[忘/わす]れっぽい**[人/ひと]**は[不便/ふべん]だ。' trans='健忘的人不方便。' />
4. <grammer-content sentence='[彼/かれ]は**[怒/おこ]りっぽい**[人/ひと]**です。' trans='他是容易发火的人。' />
:::
