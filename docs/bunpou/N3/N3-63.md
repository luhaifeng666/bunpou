# ～Vかける<动作的阶段> <Badge type="warning" text="N3" />

* 意义：表示某动作刚刚开始或正在进行中尚未完成。也可以用「VかけのN」的形式。
* 接续：动词第一连用形+かける
* 译文：开始……；……了一半

::: info :books: 例句

1. <grammer-content sentence='**[言/い]いかけた[/か]**、[相手/あいて]が[先/さきに][話/はな]し[出/だ]した。' trans='刚要开口，对方就先说了。' />
2. <grammer-content sentence='**[手紙/てかみ]を[書/か]き[か]けた[ところ/ところ]**に、[電話/でんわ]が[かかって/かか]きた。' trans='正在写信的时候，电话响了。' />
3. <grammer-content sentence='**[読/よ]み[か]けの[小説/しょうせつ]**を[友達/ともだち]に[持/も]って[行/い]かれた。' trans='没看完的小说被朋友拿走了。' />
:::
