# それで～ん（の）だ＜結果＞ <Badge type="warning" text="N3" />

* 意义：表示说话人根据前面的内容推导出的结果。
* 接续：それで＋动词、形容词的连体形＋んだ
* 译文：所以才……

::: info :books: 例句

1. <grammer-content sentence='[王/おう]：[蒙古/もんごる]**といえば**、確かすもうがあったと[思/おも]いますけど。[マイク/まいく]：ああ、**それで**[蒙古/もんごる]の[力士/りきし]って[強/つよ]い**んだね**。' trans='王：说起蒙古，好像有相扑。迈克：啊，所以蒙古的力士很强啊。' />
2. <grammer-content sentence='Ａ：[陳/ちん]さんはご[両親/りょうしん]と[日本/にほん]で10[年間/ねんかん][暮/く]らしたそうですよ。Ｂ：**それで**[日本語/にほんご]が[上手/じょうず]な**んですね**。' trans='A：听说小陈和父母在日本生活了10年。B：所以才日语这么好。' />
3. <grammer-content sentence='Ａ：[今朝/けさ][寝坊/ねぼう]しちゃって、[起/お]きたらもう8[時/じ][過/す]ぎだったよ。Ｂ：ああ、**それで**[遅刻/ちこく]した**んだ**ね。' trans='A：今天睡过头了，起来的时候已经8点多了。B：啊，所以才迟到了。' />
4. <grammer-content sentence='[君/きみ]がとても[気持/きも]ちよさそうに[眠/ねむ]っていて、**それで**[起/お]こせなかった**んだ**。' trans='你睡得很香，所以才没能叫醒你。' />

:::
