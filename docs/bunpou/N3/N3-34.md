# まで＜極端的事例＞ <Badge type="warning" text="N3" />

* 意义：表示超出一般常识的极端事例。
* 接续：名词＋（に／で／と＋）まで
* 译文：（不仅……）甚至……；连……也……

::: tip :bookmark: 説明

凸显助词「まで」常常和前面从句中的「ばかりか・だけでなく」等形式呼应。

:::

::: info :books: 例句

1. <grammer-content sentence='[実際/じっさい]、[最近/さいきん]では[車内/しゃない]で[化粧/けしょう]をする[女性/じょせい]ばかりか、[床/ゆか]に[直接/ちょくせつ][座/すわ]り[込/こ]んでいる[若者/わかもの]や、カップめんを[食/た]べる[者/もの]**まで**[見/み]かけるようになった。' trans='实际上最近不仅是女性，甚至坐在地板上的年轻人和吃泡面的人都能看到了。' />
2. <grammer-content sentence='[最近/さいきん]は、[女性/じょせい]ばかりか[男性/だんせい]**まで**[化粧/けしょう]をするようになった**とか**。' trans='最近不仅是女性，甚至男性也开始化妆了。' />
3. <grammer-content sentence='[最近/さいきん]は[大人/おとな]だけでなく、[子供/こども]**まで**[体/からだ]を[動/うご]かさなくなっているようだ。' trans='最近不仅大人，连孩子都不怎么运动了。' />
4. <grammer-content sentence='[彼/かれ]は[一番/いちばん][信頼/しんらい]していた[友達/ともだち]に**まで**[裏切/うらぎ]られた。' trans='他甚至被最信任的朋友背叛了。' />

:::
