# Nを通して〈途径、手段〉 <Badge type="warning" text="N3" />

- 意义：表示利用某种途径或手段达到目的。
- 接续：名词 + を通して（名词 + を通じて）
- 译文：通过......

::: info :books: 例句

1. <grammer-content sentence='[動物/どうぶつ]を[通/とお]して[人間社会/にんげんしゃかい]を[反映/はんえい]したユーモアあふれる[作品/さくひん]。' trans='通过动物反映人类社会的幽默作品。' />
2. <grammer-content sentence='インターネットを[通/とお]して[必要/ひつよう]な[情報/じょうほう]を[入手/にゅうしゅ]する。' trans='通过互联网获取必要的信息。' />
3. <grammer-content sentence='アンケート[調査/ちょうさ]を[通/とお]して[大学生/だいがくせい]の[経済状態/けいざいじょうたい]がわかった。' trans='通过问卷调查了解了大学生的经济状况。' />
4. <grammer-content sentence='[今/いま]の[時代/じだい]、[私/わたし]たちは、SNSを[通/とお]して[世界中/せかいじゅう]の[人々/ひとびと]と[繋/つな]がっている。' trans='我们通过SNS与全世界的人联系。' />

:::
