# Vばいいじゃない？＜婉曲な提案＞ <Badge type="warning" text="N3" />

* 意义：用于说话人的委婉的建议，有劝说听话人同意自己的主张的含义。
* 接续：动词的条件形＋いい＋じゃない
* 译文：……不就行了；……就可以了

::: tip :bookmark: 説明

句尾读升调。

:::

::: info :books: 例句

1. <grammer-content sentence='[自分/じぶん]の[好/す]きな[力士/りきし]を[応援/おうえん]すれ**ばいいじゃない**。' trans='支持自己喜欢的力士不就行了。' />
2. <grammer-content sentence='[来/こ]たくなければ[断/ことわ]れ**ばいいじゃない**。' trans='不想来拒绝不就行了。' />
3. <grammer-content sentence='どうしても[行/い]きたいなら[行/い]け**ばいいじゃない**。' trans='如果一定要去的话，去不就行了。' />
4. <grammer-content sentence='Ａ：[電車/でんしゃ]がなくて[帰/かえ]れなかったら、うちに[泊/と]まれ**ばいいじゃない**。Ｂ：え、いいの？[助/たす]かる！' trans='A：如果没电车回不去了，住我家不就行了。B：诶，可以吗？得救了！' />

:::
