# Vるには＜目的＞ <Badge type="warning" text="N3" />

* 意义：表示目的。
* 接续：动词的词典形＋には
* 译文：（想）要……的话

::: tip :bookmark: 説明

`Vるには`基本上与`Vるために`同义，也可以使用`Vるためには`的形式。后句表示要达到该目的所需要的条件或采用的手段、方法、付出的努力等。

:::

::: info :books: 例句

1. <grammer-content sentence='[自分/じぶん]で[何/なに]かを[深/ふか]く[考/かんが]える**には**、ほんとうに[興味/きょうみ]があることじゃないと[難/むずか]しいです。' trans='要想自己深入思考什么的话，如果不是真正感兴趣的事情就很难。' />
2. <grammer-content sentence='この[町/まち]の[歴史/れきし]を[調/しら]べる**には**、この[本/ほん]が[参考/さんこう]になると[思/おも]います。' trans='要调查这个城市的历史的话，我觉得这本书有参考价值。' />
3. <grammer-content sentence='[日本/にほん]の[大学/だいがく]に[留学/りゅうがく]する**には**、どうしたらいいでしょうか。' trans='要想去日本的大学留学的话，该怎么做呢？' />
4. <grammer-content sentence='ダイエットを[成功/せいこう]させる**には**、[強/つよ]い[意志/いし]が[必要/ひつよう]です。' trans='要想减肥成功，需要有坚强的意志。' />
5. <grammer-content sentence='[日本語/にほんご]が[上手/じょうず]になる**には**、[一生懸命/いっしょうけんめい][勉強/べんきょう]しなければならない。' trans='要想日语变好，必须拼命学习。' />
6. <grammer-content sentence='ウイルスに[感染/かんせん]しない**には**、どんな[注意/ちゅうい]をしたらいいですか。' trans='要不感染病毒的话，应该注意什么？' />

:::
