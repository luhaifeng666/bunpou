# ～ような気がする＜感覚＞ <Badge type="warning" text="N3" />

* 意义：表示说话人的某种感觉、主观感受。
* 接续：动词、形容词的连体形＋ような気がする　名词＋の＋ような気がする
* 译文：觉得好像……；觉得似乎……

::: info :books: 例句

1. <grammer-content sentence='[周/まわ]りの[人/ひと]がみんな[頭/あたま]がよさそうで、ついていけない**ような気がします**。' trans='周围的人好像都很聪明，我觉得自己有些跟不上了。' />
2. <grammer-content sentence='あの[人/ひと]とどこかで[会/あ]った**ような気がする**。' trans='我觉得好像在哪里见过那个人。' />
3. <grammer-content sentence='そのお[金/かね]をもらったら、よくない**ような気がする**。' trans='如果拿了那笔钱，我觉得不太好。' />
4. <grammer-content sentence='[王/おう]さんは[高橋/たかはし]さんのことが[好/す]きな**ような気がする**。' trans='我觉得小王好像喜欢高桥。' />
5. <grammer-content sentence='こんな[経験/けいけん]は、[初/はじ]めての**ような気がする**。' trans='我觉得这种经验好像是第一次。' />

:::
