# ～てしかたがない＜極端の心理状態＞ <Badge type="warning" text="N3" />

* 意义：表示某种感情或感觉自然产生，难以控制。
* 接续：Vて／AⅠくて／AⅡで＋しかたがない
* 译文：……得不得了；……得很

::: tip :bookmark: 説明

该句式的主语一般为第一人称，形容词或动词大多表示感情、感觉或生理现象。

口语中也说：`～てしようがない`、`～てしょうがない`。

:::
