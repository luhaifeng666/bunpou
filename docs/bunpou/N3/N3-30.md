# ～ものだから＜理由＞ <Badge type="warning" text="N3" />

* 意义：表示理由、借口，用于委婉的说明。
* 接续：动词、形容词的连体形＋ものだから
* 译文：因为……

::: tip :bookmark: 説明

口语中也说「もんだから」。当主句是表示说话人的命令、推测等的句子时，通常不能用「ものだから」。

:::

::: info :books: 例句

1. <grammer-content sentence='やってみたかったというより、アメリカ人の[友達/ともだち]に[強引/ごういん]に[誘/さそ]われちゃった**ものですから**……。' trans='比起想试试，不如说是被美国朋友强行邀请的……' />
2. <grammer-content sentence='ミスが[多/おお]くてすみません。[最近/さいきん]ずっと[仕事/しごと]が[忙/いそが]しかった**ものですから**。' trans='错误很多对不起。因为最近工作一直很忙。' />
3. <grammer-content sentence='[不思議/ふしぎ]に[思/おも]った**ものですから**、ちょっとインターネットで[調/しら]べてみました。' trans='因为觉得不可思议，所以在网上查了一下。' />
4. <grammer-content sentence='[少/すこ]し[急/いそ]いで[歩/ある]いてきた**もんだから**、[汗/あせ]をかいちゃった。' trans='因为走得有点急，所以出汗了。' />
5. <grammer-content sentence='ゲームのしすぎで、[目/め]が[疲/つか]れた**ものだから**。' trans='因为游戏玩太多，眼睛累了。' />

:::
