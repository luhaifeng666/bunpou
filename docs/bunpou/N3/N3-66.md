# ～ことに<主观评价> <Badge type="warning" text="N3" />

* 意义：表示对后句所述事态的主观评价。一般用于句首。
* 接续：表示感情、心理活动的形容词的连体形、动词的「た形」+ことに
* 译文：（令人）……的是……

::: info :books: 例句

1. <grammer-content sentence='**[驚/おどろ]いたことに**[本人/ほにんん]は[昔/むかし]から[黒沢/くろざわ][作品/さくひん]が[好/す]きだったという。' trans='令人惊讶的是，据说本人从以前就喜欢黑泽的作品。' />
2. <grammer-content sentence='**[困/くるし]ったことに**[電波/でんぱ]の[状態/じょうたい]が[悪/わる]くて[電話/でんわ]がなかなか[かから/かから]ない。' trans='令人苦恼的是，信号不好，电话怎么也打不通。' />
3. <grammer-content sentence='**[嬉/られ]しいことに**[大学/だいがく]に[入/はい]って[一/いち]番**[やり/やり]たかった[活動/かつどう]に[参加/さんか]させてもらえることになった**。' trans='令人高兴的是，得以参加进大学后最想参加的活动。' />
:::
