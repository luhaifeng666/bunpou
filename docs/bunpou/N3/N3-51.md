# ～Vてばかりだ／Vてばかりいる＜限定状态＞ <Badge type="warning" text="N3" />

* 意义：表示事情反复发生或状态一直持续，说话人带有不满、批评、抱怨的语气。
* 接续：Vて＋ばかりだ／ばかりいる
* 译文：光……；总是……

::: info :books: 例句

1. <grammer-content sentence='[日本/にほん]に[来/き]たばかりのときは[戸惑/とまど]って**ばかりだった**な。' trans='刚来日本的时候，一直感到困惑。' />
2. <grammer-content sentence='[最近/さいきん]、[仕事/しごと]がうまくいかず[上司/じょうし]に[怒/おこ]られて**ばかりだ**。' trans='最近工作不顺，总是被上司骂。' />
3. <grammer-content sentence='[座/すわ]って**ばかりいないで**、ちゃんと[手伝/てつだ]ってください。' trans='别光坐着，好好帮忙。' />
4. <grammer-content sentence='[一日中/いちにちじゅう][寝/ね]て**ばかりいる**と[体/からだ]が[弱/よわ]ってしまうよ。' trans='整天光睡觉的话，身体会变弱的。' />
5. <grammer-content sentence='[大学/だいがく]に[入/はい]ってから[遊/あそ]んで**ばかりいて**、まじめに[勉強/べんきょう]していない。' trans='进了大学之后光玩，没有认真学习。' />
:::
