# ～とは限らない＜否定性可能＞ <Badge type="warning" text="N3" />

* 意义：表示事实未必如此、不能如此断定之意，暗示存在例外情况。
* 接续：简体句子＋とは限らない；Ⅱ类形容词词干／名词＋とは限らない
* 译文：不一定……；并非……；未必……

::: tip :bookmark: 説明

常与副词`必ずしも`搭配使用。

:::

::: info :books: 例句

1. <grammer-content sentence='でも、[面接/めんせつ]をしても、すぐにお[願/ねが]いする**とは限りません**が、よろしいですね。' trans='但是，即便面试了也不一定马上就能录用，可以吗？' />
2. <grammer-content sentence='[店/みせ]の[人/ひと]が[勧/すす]めるものがいいものだ**とは限らない**。' trans='店员推荐的东西不一定是好东西。' />
3. <grammer-content sentence='[値段/ねだん]が[高/たか]いものが、**必ずしも**[丈夫/じょうぶ]**とは限らない**。' trans='价格贵的东西不一定就结实。' />
4. <grammer-content sentence='[先生/せんせい]の[言/い]うことがいつも[正/ただ]しい**とは限らない**。' trans='老师说的话不一定总是对的。' />

:::
