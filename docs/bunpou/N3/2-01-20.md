# Nからなる〈构成〉 <Badge type="warning" text="N3" />

- 意义：表示事物的组成部分。
- 接续：名词 + からなる
- 译文：由......组成；由......构成

::: info :books: 例句

1. <grammer-content sentence='[前半/ぜんはん]が[人物戯画/じんぶつぎが]、[後半/こうはん]が[動物戯画/どうぶつぎが]からなる。' trans='前半部分是人物戏画，后半部分由动物戏画组成。' />
2. <grammer-content sentence='[水/みず]は[水素/すいそ]と[酸素/さんそ]からなる。' trans='水由氢和氧组成。' />
3. <grammer-content sentence='[卒業論文/そつぎょうろんぶん]は5つの[章/しょう]からなっている。' trans='毕业论文由五个章节构成。' />
4. <grammer-content sentence='チームは6[人/にん]のメンバーからなっている。' trans='团队由六名成员组成。' />

:::
