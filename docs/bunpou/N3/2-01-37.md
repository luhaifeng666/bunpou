# 〜のではないだろうか〈委婉的主张〉 <Badge type="warning" text="N3" />

- 意义：表示说话人的推断。
- 接续：动词、形容词的连体形 + のではないだろうか；名词 + な + のではないだろうか
- 译文：可能是......吧；是不是......呢

::: info :books: 说明

这个句式与「〜のではないか」意义一样，只不过更加委婉，含有向对方确认的语气。

:::

::: info :books: 例句

1. <grammer-content sentence='Uターン就職は[出身/しゅっしん][地域/ちいき]の[活性化/かっせいか]に[貢献/こうけん]するのではないだろうか。' trans='U turn 就业是不是会对搞活家乡的经济做出贡献呢。' />
2. <grammer-content sentence='この[本/ほん]は[一年生/いちねんせい]には[難/むずか]しいのではないでしょうか。' trans='这本书对一年级学生来说是不是太难了呢。' />
3. <grammer-content sentence='[連絡/れんらく]がないけれど、高橋さんは[病気/びょうき]なのではないでしょうか。' trans='虽然没有联系，但高桥先生是不是生病了呢。' />
4. <grammer-content sentence='[一生懸命/いっしょうけんめい][練習/れんしゅう]して、あそこまで[上手/じょうず]になったのだから、李さんは[弁論大会/べんろんたいかい]で[優勝/ゆうしょう]するのではないだろうか。' trans='因为拼命练习变得那么厉害，小李可能在辩论大会上获得冠军吧。' />

:::
