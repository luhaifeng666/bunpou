# ～わけだ＜説明＞ <Badge type="warning" text="N3" />

* 意义：用于说明，即陈述某一个事实，认为它是事情发展的必然结果，也可以表示对因果关系的认同。
* 接续：动词、形容词的连体形＋わけだ
* 译文：（当然）就……
