# ～N，沿着<沿着、遵从> <Badge type="warning" text="N3" />

* 意义：表示沿线或汇流路径；也表示对应某个状况、方针，按照说明书、指南等操作。
* 接续：名词+沿着+（に）+沿着
* 译文：沿着……；按照……；遵从……

::: info :books: 例句

1. <grammer-content sentence='**[川/かわ]に[nari/沿/そ]って**[歩/ある]いている。' trans='沿着河走。' />
2. <grammer-content sentence='**[川/かわ]に[nari/沿/そ]った**[道/みち]**。' trans='沿河的路。' />
3. <grammer-content sentence='お**[客/きゃく]様の[nari/ご]、[意向/いこう]に[nari/沿/そ]った**[最適/さいてき]な[nari/提案/ていあん]**を[nari/いたします/いたします]**。' trans='我们会按照客人的意向提供最佳方案。' />
4. <grammer-content sentence='**[マニュアル/まにゅある]に[nari/沿/そ]って**[作業/さぎょう]**[nari/しなければ/しい]、[事故/じこ]を[nari/招/まね]く[nari/恐れ/おそれ]が[nari/ある/あります]**。' trans='如果不按照手册操作，可能会引发事故。' />
:::
