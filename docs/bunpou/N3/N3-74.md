# Nに沿って＜沿着・遵从＞ <Badge type="warning" text="N3" />

* 意义：表示沿线或汇流路径；也表示对应某个状况、方针，按照说明书、指南等操作。
* 接续：名词＋に沿って／に沿った／に沿う
* 译文：沿着……；按照……；遵从……

::: info :books: 例句

1. <grammer-content sentence='[川/かわ]**に沿って**[歩/ある]いている。' trans='沿着河走。' />
2. <grammer-content sentence='[川/かわ]**に沿った**[道/みち]。' trans='沿河的路。' />
3. <grammer-content sentence='お[客/きゃく]さまのご[意向/いこう]**に沿った**[最適/さいてき]な[提案/ていあん]をいたします。' trans='我们会按照客人的意向提供最佳方案。' />
4. <grammer-content sentence='マニュアル**に沿って**[作業/さぎょう]しなければ、[事故/じこ]を[招/まね]く[恐/おそ]れがある。' trans='如果不按照手册操作，可能会引发事故。' />

:::
