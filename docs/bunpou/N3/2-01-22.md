# お/ご V ください〈请求〉 <Badge type="warning" text="N3" />

- 意义：表示请求对方做某事，是一种尊他表现形式。
- 接续：お + 和语动词的第一连用形 + ください；ご + 汉语动词词干 + ください
- 译文：请您......

::: info :books: 例句

1. <grammer-content sentence='どうぞ、おかけください。' trans='请坐。' />
2. <grammer-content sentence='[夕食/ゆうしょく]は[京料理/きょうりょうり]をお[楽/たの]しみください。' trans='晚餐请享受京都料理。' />
3. <grammer-content sentence='[来週/らいしゅう]、[遣唐使/けんとうし]の[会/かい]があります。みなさん、ぜひご[参加/さんか]ください。' trans='下周有遣唐使之会，请大家务必参加。' />
4. <grammer-content sentence='もう[大丈夫/だいじょうぶ]ですので、どうぞご[安心/あんしん]ください。' trans='已经没问题了，请放心。' />

:::
