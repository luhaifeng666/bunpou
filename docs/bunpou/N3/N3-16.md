# Vている／Vるうちに＜変化が起こる時間範囲＞ <Badge type="warning" text="N3" />

* 意义：表示某一状态时或某一动作（反复）进行过程中发生了某一变化。
* 接续：动词的持续体或词典形＋うちに
* 译文：在……过程中；……着……着，就……

::: info :books: 例句

1. <grammer-content sentence='[何回/なんかい]か[会/あ]っている**うちに**、[相互学習/そうごがくしゅう]を[始/はじ]めることになった。' trans='见了几次面之后，就开始相互学习了。' />
2. <grammer-content sentence='ゆうべ、[部屋/へや]で[小説/しょうせつ]を[読/よ]んでいる**うちに**、[眠/ねむ]ってしまいました。' trans='昨晚在房间里看小说，看着看着就睡着了。' />
3. <grammer-content sentence='テレビで[日本/にほん]のドラマを[見/み]ている**うちに**、[日本語/にほんご]に[興味/きょうみ]を[持/も]つようになった。' trans='看着日本电视剧，就对日语产生了兴趣。' />
4. <grammer-content sentence='[日本人/にほんじん]と[話/はな]す**うちに**、[日本語/にほんご]の[発音/はつおん]がよくなった。' trans='和日本人说话的过程中，日语发音变好了。' />

:::
