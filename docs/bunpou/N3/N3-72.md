# Vかねる＜難以＞ <Badge type="warning" text="N3" />

* 意义：表示说话人觉得某件事情难以做到。多用于委婉地拒绝对方的愿望和命令。一般拒绝的理由多是心理方面的原因，比「できない」委婉。
* 接续：动词第一连用形＋かねる
* 译文：难以……；很难……

::: info :books: 例句

1. <grammer-content sentence='[確証/かくしょう]が[無/な]いので[判断/はんだん]し**かねます**。' trans='没有确凿的证据，难以判断。' />
2. <grammer-content sentence='[自己責任/じこせきにん]で[行/い]ってください。こちらで[責任/せきにん]は[負/お]い**かねます**。' trans='请自行负责。我方难以承担责任。' />
3. <grammer-content sentence='[専門家/せんもんか]ではないので[適切/てきせつ]なアドバイスは**しかねます**。' trans='我不是专家，很难给出恰当的建议。' />
4. <grammer-content sentence='ご[意見/いけん]は[承/うけたまわ]りましたが、[同意/どうい]し**かねます**。' trans='您的意见我收到了，但难以苟同。' />

:::
