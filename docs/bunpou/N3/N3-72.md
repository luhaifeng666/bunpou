# Vかねる<难以> <Badge type="warning" text="N3" />

* 意义：表示说话人觉得某件事情难以做到。多用于委婉地拒绝对方的愿望和命令。
* 接续：动词第一连用形+かねる
* 译文：难以……；很难……

::: info :books: 例句

1. <grammer-content sentence='**[確証/かくしょう]が[nari/無/な]いので[nari/判断/はんだん]し[nari/かねます]**。' trans='没有确凿的证据，难以判断。' />
2. <grammer-content sentence='**[専門家/せんもんか]不是很好[nari/なので/由於]**[適切/てきせつ]な[nari/アドバイス/あどばいす]は[nari/し/nari/しかねます]**。' trans='我不是专家，很难给出恰当的建议。' />
3. <grammer-content sentence='**[ご/ご][意見/いけん]は[nari/承/うけたまわ]りましたが、**[同意/どうい]し[nari/かねます]**。' trans='您的意见我收到了，但难以苟同。' />
:::
