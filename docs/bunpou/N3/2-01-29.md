# V（ら）れた / Vることができた / Nできた〈最终实现〉 <Badge type="warning" text="N3" />

- 意义：表示动作主体期望的事情最终得以实现或目标达成，含有难得、不易之意。
- 接续：
  - 动词的能动态V（ら）れる + た
  - 动词的词典形 + ことができた
  - 动作性名词 + できた
- 译文：终于......成了；总算......

::: info :books: 说明

经常与副词「やっと」「ようやく」等搭配使用。

:::

::: info :books: 例句

1. <grammer-content sentence='キャンセルが出たみたいで、[昨日/きのう]やっと[予約/よやく]できたんです。' trans='好像有人取消了，昨天终于预订上了。' />
2. <grammer-content sentence='3回目でやっと[試験/しけん]に[合格/ごうかく]できた。' trans='第三次终于通过考试了。' />
3. <grammer-content sentence='[昨日/きのう][久/ひさ]しぶりに8[時間/じかん][寝/ね]ることができた。' trans='昨天好不容易睡了8小时。' />
4. <grammer-content sentence='ずっと欲しかった[本/ほん]が[買/か]えたので[嬉/うれ]しい。' trans='一直想要的书终于买到了，很开心。' />

:::
