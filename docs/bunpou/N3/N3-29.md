# ～わけじゃない＜否定＞ <Badge type="warning" text="N3" />

* 意义：表示对前述内容进行否定，强调事实并非如此。
* 接续：动词、形容词的连体形＋わけじゃない／ではない
* 译文：并非……

::: info :books: 例句

1. <grammer-content sentence='[見/み]られてるって[意識/いしき]がない**わけじゃない**と[思/おも]うけど。' trans='我觉得并不是没有被人看着的意识。' />
2. <grammer-content sentence='[両親/りょうしん]の[反対/はんたい]の[気持/きも]ちが[分/わ]からない**わけではない**が、[自分/じぶん]の[夢/ゆめ]は[諦/あきら]めたくない。' trans='并不是不理解父母的反对心情，但不想放弃自己的梦想。' />
3. <grammer-content sentence='やりたい[仕事/しごと]がいつもできる**わけじゃない**から、このチャンスを[大切/たいせつ]にしたいと[思/おも]う。' trans='想做的工作并不总能做，所以想珍惜这个机会。' />
4. <grammer-content sentence='[彼/かれ]の[言/い]うことをすべて[信/しん]じている**わけではない**。' trans='并不是相信他说的一切。' />
5. <grammer-content sentence='[子供/こども]は[嫌/きら]いな**わけじゃない**が、ほしくないという[若者夫婦/わかものふうふ]が[増/ふ]えている。' trans='并不是讨厌孩子，但不想要孩子的年轻夫妇在增加。' />

:::
