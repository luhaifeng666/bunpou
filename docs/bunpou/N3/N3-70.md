# ～恐れがある＜可能性＞ <Badge type="warning" text="N3" />

* 意义：用于预测可能会发生不好的事情。书面语，一般为基于客观情况的推测。「～恐れもある」添加强调的含义。
* 接续：名词＋の／动词的辞典形或否定形＋恐れがある
* 译文：有可能……；令人担心会……

::: info :books: 例句

1. <grammer-content sentence='[原語/げんご]の[意味/いみ]との[ずれ/ずれ]を[意識/いしき]しないまま[使/つか]ってしまえば、ミスコミュニケーションを[起/お]こす**恐れもある**。' trans='如果没有意识到和原词意思的差异就这么使用，可能会引发交流错误。' />
2. <grammer-content sentence='[少子化/しょうしか]は[地域/ちいき]の[将来/しょうらい]にマイナスの[影響/えいきょう]を[与/あた]える**恐れがある**。' trans='少子化可能会对地区的未来产生负面影响。' />
3. <grammer-content sentence='[急/いそ]がないと、[期日/きじつ]に[間/ま]に[合/あ]わない**恐れがある**。' trans='不快点的话可能会赶不上截止日期。' />
4. <grammer-content sentence='この[地震/じしん]による[津波/つなみ]の**恐れはありません**。' trans='此次地震没有海啸的危险。' />
