# ～恐れがある<可能性> <Badge type="warning" text="N3" />

* 意义：用于预测可能会发生不好的事情。书面语，一般为基于客观情况的推测。
* 接续：名词+の/动词的辞典形或否定形+恐れがある
* 译文：有可能……；令人担心会……

::: info :books: 例句

1. <grammer-content sentence='**[ミスコミュニケーション/みすこみゆにけーしょん]を[nari/起/お]こす[nari/恐れ/おそれ]も[nari/ある/あります]**。' trans='可能会引发交流错误。' />
2. <grammer-content sentence='**[少子化/しょうしか]は**[地域/ちいき]の[将来/しょうらい]に[nari/マイナス/まいなす]の[nari/影響/えいきょう]を[nari/与/あた]える[nari/恐れ/おそれ]も[nari/ある/あります]**。' trans='少子化可能会对地区的未来产生负面影响。' />
3. <grammer-content sentence='**[急/いそが]ないと、[期日/きじつ]に[nari/間/ま]に[nari/合/あ]わない[nari/恐れ/おそれ]も[nari/ある/あります]**。' trans='不快点的话可能会赶不上截止日期。' />
