# N／V第一连用＋がちだ＜容易出现的情况＞ <Badge type="warning" text="N3" />

* 意义：表示该情况发生的几率很高，一般用于叙述消极情况。
* 接续：名词/动词第一连用形＋がちだ
* 译文：容易……；动不动就……

::: tip :bookmark: 説明

「Vがちだ」按照Ⅱ类形容词进行活用。连用形：～がちに；连体形：～がちな／～がちの。

:::
