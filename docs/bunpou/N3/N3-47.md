# Nに関する＜内容＞ <Badge type="warning" text="N3" />

* 意义：表示与名词所指称的事物有关的内容。
* 接续：名词＋に関する／に関して
* 译文：关于……

::: info :books: 例句

1. <grammer-content sentence='この[問題/もんだい]**に関する**[報告書/ほうこくしょ]を[書/か]いてください。' trans='请写一份关于这个问题的报告。' />
2. <grammer-content sentence='[環境問題/かんきょうもんだい]**に関して**、[議論/ぎろん]が[行/おこな]われた。' trans='关于环境问题进行了讨论。' />
3. <grammer-content sentence='[事件/じけん]**に関する**[詳細/しょうさい]はまだ[分/わ]からない。' trans='关于事件的详情还不清楚。' />

:::
