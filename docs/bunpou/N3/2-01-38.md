# VたN〈连体修饰〉 <Badge type="warning" text="N3" />

- 意义：表示状态的「Vている」形式在做连体修饰语修饰名词时，经常变为「Vた」之后再接名词。

::: info :books: 例句

1. <grammer-content sentence='ゆったりした[生活/せいかつ]を[懐/なつ]かしく[思/おも]い、[出身地/しゅっしんち]へのUターンを[考え/かんがえ]始めるのかもしれません。' trans='怀念悠闲的生活，可能会开始考虑回乡就业。' />
2. <grammer-content sentence='[晴/は]れた[日/ひ]が[続/つづ]いていて[夏/なつ]らしくなってきました。' trans='持续晴天，变得像夏天一样了。' />
3. <grammer-content sentence='[退職/たいしょく]したら[田舎/いなか]でのんびりした[生活/せいかつ]を[送/おく]りたい。' trans='退休后想在乡下过着悠闲的生活。' />
4. <grammer-content sentence='[妹/いもうと]はどちらかと言うと父に[似/に]た[性格/せいかく]だと[思/おも]う。' trans='妹妹的性格好像更像爸爸。' />

:::
