# で＜動作主体＞ <Badge type="warning" text="N3" />

* 意义：表示动作的主体。
* 接续：表示组织、机构、团体等名词＋で
* 译文：……（做的）；由……

::: tip :bookmark: 説明

格助词`で`接在表示组织、机构、团体等名词的后面，可以表示动作的主体。谓语动词为自主动词。

:::

::: info :books: 例句

1. <grammer-content sentence='[実/じつ]は、[山田/やまだ]さんからそちらの[会社/かいしゃ]**で**[翻訳/ほんやく]ができる[人/ひと]を[探/さが]していらっしゃるとお[聞/き]きしたんですが……。' trans='其实，我听山田说贵公司正在找能翻译的人……' />
2. <grammer-content sentence='[食事会/しょくじかい]の[費用/ひよう]は[会社/かいしゃ]**で**[払/はら]ってくれる。' trans='聚餐的费用由公司支付。' />
3. <grammer-content sentence='[詳/くわ]しい[原因/げんいん]については[警察/けいさつ]**で**[調/しら]べている。' trans='关于详细原因，警察正在调查。' />
4. <grammer-content sentence='[問題解決/もんだいかいけつ]のため、[大学/だいがく]**で**[対策/たいさく]を[検討/けんとう]している。' trans='为了解决问题，大学正在讨论对策。' />

:::
