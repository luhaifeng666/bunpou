# ～ものか＜否定＞ <Badge type="warning" text="N3" />

* 意义：表示强烈的否定意愿。
* 接续：动词、形容词连体形＋ものか
* 译文：谁会……啊；不会……

::: tip :bookmark: 説明

「ものか」多用于口语，表示说话人强烈的否定或拒绝。口语中常说「もんか」。

:::

::: info :books: 例句

1. <grammer-content sentence='そんなことできる**ものか**。' trans='谁能做那样的事啊。' />
2. <grammer-content sentence='[彼/かれ]の[言葉/ことば]を[信/しん]じる**ものか**。' trans='我才不会相信他的话呢。' />
3. <grammer-content sentence='[こんな/こんな][仕事/しごと]、[続/つづ]ける**ものか**。' trans='这样的工作，我才不会继续做呢。' />
4. <grammer-content sentence='[絶対/ぜったい]に[負/ま]ける**ものか**。' trans='绝对不会输。' />
5. <grammer-content sentence='[誰/だれ]が[そんな][馬鹿/ばか]なことをする**もんか**。' trans='谁会做那么蠢的事啊。' />

:::
