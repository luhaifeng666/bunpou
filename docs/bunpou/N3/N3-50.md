# ～くせに＜转折＞ <Badge type="warning" text="N3" />

* 意义：构成表示转折关系的从句，主句表示说话人的不满、责备、轻蔑等语气。（不会对自己用）
* 接续：名词+の/动词、形容词的连体形+くせに
* 译文：明明……，却……

::: tip :bookmark: 説明

从句与主句必须是同一主语。「くせに」不用于表述事物，也不用于说话人自己的行为。比「のに」责备语气更强。

:::

::: info :books: 例句

1. <grammer-content sentence='[誰/だれ]の[好意/こうい]にも[断/ことわ]らない**くせに**、「[自分/じぶん]は[浮気/うわき]しない」なんて[言/い]ってる！' trans='明明不拒绝任何人的好意，却说「我不会出轨」！' />
2. <grammer-content sentence='もらうときはものすごく[遠慮/えんりょ]する**くせに**、もらったあとはいつまでもほめるのよね。' trans='收礼物的时候明明非常客气，收到之后就一个劲地夸。' />
3. <grammer-content sentence='[お金/かね]がない**くせに**、[毎日/まいにち]たくさん[高/たか]い[買/か]い物をしている。' trans='明明没钱，每天却买很多贵的东西。' />
4. <grammer-content sentence='[働/はたら]かない**くせに**、[文句/もんく]ばかり[言/い]っている。' trans='明明不工作，却整天抱怨。' />
5. <grammer-content sentence='[食/た]べることは[大好/だいす]きな**くせに**、なぜか[料理/りょうり]は[全然/ぜんぜん]できないんだ。' trans='明明很喜欢吃，不知为什么完全不会做饭。' />
:::
