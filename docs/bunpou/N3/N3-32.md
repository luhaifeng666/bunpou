# ～ということだ＜間接引語＞ <Badge type="warning" text="N3" />

* 意义：表示间接引语，说明该句所传达的信息是从别人那里获得的。
* 接续：简体句子＋ということだ
* 译文：据说……

::: tip :bookmark: 説明

较之「そうだ」，它更多用于书面语。也可以用「とのことだ」「という」的形式。

:::

::: info :books: 例句

1. <grammer-content sentence='[彼女/かのじょ]**ということだ**。' trans='据说她就是因为对见面的人感到羞涩和敬意才化妆的。' />
2. <grammer-content sentence='[阿国/おくに]という[女性/じょせい]が[男性/だんせい]の[格好/かっこう]をして、ストーリーのない[踊/おど]りを[踊/おど]ったのが[歌舞伎/かぶき]の[最初/さいしょ]である**ということだ**。' trans='据说阿国这个女性穿着男性的装扮，跳没有故事的舞蹈是歌舞伎的起源。' />
3. <grammer-content sentence='[日本/にほん]では、そばを[食/た]べるとき[音/おと]を[立/た]てて[食/た]べてもいい**ということだ**。' trans='据说在日本，吃荞麦面时发出声音也没关系。' />

:::
