# ～けれども＜単純接続＞ <Badge type="warning" text="N3" />

* 意义：连接不具有转折意义的前后句，有时前面的分句可以起到提出话题的作用。
* 接续：句子＋けれども
* 译文：虽然……；……

::: info :books: 例句

1. <grammer-content sentence='[中村/なかむら]さんは[男/おとこ]の[人/ひと]です**けれども**、[女性/じょせい]のような[話/はな]し[方/かた]をします。' trans='中村先生虽然是男性，但说话方式像女性。' />
2. <grammer-content sentence='[明日/あした][都合/つごう]がいいです**けれども**、[何/なに]かありますか。' trans='明天方便，有什么事吗？' />
3. <grammer-content sentence='[日本語/にほんご]は[面白/おもしろ]いです**けれども**、[難/むずか]しいです。' trans='日语很有趣，但很难。' />

:::
