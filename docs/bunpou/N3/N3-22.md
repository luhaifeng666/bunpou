# それより（も）＜递进・话题转换＞ <Badge type="warning" text="N3" />

* 意义：通过比较的方式凸显后句所述的内容更加重要。也用于转换话题。
* 接续：句子＋それより（も）
* 译文：与此相比，……；话说，……

::: info :books: 例句

1. <grammer-content sentence='[私/わたし]がやったのは[中学校/ちゅうがっこう]のときで、もう[忘/わす]れてしまいました。**それよりも**マイクさん、[空手/からて]の[型/かた]をぜひ[見/み]せてください。' trans='我做的时候还是初中，已经忘了。话说，迈克，请一定要给我看看空手道的型。' />
2. <grammer-content sentence='その[話/はなし]は[前/まえ]に[聞/き]いたよ。**それよりも**[今度/こんど]の[困/こま]り[事/ごと]って[何/なに]？' trans='那件事之前听过了。话说，这次的麻烦事是什么？' />
3. <grammer-content sentence='[試合/しあい]に[勝/か]ってよかった。**それよりも**[選手/せんしゅ]たちの[一生懸命/いっしょうけんめい]な[姿/すがた]に[深/ふか]く[感動/かんどう]した。' trans='比赛赢了真好。但更让人感动的是选手们拼命的样子。' />
4. <grammer-content sentence='[科学技術/かがくぎじゅつ]の[発展/はってん]は[大切/たいせつ]だ。しかし、**それよりも**[教育/きょういく]を[充実/じゅうじつ]させるべきだ。' trans='科学技术的发展很重要。但更重要的是充实教育。' />

:::
