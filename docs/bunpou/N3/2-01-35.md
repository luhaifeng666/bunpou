# 使动被动态与使动被动句 <Badge type="warning" text="N3" />

- 意义：在他人的强迫或命令下，不得不去做不情愿的事情。
- 译文：被迫；不得不
- 接续：先变成使动态「～せる / させる」，然后在使动态的第一连用形后面接上表示被动的后缀「られる」

::: info :books: 说明

动词类别变化规则：
- I类动词：買う → 買わせる → 買わせられる / 買わされる
- II类动词：出る → 出させる → 出させられる
- III类动词：来る → 来させる → 来させられる

:::
