# 被动态与被动句(1)〈直接被动句〉 <Badge type="warning" text="N3" />

- 意义：表示某动作被（某人）完成。
- 接续：
  - I类动词：词尾「う」段假名 → 「あ」段假名 + 「れる」（書く → 書かれる）
  - II类动词：词尾「る」+ 「られる」（見る → 見られる）
  - III类动词：来る → 来られる；する → される
- 译文：被......；受到......；遭到......

::: info :books: 说明

根据句子意义和结构特点，日语的被动句可以分为三类：（1）直接被动句；（2）物主被动句；（3）间接被动句。直接被动句与主动句具有明确的对应关系，其主语是主动句中的"を格"或"に格"名词。

:::

### 直接被动句 No.1

**N1（人）が N2（人）に V（ら）れる**

最典型的被动句，与主动句「N2（人）が N1（人）を/に Vる」对应，二者可以自由转换。

::: info :books: 例句

1. <grammer-content sentence='[子供/こども]が[母親/ははおや]に[叱/しか]られる。' trans='孩子被母亲训斥。' />
2. <grammer-content sentence='[息子/むすこ]がクラスメートに[苛/いじ]められる。' trans='儿子被同学欺负。' />
3. <grammer-content sentence='[孫/まご]がおばあさんに[育/そだ]てられる。' trans='孙子被奶奶抚养。' />
4. <grammer-content sentence='（[私/わたし]は）[高橋/たかはし]さんに[招待/しょうたい]された。' trans='我被高桥先生招待了。' />

:::

### 直接被动句 No.2

**N1（人）が N2（人）に/から N3を V（ら）れる**

当谓语动词为表示语言行为或感情、态度的他动词时，往往句中会增加一个表行为、态度对象的"を格"名词。

::: info :books: 例句

1. <grammer-content sentence='[私/わたし]は[上司/じょうし]に[仕事/しごと]を[頼/たの]まれた。' trans='我被上司拜托了工作。' />
2. <grammer-content sentence='[私/わたし]は[先生/せんせい]にいろいろなことを[聞/き]かれた。' trans='我被老师问了各种各样的事情。' />

:::

### 直接被动句 No.3

**N1（物・こと）が N2（人）に V（ら）れる**

主语为表示物体或事件的名词，多用于客观地描述某一现象。

::: info :books: 例句

1. <grammer-content sentence='この[本/ほん]は[多/おお]くの[人/ひと]に[読/よ]まれている。' trans='这本书被很多人读。' />
2. <grammer-content sentence='[緑茶/りょくちゃ]は[中国/ちゅうごく]でよく[飲/の]まれている。' trans='绿茶在中国经常被喝。' />
3. <grammer-content sentence='2022[年/ねん]の[冬季/とうき]オリンピックは[北京/ペキン]で[開/ひら]かれました。' trans='2022年冬季奥运会在北京召开。' />

:::

### 直接被动句 No.4

**N1（物・こと）が N2（人）によって V（ら）れる**

主语为表示物体或事件的名词，谓语是表示发明、创造或发现一类的动词（如「作る、発明する、設計する、書く」等），此时动作的主体一般不用"に格"表示，而要用「によって」。

::: info :books: 例句

1. <grammer-content sentence='この[製品/せいひん]はイギリス[人/じん]のデザイナーによって[作/つく]られた。' trans='这个产品是由英国设计师制作的。' />
2. <grammer-content sentence='この[建物/たてもの]は[有名/ゆうめい]な[建築家/けんちくか]によって[設計/せっけい]されました。' trans='这个建筑物是由著名的建筑师设计的。' />
3. <grammer-content sentence='[秦/しん]は[始皇帝/しこうてい]によって[統一/とういつ]されました。' trans='秦朝被始皇帝统一了。' />

:::
