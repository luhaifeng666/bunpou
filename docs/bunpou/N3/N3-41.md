# ～なり＜二者選一＞ <Badge type="warning" text="N3" />

* 意义：表示从二者中选择一个（通常可供选择的对象不仅限于这两个事物）。
* 接续：动词词典形＋なり
* 译文：……或者……

::: info :books: 例句

1. <grammer-content sentence='[困/こま]ったときは、[親/おや]に[相談/そうだん]する**なり**、[友達/ともだち]に[相談/そうだん]する**なり**してください。' trans='遇到困难的时候，请找父母商量或者找朋友商量。' />
2. <grammer-content sentence='[暇/ひま]なときは、[本/ほん]を[読/よ]む**なり**、[音楽/おんがく]を[聞/き]く**なり**してください。' trans='空闲的时候，请看书或者听音乐。' />
3. <grammer-content sentence='[電話/でんわ]をかける**なり**、メールを[送/おく]る**なり**して、[連絡/れんらく]してください。' trans='请打电话或者发邮件联系。' />

:::
