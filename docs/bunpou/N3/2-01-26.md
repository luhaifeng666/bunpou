# 〜らしい〈传闻、推测〉 <Badge type="warning" text="N3" />

- 意义：表示传闻或有客观依据的推测。
- 接续：简体句子 + らしい；II类形容词词干 / 名词 + らしい
- 译文：据说......；听说......；似乎......

::: info :books: 例句

1. <grammer-content sentence='あまりうまくいかないらしいですよ。' trans='好像不太顺利。' />
2. <grammer-content sentence='あの[映画/えいが]はおもしろいらしいので、ぜひ[見/み]てみたいです。' trans='听说那部电影很有趣，所以想看。' />
3. <grammer-content sentence='[先週/せんしゅう][渡辺/わたなべ]さんが[買/か]った[辞書/じしょ]はとても[便利/べんり]らしいですよ。' trans='听说上周渡边买的字典很方便。' />
4. <grammer-content sentence='[集/あつ]まった[学生/がくせい]は1[年生/ねんせい]だけらしいです。' trans='聚集的学生似乎只有一年级。' />
5. <grammer-content sentence='[電気/でんき]が[消/き]えているから、[王/おう]さんはもう[寝/ね]たらしい。' trans='灯灭了，王先生好像已经睡了。' />

:::
