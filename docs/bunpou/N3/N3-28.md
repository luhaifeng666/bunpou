# よく～＜評価＞ <Badge type="warning" text="N3" />

* 意义：表示被评价者的所作所为超出一般程度或评价者的想象。
* 接续：よく＋句子
* 译文：竟然……；居然……

::: tip :bookmark: 説明

多用于感叹句，句尾一般使用语气助词进行呼应。用于贬义时，也可使用「よくも」的形式。

:::

::: info :books: 例句

1. <grammer-content sentence='[人目/ひとめ]もかまわず**よく**お[化粧/けしょう]できるなって[私/わたし]も[思/おも]うけど。' trans='不管别人的目光，居然能这样化妆，我也这么觉得。' />
2. <grammer-content sentence='テレビを[見/み]ながら、**よく**レポートが[書/か]けますね。' trans='一边看电视，居然还能写报告。' />
3. <grammer-content sentence='1[日/にち]で、**よく**こんなにたくさんの[資料/しりょう]を[集/あつ]めたなと[感心/かんしん]した。' trans='一天就收集了这么多资料，让人佩服。' />
4. <grammer-content sentence='**よくも**そんなひどいことが[言/い]えるね。' trans='竟然能说出这么过分的话。' />
5. <grammer-content sentence='**よくも**[私/わたし]にそんなことがしたな。' trans='居然对我做出这种事。' />

:::
