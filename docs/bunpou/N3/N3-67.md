# ～に決まっている<确信、断定> <Badge type="warning" text="N3" />

* 意义：表示说话人对某事的确信、断定。
* 接续：名词/II类形容词词干/动词、I类形容词连体形+に決まっている
* 译文：一定……；准是……；肯定……

::: info :books: 例句

1. <grammer-content sentence='**[言/い]おうものなら**[怒/おこ]り[出/だ]す**に[決/き]まっている**。' trans='如果说的话，对方一定会生气的。' />
2. <grammer-content sentence='この[試合/しあい]**A[チーム/ちーむ]が[勝/か]つ[に決まっている]**から、[最後/さいご]まで[見/み]なくてもいいよ。' trans='这场比赛A队一定会赢，不用看到最后也可以。' />
3. <grammer-content sentence='あそこに[行/い]くなら、**[タクシー/たくしー]の[方/ほう]が[便利/べんり][に決まっている]**。' trans='去那里的话，出租车一定更方便。' />
:::
