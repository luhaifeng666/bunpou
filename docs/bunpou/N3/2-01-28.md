# 〜みたいだ〈推测〉 <Badge type="warning" text="N3" />

- 意义：表示说话人根据实际体验进行的推测。
- 接续：简体句子 + みたいだ；II类形容词词干 / 名词 + みたいだ
- 译文：好像......；似乎......

::: info :books: 说明

一般用于口语。书面语或正式场合则使用「〜ようだ」。

:::

::: info :books: 例句

1. <grammer-content sentence='キャンセルが出たみたいで、[昨日/きのう]やっと[予約/よやく]できたんです。' trans='好像有人取消了，昨天好不容易预订上了。' />
2. <grammer-content sentence='李さんは[最近/さいきん][忙/いそが]しいみたいだね。' trans='小李最近好像很忙。' />
3. <grammer-content sentence='あの[二人/ふたり]は[付/つ]き[合/あ]っているみたいだよ。' trans='那两个人好像正在交往。' />
4. <grammer-content sentence='高橋さんの[風邪/かぜ]はもう[大丈夫/だいじょうぶ]みたいだ。' trans='高桥的风好像已经好了。' />
5. <grammer-content sentence='この[小説/しょうせつ]の[作者/さくしゃ]は[女性/じょせい]みたいですね。' trans='这个小说的作者好像是女性。' />

:::
