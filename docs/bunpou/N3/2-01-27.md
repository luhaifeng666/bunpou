# Vるところだ / Vているところだ / Vたところだ / Vていたところだ〈动作阶段与说话时点的关系〉 <Badge type="warning" text="N3" />

- 意义：表示动作所处的时点。
- 译文：
  - Vるところだ：现在正要......；刚要......
  - Vているところだ：现在正在......
  - Vたところだ：刚刚......完
  - Vていたところだ：一直在......

### a. Vるところだ

表示该动作行为在说话时即将开始进行，经常与「今、ちょうど、これから」等时间副词搭配使用。

::: info :books: 例句

1. <grammer-content sentence='それでお[土産/みやげ]を[買/か]いに[行/い]くところなんです。' trans='因此要去买土特产。' />
2. <grammer-content sentence='[今/いま]、[出/で]かけるところです。' trans='现在正要出门。' />
3. <grammer-content sentence='これから[寮/りょう]を[出/で]るところですから、15[分/ぷん]ぐらい[待/ま]ってください。' trans='马上就要离开宿舍了，请等15分钟左右。' />

:::

### b. Vているところだ

表示该动作行为在说话时正在进行之中，经常与「今、ちょうど」等时间副词搭配使用。

::: info :books: 例句

1. <grammer-content sentence='[今/いま]、彼女とチャットをしているところです。' trans='现在正在和女朋友聊天。' />
2. <grammer-content sentence='ちょうどお[風呂/ふろ]に[入/はい]っているところです。' trans='现在正在洗澡。' />

:::

### c. Vたところだ

表示该动作行为在说话时刚刚结束，经常与「今、たったいま」等时间副词搭配使用。

::: info :books: 例句

1. <grammer-content sentence='[今/いま]、[授業/じゅぎょう]が[終/お]わったところです。' trans='现在刚刚下课。' />
2. <grammer-content sentence='李さんはたった[今/いま]、[大学/だいがく]から[寮/りょう]に[帰/き]って来たところです。' trans='小李现在刚从大学回到宿舍。' />

:::

### d. Vていたところだ

表示从过去某一时点到说话之前该状态一直在持续，多用来说明之前持续的思考行为或心理状态。

::: info :books: 例句

1. <grammer-content sentence='僕もそろそろ[帰/き]ろうと[思/おも]っていたところだ。' trans='我刚才也一直在想差不多该回去了。' />
2. <grammer-content sentence='なかなか[連絡/れんらく]がないから[心配/しんぱい]していたところだよ。' trans='因为一直没有联系，我刚才一直在担心。' />

:::
