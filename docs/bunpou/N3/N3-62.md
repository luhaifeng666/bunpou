# ～Vてからでないと<必要条件> <Badge type="warning" text="N3" />

* 意义：表示必要条件，即如果不先完成前项动作，就不能进行后续动作，后句往往是表示否定或消极意义的内容。
* 接续：Vて+からでないと/からでなければ
* 译文：如果不先……就（不）……

::: info :books: 例句

1. <grammer-content sentence='**[見/み]てからでないと**[言/い]いかけた[/か]**、[母/はは]と[一緒/いっしょ]に』と[さっさと/さっさと][チケット/ちけっと]を2[枚/まい][渡/わた]された。' trans='刚说「得先看看日程表」，对方就说「可以和妈妈一起来」，然后迅速塞过来两张票。' />
2. <grammer-content sentence='**[両親/りょうしん]に[相談/そうだん]して からでないと**、[返事/へんじ]できません。' trans='关于这件事，如果不先和父母商量就无法回复。' />
3. <grammer-content sentence='**[免許/めんきょ]を[取/と]ってからでないと**[運転/うんてん]してはいけない。' trans='不先拿到驾照就不能开车。' />
4. <grammer-content sentence='**[説明/せつめい]を[読/よ]んでからでないと**[使/つか]い[方/かた]が[分/わ]からない。' trans='不先看看说明书的话就不知道用法。' />
