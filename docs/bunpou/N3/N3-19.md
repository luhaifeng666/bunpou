# ～ことになっている＜約定・慣例＞ <Badge type="warning" text="N3" />

* 意义：表示习惯、惯例、计划、约定，法律或规章制度的约束以及客观规律等。
* 接续：名词＋という／动词的词典形或否定形＋ことになっている
* 译文：（规定）……；习惯于……

::: info :books: 例句

1. <grammer-content sentence='[本学/ほんがく]では、3[年/ねん]になると、ゼミに[所属/しょぞく]する**ことになっている**。' trans='在我们学校，从三年级开始，就要选修研讨课。' />
2. <grammer-content sentence='4[月/がつ]から[日本/にほん]へ[留学/りゅうがく]に[行/い]く**ことになっている**。' trans='从4月开始要去日本留学。' />
3. <grammer-content sentence='[日本/にほん]では18[歳/さい]からが[成人/せいじん]という**ことになっている**。' trans='在日本，18岁就算是成年人。' />
4. <grammer-content sentence='[店内/てんない]ではタバコを[吸/す]ってはいけない**ことになっている**。' trans='店内规定不能吸烟。' />
5. <grammer-content sentence='[朝食/ちょうしょく]は[必/かなら]ず[家族/かぞく]そろって[食/た]べる**ことになっていた**のに、[最近/さいきん]はなかなかそうはいかない。' trans='本来规定早餐一定要全家人一起吃，但最近很难做到。' />

:::
