# N をきっかけに（して）〈契机〉 <Badge type="warning" text="N3" />

- 意义：表示以某事为契机或开端。
- 接续：名词 + をきっかけに（して）
- 译文：以......为契机；以......为开端

::: info :books: 说明

也可以使用「Nがきっかけで」的形式。

:::

::: info :books: 例句

1. <grammer-content sentence='[比較文化/ひかくぶんか]の[授業/じゅぎょう]をきっかけに、[日本/にほん]のポップカルチャーに[興味/きょうみ]を[持/も]ちました。' trans='以比较文化的课为契机，对日本的流行文化产生了兴趣。' />
2. <grammer-content sentence='[姉/あね]は[結婚/けっこん]をきっかけに、[仕事/しごと]を[辞/や]めた。' trans='姐姐以结婚为契机辞职了。' />
3. <grammer-content sentence='パーティーでの[出会/であ]いをきっかけに、[二人/ふたり]は[付/つ]き[合/あ]うことになった。' trans='以派对上的相遇为契机，两人开始交往了。' />
4. <grammer-content sentence='3[月/がつ]の[北京/ペキン][旅行/りょこう]をきっかけに、[中国語/ちゅうごくご]の[勉強/べんきょう]を[始/はじ]めた。' trans='以3月的北京旅行为契机，开始学习中文了。' />

:::
