# 使动态与使动句 <Badge type="warning" text="N3" />

- 意义：表示动作主体强制、许可、放任对方做某事，也可表示动作主体引发对方产生某种感情。
- 接续：动词词干后面接后缀「（さ）せる」
- 译文：使；让

::: info :books: 说明

动词类别变化规则：
- I类动词：词尾「う」段假名 → 「あ」段假名 + 「せる」
- II类动词：词尾「る」 → 词尾「る」 + 「させる」
- III类动词：来る → 来させる；する → させる

:::

### a. 他动词使动句

动作的主体变为补语，用格助词「に」表示。原主动句的客体在使动句中不变，仍用格助词「を」表示。

::: info :books: 例句

1. <grammer-content sentence='母は[子供/こども]に[本/ほん]を[読/よ]ませる。' trans='妈妈让孩子读书。' />
2. <grammer-content sentence='[先生/せんせい]は[学生/がくせい]に[日本語/にほんご]を[話/はな]させる。' trans='老师让学生说日语。' />
3. <grammer-content sentence='[娘/むすめ]が[日本/にほん]から[帰国/きこく]するとき、いつも1か[月/げつ][前/まえ]から[予約/よやく]させてるの。' trans='女儿从日本回国的时候，我总是让她提前一个月左右订票。' />

:::

### b. 自动词使动句

动作的主体一般用「を格」表示，有时也可以用「に格」表示。

::: info :books: 例句

1. <grammer-content sentence='親は子供[を/に][買物/かいもの]に[行/い]かせる。' trans='父母让孩子去买东西。' />
2. <grammer-content sentence='[次/つぎ]は僕にやらせてください。' trans='下一次请让我来做。' />

:::

### c. 使动句的含义

1. 命令、强制对方做某事
2. 许可、放任对方做某事
3. 引发对方产生某种感情（动作的主体只能用"を格"表示）

::: info :books: 例句

1. <grammer-content sentence='先生が学生に[日本語/にほんご]で[作文/さくぶん]を[書/か]かせる。' trans='老师让学生用日语写作文。' />
2. <grammer-content sentence='先生は学生に[自由/じゆう]に[発表/はっぴょう]テーマを[選/えら]ばせた。' trans='老师让学生自由选择发表主题。' />
3. <grammer-content sentence='先生はいつも[冗談/じょうだん]を[言/い]ってみんなを[笑/わら]わせる。' trans='老师总是开玩笑让大家笑。' />

:::
