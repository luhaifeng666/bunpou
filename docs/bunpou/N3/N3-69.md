# ～まま＜保持原状＞ <Badge type="warning" text="N3" />

* 意义：表示原来的状态一直维持，没有改变。
* 接续：名词＋の／形容词的连体形＋まま；动词的た形或否定形＋まま
* 译文：一直……；保持着……

::: info :books: 例句

1. <grammer-content sentence='[原語/げんご]の[意味/いみ]との[ずれ/ずれ]を[意識/いしき]しない**まま**[使/つか]ってしまえば、ミスコミュニケーションを[起/お]こす[恐/おそ]れもある。' trans='如果没有意识到和原词意思的差异就这么使用，可能会引发交流错误。' />
2. <grammer-content sentence='[薬/くすり]を[飲/の]んで[頭痛/ずつう]がなくなったが、[熱/ねつ]は[高/たか]い**まま**だ。' trans='吃了药头痛好了，但还在发烧。' />
3. <grammer-content sentence='この[山/やま]はずっと[自然/しぜん]の**まま**、[静/しず]かな**まま**でいてほしい。' trans='希望这座山一直保持自然的状态，保持安静。' />
4. <grammer-content sentence='キャベツは[生/なま]の**まま**で[食/た]べる[人/ひと]が[多/おお]いでしょう。' trans='生吃卷心菜的人应该很多吧。' />
