# ～せいで／～せいだ＜消極原因＞ <Badge type="warning" text="N3" />

* 意义：表示消极的原因。
* 接续：动词、形容词的连体形＋せいで／せいだ　名词＋の＋せいで／せいだ
* 译文：……就因为……；……是由于……

::: tip :bookmark: 説明

使用`～せいで`时，表示由于该原因，造成了后面句子所叙述的不好的结果；使用`～せいだ`时，表示前面句子叙述的不好的结果，是由该原因造成。

如果`Nのせいで／せいだ`中的名词是指称人的，则带有谴责该人的语气。

:::

::: info :books: 例句

1. <grammer-content sentence='ねえ、[高橋/たかはし]さん。[渡辺/わたなべ]さん、このごろ[元気/げんき]がありませんね。やっぱり、[受験勉強/じゅけんべんきょう]の**せい**でしょうか。' trans='高桥，渡边最近情绪不高。是不是因为考试呀？' />
2. <grammer-content sentence='[食欲/しょくよく]がないのは、[暑/あつ]さの**せい**かもしれない。' trans='没有食欲可能是因为热。' />

4. <grammer-content sentence='[成績/せいせき]が[悪/わる]かった**せいで**、[希望/きぼう]の[大学/だいがく]に[行/い]けなかった。' trans='因为成绩不好，没能去成理想的大学。' />
5. <grammer-content sentence='[私/わたし]が[先生/せんせい]に[叱/しか]られたのは、あなたの**せいです**。' trans='我被老师训斥是你的错。' />
