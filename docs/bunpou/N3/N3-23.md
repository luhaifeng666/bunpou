# Nとして＜資格・性質＞ <Badge type="warning" text="N3" />

* 意义：表示动作主体进行某动作时的身份、资格、立场、性质等，还可以表示事物的性质。
* 接续：N＋として
* 译文：作为……

::: info :books: 例句

1. <grammer-content sentence='[万里/ばんり]の[長城/ちょうじょう]は[世界遺産/せかいいさん]**として**[国内外/こくないがい]に知られている。' trans='万里长城作为世界遗产闻名国内外。' />
2. <grammer-content sentence='[先生/せんせい]**として**[言/い]いたいことがあります。' trans='作为老师，我有话想说。' />
3. <grammer-content sentence='この[問題/もんだい]はみんなの[問題/もんだい]**として**[考/かんが]えることが[必要/ひつよう]だ。' trans='这个问题有必要作为大家的问题来考虑。' />

:::
