# 日本語簡体会話の特徴（3）－略形式 <Badge type="warning" text="N3" />

* 意义：在简体会话中，动词与补助动词间常出现缩略形式以及助词的省略。这些表达方式都比较随意，因此一般不能用于正式场合，也不能直接用于与上级或年长者的会话。

::: tip :bookmark: よくある略形式

**a. ておく → とく　　でおく → どく**

**b. てしまう → ちゃう　　でしまう → じゃう**

:::

::: info :books: 例句

1. <grammer-content sentence='じゃあ、[欠席/けっせき]って[返事/へんじ][書/か]いて[出/だ]し**とこう**か？' trans='那我就回复说你去不了？' />
2. <grammer-content sentence='お[客/きゃく]さんが[来/く]るから、ビール[冷/ひ]やし**といて**ね。' trans='客人要来了，先把啤酒冰着。' />
3. <grammer-content sentence='[帰/かえ]ったらすぐに[明日/あした]の[予習/よしゅう]をし**とこう**かな。' trans='回去之后先把明天的预习做了吧。' />
4. <grammer-content sentence='[今晩/こんばん]、この[本/ほん]の30ページまで[読/よ]ん**どく**。' trans='今晚把这本书读到第30页。' />
5. <grammer-content sentence='[最近/さいきん]ちょっとスランプ。よく[眠/ねむ]れなくなっ**ちゃって**……。' trans='最近有点消沉。变得睡不好觉了……' />
6. <grammer-content sentence='お[皿/さら]を[落/お]として[割/わ]っ**ちゃった**。' trans='把盘子掉下来摔碎了。' />
7. <grammer-content sentence='[遅刻/ちこく]して[先生/せんせい]に[怒/おこ]られ**ちゃった**。' trans='迟到被老师骂了。' />
8. <grammer-content sentence='お[酒/さけ]、[弱/よわ]いのに5[杯/はい]も[飲/の]ん**じゃって**、[頭/あたま]が[痛/いた]い。' trans='酒量很差却喝了5杯，头很痛。' />

:::
