# Nったら＜評価の対象＞ <Badge type="warning" text="N3" />

* 意义：表示说话人对某人物的轻微不满，后续的内容是对该人物的负面评价。
* 接续：名词＋ったら
* 译文：说到……；那个……

::: info :books: 例句

1. <grammer-content sentence='[彼/かれ]**ったら**、また[遅刻/ちこく]した。' trans='说到他，又迟到了。' />
2. <grammer-content sentence='[お母さん]**ったら**、[心配/しんぱい]しすぎだよ。' trans='说到妈妈，太担心了。' />
3. <grammer-content sentence='[妹/いもうと]**ったら**、[私/わたし]の[日記/にっき]を[読/よ]んだの。' trans='说到妹妹，看了我的日记。' />
4. <grammer-content sentence='[田中/たなか]**ったら**！' trans='说到田中！（表示不满）' />

:::
