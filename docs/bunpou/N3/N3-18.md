# ～んじゃない？＜反問＞ <Badge type="warning" text="N3" />

* 意义：读升调，以否定的形式向听话人传达自己的判断，并向听话人进行确认。
* 接续：普通形（ナAな／Nな）＋んじゃない
* 译文：……不是吗？

::: info :books: 例句

1. <grammer-content sentence='ほかにも[行/い]きたい[人/ひと]がいる**んじゃない**？' trans='还有其他想去的人不是吗？' />
2. <grammer-content sentence='[教室/きょうしつ]より[図書館/としょかん]のほうが[静/しず]かな**んじゃない**？' trans='比起教室，图书馆更安静不是吗？' />
3. <grammer-content sentence='[彼/かれ]、[体/からだ]の[調子/ちょうし]でも[悪/わる]い**んじゃない**？' trans='他身体不舒服之类的吗？' />

:::
