# ～というわけだ＜説明＞ <Badge type="warning" text="N3" />

* 意义：表示根据前面的叙述自然而然得出的结论。
* 接续：简体句子＋というわけだ
* 译文：就是说……

::: info :books: 例句

1. <grammer-content sentence='[京都/きょうと]の[建物/たてもの]や[庭園/ていえん]は[確/たし]かに[素晴/すば]らしい、だからといって[伝統/でんとう]にばかり[頼/たよ]ってはいられない、**というわけだ**。' trans='京都的建筑和庭院确实很棒，但不能因此就一味地依赖传统，就是这个意思。' />
2. <grammer-content sentence='このように、[毎日/まいにち][続/つづ]ければ[効果/こうか]が[出/で]る**というわけだ**。' trans='像这样，每天坚持的话就会有效果。' />
3. <grammer-content sentence='[友達/ともだち]がみんな[応援/おうえん]してくれました。だから、スピーチコンテストで[優勝/ゆうしょう]できた**というわけです**。' trans='朋友们都支持我。所以演讲比赛获胜了。' />
4. <grammer-content sentence='Ａ：[合格/ごうかく]したのは[一年生/いちねんせい]の5[人/にん]と[二年生/にねんせい]の12[人/にん]です。Ｂ：つまり、[合/あ]わせて17[人/にん]**というわけですね**。' trans='A：合格的是一年级的5人和二年级的12人。B：也就是说一共17人。' />

:::
