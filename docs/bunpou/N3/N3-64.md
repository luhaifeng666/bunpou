# ～Vかける<将影响施加于人> <Badge type="warning" text="N3" />

* 意义：对某人做某事，行为涉及对方。
* 接续：动词第一连用形+かける
* 译文：对……（做）……；开始向……（做）……

::: tip :bookmark: 説明

固定搭配：話しかける（搭话）、問いかける（质问）、呼びかける（呼吁）等。

:::
