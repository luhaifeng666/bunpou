# ～っぽい＜性質・傾向＞ <Badge type="warning" text="N3" />

* 意义：表示具有名词所示意义的那种感觉。
* 接续：名词／动词的第一连用形＋っぽい
* 译文：容易……；有点……

::: tip :bookmark: 説明

「っぽい」多带有负面的评价。

:::

::: info :books: 例句

1. <grammer-content sentence='この[肉/にく]は[焼/や]きすぎで[焦/こげ]**っぽい**。' trans='这肉烤过头了，有点焦。' />
2. <grammer-content sentence='[彼/かれ]は[忘/わす]れ**っぽい**ので、[注意/ちゅうい]してください。' trans='他容易忘事，请注意。' />
3. <grammer-content sentence='[子供/こども]**っぽい**[言/い]い[方/かた]をしないでください。' trans='请不要用孩子气的说法。' />
4. <grammer-content sentence='[最近/さいきん][太/ふと]り**っぽく**なった。' trans='最近变得容易发胖了。' />

:::
