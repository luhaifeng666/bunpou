# N1からN2にかけて〈时间、空间、范围〉 <Badge type="warning" text="N3" />

- 意义：表示事件、现象存在的时间或空间范围。
- 接续：表示时间或处所的名词 + から + 同类名词 + にかけて
- 译文：从......到......

::: info :books: 说明

修饰名词时，使用「N1からN2にかけてのN3」的形式。与「N1からN2まで」相比，「N1からN2にかけて」一般不表示明确的起点和终点，只表示大致的时间或空间范围。

:::

::: info :books: 例句

1. <grammer-content sentence='[今/いま]から800[年/ねん]ほど[前/まえ]、[平安時代/へいあんじだい]の[終/お]わり[頃/ころ]から[鎌倉時代/かまくらじだい]の[初/はじ]め[頃/ころ]にかけての[作品/さくひん]である。' trans='距今八百年左右以前，大概是平安时代的终末到镰仓时代的初始的作品。' />
2. <grammer-content sentence='この[病気/びょうき]は[夏/なつ]から[秋/あき]にかけて[流行/りゅうこう]する。' trans='这种疾病从夏天到秋天流行。' />
3. <grammer-content sentence='[今晩/こんばん]、[関東/かんとう]から[東北/とうほく]にかけて[大雨/おおあめ]になるそうだ。' trans='听说今晚从关东到东北会下大雨。' />
4. <grammer-content sentence='[肩/かた]から[腕/うで]にかけて[痛/いた]みを[感/かん]じる。' trans='从肩膀到手腕感觉疼痛。' />

:::
