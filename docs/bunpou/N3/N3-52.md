# ～っぱなし＜放任＞ <Badge type="warning" text="N3" />

* 意义：接他动词时，表示做完了某一动作后放置不管；接自动词时，表示某一状态一直持续或某一动作反复进行。带有不满、批评的语气。
* 接续：动词第一连用形＋っぱなし
* 译文：一直……；光……

::: tip :bookmark: 説明

描述负面情况。与「ている」不同，「っぱなし」只能用于负面。

:::
