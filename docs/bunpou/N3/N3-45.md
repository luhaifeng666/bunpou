# N向き＜適合＞ <Badge type="warning" text="N3" />

* 意义：表示本来就拥有的特质，适于某方面。
* 接续：名词＋向き
* 译文：适合……的

::: tip :bookmark: 説明

「N向け」表示以某人或群体为对象，「N向き」表示本身就拥有的特质，适于某方面。

:::

::: info :books: 例句

1. <grammer-content sentence='[彼女/かのじょ]はアニメの[声優/せいゆう]**向き**の[声/こえ]をしている。' trans='她的声音适合做动画声优。' />
2. <grammer-content sentence='この[山/やま]は、[道/みち]が[狭/せま]くて[険/けわ]しいので[経験者/けいけんしゃ]**向き**だ。' trans='这座山道路狭窄险峻，适合有经验的人。' />
3. <grammer-content sentence='この[仕事/しごと]は[若者/わかもの]**向き**だ。' trans='这份工作适合年轻人。' />

:::
