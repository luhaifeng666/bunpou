# Vてはならない＜禁止＞ <Badge type="warning" text="N3" />

* 意义：表示禁止，多用于阐述规定、提示、训诫等。
* 接续：Vて＋はならない
* 译文：不能……；不可以……

::: tip :bookmark: 説明

属于书面用语。「忘れてならない」为惯用形式，意为"不能忘记"。

:::

::: info :books: 例句

1. <grammer-content sentence='[外国人力士/がいこくじんりきし]の[活躍/かつやく]を[考/かんが]える[際/さい]、**忘れてならない**[存在/そんざい]は[ハワイ/はわい][出身/しゅっしん]の[高見山/たかみやま]という[力士/りきし]のことである。' trans='在考虑外国人力士的活跃时，不能忘记的存在是夏威夷出身的高见山力士。' />
2. <grammer-content sentence='[若/わか]いときに[失敗/しっぱい]を**恐れてはならない**。' trans='年轻时不能害怕失败。' />
3. <grammer-content sentence='お[客様/きゃくさま]の[情報/じょうほう]を[決/けっ]して[他人/たにん]に**話してはならない**。' trans='绝对不能把客人的信息告诉别人。' />
4. <grammer-content sentence='あって**はならない**[事件/じけん]が[起/お]こってしまった。' trans='发生了不应该发生的事件。' />
