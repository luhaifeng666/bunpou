# ～だけあって<成正比的因果关系> <Badge type="warning" text="N3" />

* 意义：表示前后句所述的情况相符合、成正比。常与副词「さすがに」呼应使用。
* 接续：名词/动词、形容词的连体形+だけあって
* 译文：不愧是……；正因为……

::: info :books: 例句

1. <grammer-content sentence='**[黒沢/くろざわ][監督/かんとく]の[講演/こうえん]という[だけ/だけ][あって]**、[場内/じょうない]は[満員/まんいん]だ。' trans='不愧是黑泽导演的演讲，现场座无虚席。' />
2. <grammer-content sentence='**[日本/にほん]に20[年/ねん]も[いた/いた][だけ/だけ][あって]**、[日本/にほん]のことに**[詳/くわ]しい**。' trans='不愧是在日本待了20年，对日本的事情非常了解。' />
3. <grammer-content sentence='**[若/わか]い[だけ/だけ][あって]**、[回復/かいふく]が[早/はや]かった。' trans='正因为年轻，恢复得很快。' />
:::
