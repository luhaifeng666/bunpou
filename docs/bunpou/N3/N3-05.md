# Vずに＜動作の否定＞ <Badge type="warning" text="N3" />

* 意义：表示在未实施该动作的情况下进行了后句所述的动作。
* 接续：Vない＋ずに（する→せずに）
* 译文：不（没）……（的情况下）

::: tip :bookmark: 説明

`Vずに`是`Vないで`的书面语形式。

`Vずに`与`Vないで``Vず`大多数情况下可以互换使用，但是在前后句表达并列的含义时，不能使用`Vずに`。

:::
