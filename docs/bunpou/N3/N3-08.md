# やる／Vてやる＜授受・施恩＞ <Badge type="warning" text="N3" />

* 意义：`やる`表示说话人自己或自己一方的人将物品赠予他人，主要用于上对下、长对幼，接受者也可以是动植物（表示喂食或浇水）。`Vてやる`表示自己或属于自己一方的人为别人做某事，主要用于上对下、长对幼的动作，也可以表示对动植物施行的动作。
* 接续：动词て形＋やる
* 译文：给……／我为（别人）……

::: warning :warning: 注意

`やる`与`あげる／くれる／もらう`一样，也是表示授受关系的动词。在使用`やる`或`Vてやる`时，物品的接受者或动作的接受者都不能是说话人或说话人一方的人。另外，当听话人是接受者时，仅限于父母对子女、夫妻之间、兄弟姐妹之间或上对下的场合。

:::
