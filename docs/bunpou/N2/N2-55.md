# Nをもって＜手段、方式＞ <Badge type="danger" text="N2" />

* 意义：表示手段或方式。
* 接续：名词＋をもって
* 译文：以……；用……

::: tip :bookmark: 説明

一般用于文章中或会议等正式场合，是较生硬的表达方式。

:::

::: info :books: 例句

1. <grammer-content sentence='[私/わたし]の[発/はっ]した「ウォヤオピングォ」という[音/おと]が、ある[法則/ほうそく]**をもって**[相手/あいて]の[頭/あたま]のなかに[伝/つた]わり……' trans='我发出的「woyaopingguo」的声音，基于一定的法则，传达到了对方的大脑当中……' />
2. <grammer-content sentence='[詳/くわ]しいことは、[書面/しょめん]**をもって**お[知/し]らせいたします。' trans='详细情况将以书面形式通知。' />
3. <grammer-content sentence='[最新/さいしん][技術/ぎじゅつ]**をもって**すれば、その[計画/けいかく]は[十分/じゅうぶん][実現/じつげん][可能/かのう]である。' trans='如果有最新技术，那个计划完全可以实现。' />
4. <grammer-content sentence='[現代/げんだい]の[科学/かがく]**をもって**しても[解明/かいめい]できないことがたくさんある。' trans='即使是现代科学，也有很多无法解释的事情。' />
5. <grammer-content sentence='中国式[現代化/げんだいか]**をもって**[中華民族/ちゅうかみんぞく]の[偉大/いだい]な[復興/ふっこう]を[全面/ぜんめん]てきに[推進/すいしん]する。' trans='以中国式现代化全面推进中华民族伟大复兴。' />
:::
