# Nをはじめ（として）＜代表性事物＞ <Badge type="danger" text="N2" />

* 意义：用于提出一个有代表性的或典型的人或事物，然后再举出其他相关的人或事物。
* 接续：名词＋はじめ（として）
* 译文：以……为首；以……为代表……

::: tip :bookmark: 説明

修饰名词时为「N1をはじめとするN2」。

:::

::: info :books: 例句

1. <grammer-content sentence='[蒙古/もんごる][出身/しゅっしん]の[力士/りきし]**をはじめ**、[中国/ちゅうごく]、[韓国/かんこく]、[俄罗斯/ろしあ]、[保加利亚/ぶるがりあ][出身/しゅっしん]の[力士/りきし]が[名/な]を[連/つら]ねてきた。' trans='以蒙古出身的力士为首，中国、韩国、俄罗斯、保加利亚出身的力士也纷纷崭露头角。' />
2. <grammer-content sentence='[日本/にほん]の[伝統芸能/でんとうげいのう]には、[歌舞伎/かぶき]**をはじめ**、[能/のう]、[狂言/きょうげん]などがあります。' trans='日本的传统艺能有歌舞伎、能乐、狂言等。' />
3. <grammer-content sentence='[日本語/にほんご]には[英語/えいご]**をはじめとして**、フランス語、ドイツ語、オランダ語などから[生/う]まれた[外来語/がいらいご]がある。' trans='日语中有以英语为首，从法语、德语、荷兰语等产生的外来语。' />
4. <grammer-content sentence='お[母様/かあさま]**をはじめ**、ご[家族/かぞく]の[皆様/みなさま]にどうぞよろしくお[伝/つた]えください。' trans='请代我向您母亲及家人问好。' />
5. <grammer-content sentence='[近年/きんねん]、[日本/にほん]では[中国/ちゅうごく]**をはじめとする**アジア[諸国/しょこく]からの[電気製品/でんきせいひん]の[輸入/ゆにゅう]が[増/ふ]えている。' trans='近年来，日本从中国等亚洲各国进口的电器产品在增加。' />

:::
