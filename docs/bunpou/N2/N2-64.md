# ～Vつつ（も）<转折> <Badge type="danger" text="N2" />

* 意义：与「ながら（も）」相同，连接两件相反的事情，表示转折关系。
* 接续：动词第一连用形+つつ（も）
* 译文：尽管……，但是……

::: info :books: 例句

1. <grammer-content sentence='「[愛/あい]してるよ」と**[言/い]いつつ（も）**、[浮気/うわき]ばかりしている。' trans='一边说「我爱你」，一边却一直在出轨。' />
2. <grammer-content sentence='[将来/しょうらい]の[仕事/しごと]を**[考/かんが]えつつ**[進路/しんろ]を[選/えら]ぶ。' trans='一边考虑将来的工作，一边选择进路。' />
:::
