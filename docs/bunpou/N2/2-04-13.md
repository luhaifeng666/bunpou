# Nに及ぶ〈达到〉 <Badge type="danger" text="N2" />

- 意义：表示达到的数值、范围或程度，与数量名词共现时为主观大量的表达方式。
- 接续：名词 + に及ぶ
- 译文：达到......；涉及......；波及......

::: info :books: 例句

1. <grammer-content sentence='それぞれ[長/なが]さ10メートルに[及/およ]ぶ[絵巻物/えまきもの]である。' trans='是每卷都达到十米的画卷。' />
2. <grammer-content sentence='[約/やく]3か[月/げつ]に[及/およ]ぶ[撮影/さつえい]は、[体力/たいりょく]が[必要/ひつよう]だ。' trans='长达三个月的拍摄，需要体力。' />
3. <grammer-content sentence='10[日/か]から[続/つづ]いた[雨/あめ]の[被害/ひがい]は、[九州/きゅうしゅう][各地/かくち]に[及/およ]んだ。' trans='持续了十天的雨灾，波及九州各地。' />
4. <grammer-content sentence='[話題/わだい]は[文学/ぶんがく]だけでなく、[社会/しゃかい]や[国際問題/こくさいもんだい]にも[及/およ]んだ。' trans='话题不只涉及文学，还波及社会和国际问题。' />

:::
