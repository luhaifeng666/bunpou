# V（さ）せられる＜不由自主＞ <Badge type="danger" text="N2" />

* 意义：表示不由自主地产生了某种感情或引发了思考。
* 接续：表示感情、思考行为等的动词＋（さ）せられる
* 译文：不由得……

::: tip :bookmark: 説明

引发感情、思考的原因一般用「に」表示。

:::

::: info :books: 例句

1. <grammer-content sentence='そう[思/おも]う[一方/いっぽう]で、[揺/ゆ]れる[車内/しゃない]でもきれいに[化粧/けしょう]をする[様子/ようす]を[見/み]て、[器用/きよう]なものだと[妙/みょう]に**感心させられた**りもする。' trans='另一方面，看到在摇晃的车厢里也能漂亮地化妆，不由得佩服起来。' />
2. <grammer-content sentence='ロンドンに[来/き]て、[物価/ぶっか]の[高/たか]さに**は驚かされた**。' trans='来到伦敦，不由得被物价之高吓到了。' />
3. <grammer-content sentence='[日本/にほん]に[住/す]んでいるときは、[毎年/まいとし][春/はる]になると[花粉症/かふんしょう]に**悩まされていた**。' trans='住在日本的时候，每年春天都不由得被花粉症困扰。' />
4. <grammer-content sentence='この[本/ほん]を[読/よ]んで、[自分/じぶん]の[今/いま]までの[生/い]き[方/かた]を**反省させられた**。' trans='读了这本书，不由得反省了自己至今为止的生活方式。' />
