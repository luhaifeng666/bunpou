# ～からといって～（とは限らない）＜転折＞ <Badge type="danger" text="N2" />

* 意义：表示前后句的转折关系，后句常接「とは限らない」或其他否定形表达，意为前后两句之间未必能够构成因果关系。
* 接续：简体句＋からといって
* 译文：虽然……未必……；虽说……但也（不）……

::: tip :bookmark: 説明

口语中也可以采用「からって」的形式。

:::

::: info :books: 例句

1. <grammer-content sentence='[中国人/ちゅうごくじん]だ**からといって**、みんなカンフーができるとは[限/かぎ]りませんよ。' trans='虽然是中国人，但未必都会功夫。' />
2. <grammer-content sentence='ビールが[好/す]きだ**からといって**、[毎日/まいにち][飲/の]むとは[限/かぎ]らない。' trans='虽说喜欢啤酒，但也不是每天都喝。' />
3. <grammer-content sentence='やせている**からといって**、[体/からだ]が[弱/よわ]いとは[限/かぎ]らない。' trans='虽说瘦，但身体未必弱。' />
4. <grammer-content sentence='[日本/にほん]で[暮/く]らしていた**からといって**、[日本語/にほんご]が[上手/じょうず]だとは[限/かぎ]らない。' trans='虽说在日本生活过，但日语未必好。' />
5. <grammer-content sentence='[海外/かいがい]での[仕事/しごと]がうまく[行/い]かなかった**からといって**、[帰国/きこく]を[考/かんが]えることは[必要/ひつよう]はない。' trans='虽说海外工作不顺利，但也不必考虑回国。' />

:::
