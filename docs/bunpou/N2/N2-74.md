# ～からには<推理的根据> <Badge type="danger" text="N2" />

* 意义：与「～以上」意思相同，表示基于前项做出说话人的判断。
* 接续：V/A1连/N+である+からには
* 译文：既然……就……

::: info :books: 例句

1. <grammer-content sentence='**[やると[言/い]った]からには**[、[途中/とちゅう]で[投/な]げ[nari/出/だ][nari/し]ず、[最後/さいご]まで[終/お]らせたい。' trans='既然说了要做，就不想半途而废，想坚持到最后。' />
2. <grammer-content sentence='**[社会人/しゃかいじん]になった[nari/からには]****、[仕事/しごと]を[頑張/がんば]る[nari/つもりだ]**。' trans='既然成为了社会人士，就打算努力工作。' />
3. <grammer-content sentence='**[約束/やくそく]した[nari/からには]**、[どんな/どんな]ことが[nari/あっても]**[守/まも]ら[nari/なければ/なければならない]**。' trans='既然约定了，无论发生什么事都必须遵守。' />
:::
