# ～からには＜推理的根拠＞ <Badge type="danger" text="N2" />

* 意义：与「～以上」意思相同，表示基于前项做出说话人的判断。口语里可以省略为「～からは」。
* 接续：V／A1連体形、名词＋である＋からには
* 译文：既然……就……

::: info :books: 例句

1. <grammer-content sentence='やると[決/き]めた**からには**、[途中/とちゅう]で[投/な]げ[出/だ]したりせず、[最後/さいご]まで[終/お]わらせたい。' trans='既然决定了要做，就不想半途而废，想坚持到最后。' />
2. <grammer-content sentence='[社会人/しゃかいじん]になった**からには**、[仕事/しごと]を[頑張/がんば]るつもりだ。' trans='既然成为了社会人士，就打算努力工作。' />
3. <grammer-content sentence='[約束/やくそく]した**からには**、どんなことがあっても[守/まも]らなければならない。' trans='既然约定了，无论发生什么事都必须遵守。' />

:::
