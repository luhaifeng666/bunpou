# Nに応えて<回应> <Badge type="danger" text="N2" />

* 意义：表示根据对方的行为进行回应。多是对于要求、邀请、意见等的回应。
* 接续：名词+に応えて
* 译文：回应……；响应……

::: info :books: 例句

1. <grammer-content sentence='こちら**[要望/ようぼう]に応えて**[いた/いた]だき[ありがとう/ありがとう]ございます。' trans='感谢您回应了我们的请求。' />
2. <grammer-content sentence='お[客/きゃく]様の**[要求/ようきゅう]に[答/こた]えて**[来月/らいげつ]から[営業/えいぎょう][時間/じかん]を[一/いち][時間/じかん][延長/えんちょう]させていただきます。' trans='应顾客要求，下个月起将延长营业时间一小时。' />
3. <grammer-content sentence='[結果/けっか]は[最下位/さいかい]となり、**[ご/ご][声援/せいえん]に応えられず**[抱歉/しゃべん]ございませんでした。' trans='结果是最后一名，没能回应大家的支持，非常抱歉。' />
:::
