# ～V（よ）うものなら<条件> <Badge type="danger" text="N2" />

* 意义：表示条件，意为一旦做了V这个动作，就会引发出一个比较严重的后果。用于强调说话人担心、恐惧、不安等心情。
* 接续：动词意志形+ものなら
* 译文：万一……就……；要是……就……

::: info :books: 例句

1. <grammer-content sentence='ここで「[僕/ぼく]は[留守番/るすばん]してるよ」なんて**言おうものなら**、[怒/おこ]り[出/だ]すに[決/き]まっている。' trans='在这里如果说「我在家看门哦」，对方一定会生气的。' />
2. <grammer-content sentence='[社長/しゃちょう]の[提案/ていあん]に**反対しようものなら**、すぐ[首/くび]にされてしいまうだろう。' trans='要是反对社长的提案，一定会被立刻炒鱿鱼的。' />
3. <grammer-content sentence='[彼/かれ]に[一度/いちど]でも**[金銭/きんせん]を[貸/か]そうものなら**、[何度/なんど]でも[貸してくれ/かしてくれ]ていくよ。' trans='要是借给他一次钱，他就会没完没了地来借。' />
4. <grammer-content sentence='[林/はやし]、[先生/せんせい]に[年齢/ねんれい]を**聞こうものなら**、[すごく/すっご]く[怒/おこ]られるよ。' trans='要是敢问林老师的年龄，一定会被狠狠骂的。' />
