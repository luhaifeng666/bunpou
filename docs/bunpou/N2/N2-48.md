# ～にちがいない＜有把握的判断＞ <Badge type="danger" text="N2" />

* 意义：表示说话人基于某种根据的非常有把握的判断。
* 接续：名词／Ⅱ类形容词词干＋（である＋）にちがいない；动词、Ⅰ类形容词的简体形式＋にちがいない
* 译文：肯定……；一定……

::: info :books: 例句

1. <grammer-content sentence='[彼/かれ]は[日本人/にほんじん]**にちがいない**。' trans='他肯定是日本人。' />
2. <grammer-content sentence='[明日/あした]は[晴/は]れる**にちがいない**。' trans='明天肯定会放晴。' />
3. <grammer-content sentence='[彼女/かのじょ]は[来/く]る**にちがいない**。' trans='她肯定会来。' />
4. <grammer-content sentence='[成功/せいこう]する**にちがいない**。' trans='肯定会成功。' />

:::
