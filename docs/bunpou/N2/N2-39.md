# Nに対して＜対比＞ <Badge type="danger" text="N2" />

* 意义：用于叙述相互对照的两个事物。
* 接续：句子＋それに対して＋句子；动词、形容词的连体形＋のに対して
* 译文：与此不同，……；而……

::: info :books: 例句

1. <grammer-content sentence='[東京/とうきょう]では[経済的/けいざいてき][効果/こうか]を[重視/じゅうし]したためであろうか、[高層/こうそう]ビルを[林立/りんりつ]させた。**それに対して**、[京都/きょうと]は、[町並/まちな]みを[保存/ほぞん]するのに[熱心/ねっしん]であった。' trans='在东京可能是因为重视经济效果，建起了林立的高楼。与此相对，京都热衷于保存街区风貌。' />
2. <grammer-content sentence='[最近/さいきん]、[手紙/てがみ]を[書/か]く[人/ひと]が[少/すく]なくなった。**それに対して**、メールの[利用者/りようしゃ]が[急増/きゅうぞう]している。' trans='最近写信的人减少了。与此相对，邮件使用者急增。' />
3. <grammer-content sentence='Ａ[国/こく]は[年々/ねんねん][経済/けいざい]が[発展/はってん]している**のに対して**、Ｂ[国/こく]は[不況/ふきょう]が[続/つづ]いている。' trans='A国经济年年发展，而B国不景气持续。' />
4. <grammer-content sentence='[姉/あね]が[背/せ]が[低/ひく]い**のに対して**、[弟/おとうと]はクラスで[一番/いちばん][高/たか]い。' trans='姐姐个子矮，而弟弟是班里最高的。' />

:::
