# 感情形容詞＋させる＜感情を引き起こす＞ <Badge type="danger" text="N2" />

* 意义：某物或某事使某人产生某种感情。
* 接续：表人名词（或「心」「気分」「気持ち」等）＋を＋感情形容词AⅠく／AⅡに＋させる
* 译文：令人……；使人……

::: tip :bookmark: 説明

如果是使事物的性状发生变化，则使用`属性形容词＋する`的形式，这一用法在第一册第10课学到过。

:::

::: info :books: 例句

1. <grammer-content sentence='[相手/あいて]を[不快/ふかい]に**させない**[清潔感/せいけつかん]が[重要/じゅうよう]だ。' trans='不要令对方感到不快的清洁感很重要。' />
2. <grammer-content sentence='[雨/あめ]は[人/ひと]の[心/こころ]を[寂/さび]しく**させる**。' trans='雨使人心生寂寞。' />
3. <grammer-content sentence='[急/きゅう]な[転職/てんしょく]は[家族/かぞく]を[不安/ふあん]に**させた**。' trans='突然跳槽让家人感到不安。' />
4. <grammer-content sentence='[同僚/どうりょう]は、[子供/こども]を[寂/さび]しく**させたくない**といって[仕事/しごと]を[辞/や]める。' trans='同事说不想让孩子感到寂寞，所以辞职了。' />
5. <grammer-content sentence='[暑/あつ]いから、[髪/かみ]の[毛/け]を[短/みじか]くしたい。' trans='因为热，想把头发剪短。（属性形容词＋する）' />
6. <grammer-content sentence='エアコンをつけて、[部屋/へや]を[暖/あたた]かくした。' trans='开了空调，把房间弄暖和了。（属性形容词＋する）' />

:::
