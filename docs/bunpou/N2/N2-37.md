# Nはともかく＜別当論＞ <Badge type="danger" text="N2" />

* 意义：表示该名词所指的事物暂且不予讨论，后面所提到的事物才是更为重要的。
* 接续：名词＋はともかく（として）
* 译文：……另当别论；……暂且不提

::: info :books: 例句

1. <grammer-content sentence='[賛否/さんぴ]**はともかく**、これも[都市/とし]のひとつの[姿/すがた]である。' trans='赞成与否另当别论，这也是城市的一种姿态。' />
2. <grammer-content sentence='[結果/けっか]**はともかく**、[面接/めんせつ]が[終/お]わってよかった。' trans='结果暂且不提，面试结束了真好。' />
3. <grammer-content sentence='[彼/かれ]は[成績/せいせき]**はともかく**、[性格/せいかく]がとてもいいので、クラスの[人気者/にんきもの]だ。' trans='他成绩另当别论，性格很好，是班里的人气人物。' />
4. <grammer-content sentence='この[大学/だいがく]は、[場所/ばしょ]**はともかくとして**、[建物/たてもの]は[非常/ひじょう]に[立派/りっぱ]だ。' trans='这所大学，地点暂且不提，建筑非常气派。' />

:::
