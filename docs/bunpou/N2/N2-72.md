# ～にちがいない＜有把握的判断＞ <Badge type="danger" text="N2" />

* 意义：表示说话人基于某种根据的非常有把握的判断。多用于书面语，口语中多用「きっと～と思う」。
* 接续：名词／Ⅱ类形容词词干＋（である＋）にちがいない；Ⅰ类形容词、动词的连体形＋にちがいない
* 译文：肯定……；一定……

::: info :books: 例句

1. <grammer-content sentence='これだけ[表記/ひょうき]の[揺/ゆ]れがあれば[学習者/がくしゅうしゃ]にとって[記憶/きおく]の[負担/ふたん]になる**にちがいない**。' trans='只要书写方式有这么大的差别，就一定会成为学习者记忆的负担。' />
2. <grammer-content sentence='[細/こま]かい[描写/びょうしゃ]の[仕方/しかた]から[見/み]ると、この[小説/しょうせつ]の[作者/さくしゃ]は[女性/じょせい]**にちがいない**。' trans='从细腻的描写手法来看，这部小说的作者一定是女性。' />
3. <grammer-content sentence='あの[店/みせ]はいつも[多/おお]くの[人/ひと]が[並/なら]んでいるから、おいしい**にちがいない**。' trans='那家店总是排着很长的队，一定很好吃。' />
4. <grammer-content sentence='[彼/かれ]がそう[言/い]うなら、それは[事実/じじつ]である**にちがいない**。' trans='他那样说的话，那一定是事实。' />
5. <grammer-content sentence='ちょっと[曇/くも]っていて[残念/ざんねん]だが、[晴/は]れていれば[景色/けしき]はもっと[綺麗/きれい]**にちがいない**。' trans='有点阴天很遗憾，但如果晴天的话景色一定更美。' />

:::
