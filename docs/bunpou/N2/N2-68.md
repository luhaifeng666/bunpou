# Nから＜根拠・起因＞ <Badge type="danger" text="N2" />

* 意义：表示判断的根据或事情的起因。
* 接续：名词＋から
* 译文：根据……；因为……

::: info :books: 例句

1. <grammer-content sentence='テレビ[放送/ほうそう]の[外来語/がいらいご]を[調査/ちょうさ]した[結果/けっか]**から**は、[外来語/がいらいご]の[割合/わりあい]は5%[程度/ていど]にすぎないという[報告/ほうこく]もあるそうです。' trans='根据对电视节目外来语的调查，也有报告称外来语比例不过5%左右。' />
2. <grammer-content sentence='この[証拠/しょうこ]**から**[彼/かれ]は[無罪/むざい]ということになる。' trans='根据这个证据，他被认定无罪。' />
3. <grammer-content sentence='[以上/いじょう]の[分析結果/ぶんせきけっか]**から**、[次/つぎ]のことが[判断/はんだん]される。' trans='根据以上分析结果，得出以下判断。' />
4. <grammer-content sentence='[自分/じぶん]の[不注意/ふちゅうい]**から**[事故/じこ]を[起/お]こし、[入院/にゅういん]してしまった。' trans='因为自己的疏忽引发了事故，住院了。' />

:::
