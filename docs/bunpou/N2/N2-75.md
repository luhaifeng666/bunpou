# ～に伴って＜伴随変化＞ <Badge type="danger" text="N2" />

* 意义：表示伴随着该动作、变化的发生，相应地发生了后句叙述的变化。一般用于书面语。
* 接续：动作性名词＋に伴って／に伴うN；动词的辞典形＋（の）＋に伴って／に伴うN
* 译文：随着……

::: info :books: 例句

1. <grammer-content sentence='[時代/じだい]の[変化/へんか]**に伴う**[新/あたら]たな[考/かんが]え[方/かた]・[概念/がいねん]をカタカナを[用/もち]いて[外来語/がいらいご]として[受/う]け[入/い]れる[土壌/どじょう]があったからこそ、[日本語/にほんご]は[豊/ゆた]かになり、いっそう[魅力的/みりょくてき]な[言語/げんご]になってきたのではなかろうか。' trans='正是因为有随着时代变化而用片假名作为外来语接受新思想、新概念的土壤，日语才变得丰富，成为更有魅力的语言。' />
2. <grammer-content sentence='[子供/こども]の[成長/せいちょう]**に伴って**、ライフスタイルに[大/おお]きな[変化/へんか]が[起/お]きる。' trans='随着孩子的成长，生活方式会发生很大变化。' />
3. <grammer-content sentence='この[病気/びょうき]の[発症率/はっしょうりつ]は[年/とし]が[高/たか]くなる**に伴い**[増加/ぞうか]するそうだ。' trans='据说这种疾病的发病率随着年龄增长而增加。' />
4. <grammer-content sentence='[人口/じんこう]の[減少/げんしょう]**に伴う**[人手不足/ひとでぶそく]を[補/おぎな]うためには、[人工知能/じんこうちのう]の[活用/かつよう]が[不可欠/ふかけつ]だ。' trans='为了弥补人口减少带来的劳动力不足，人工智能的活用不可或缺。' />
