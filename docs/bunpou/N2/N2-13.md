# Nとしては＜具体化された対象＞ <Badge type="danger" text="N2" />

* 意义：表示某种抽象事物的具体内容。
* 接续：抽象名词或词组＋としては
* 译文：作为……

::: info :books: 例句

1. <grammer-content sentence='[活動内容/かつどうないよう]**としては**、[週/しゅう]2[回/かい]の[合同練習/ごうどうれんしゅう]と[朝/あさ]の[自主練習/じしゅれんしゅう]です。' trans='作为活动内容，是每周两次的共同练习和早上的自主练习。' />
2. <grammer-content sentence='[予定/よてい]**としては**、まず[上海/シャンハイ]へ[行/い]って、その[次/つぎ]に[杭州/こうしゅう]へ[行/い]きたいと[思/おも]います。' trans='作为预定，首先想去上海，然后想去杭州。' />
3. <grammer-content sentence='[日本/にほん]の[伝統文化/でんとうぶんか]**としては**、[茶道/さどう]や[能楽/のうがく]や[相撲/すもう]などが[挙/あ]げられる。' trans='作为日本的传统文化，可以举出茶道、能乐、相扑等。' />

:::
