# Vてからでないと＜必要条件＞ <Badge type="danger" text="N2" />

* 意义：表示必要条件。即如果不先进行该动作，就不能进行后面的事情。
* 接续：Vて＋からでないと
* 译文：如果不先……就不……

::: info :books: 例句

1. <grammer-content sentence='[実際/じっさい]に[会/あ]って**からでないと**、[判断/はんだん]できない。' trans='如果不实际见面，就不能判断。' />
2. <grammer-content sentence='[資料/しりょう]を[読/よ]んで**からでないと**、[議論/ぎろん]に[参加/さんか]できない。' trans='如果不先读资料，就不能参加讨论。' />
3. <grammer-content sentence='[許可/きょか]を[得/え]て**からでないと**、[入場/にゅうじょう]できない。' trans='如果不先获得许可，就不能入场。' />
4. <grammer-content sentence='お金を[払/はら]って**からでないと**、[商品/しょうひん]を[受/う]け[取/と]れない。' trans='如果不先付钱，就不能领取商品。' />

:::
