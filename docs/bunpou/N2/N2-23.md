# むしろ＜比較・選択＞ <Badge type="danger" text="N2" />

* 意义：用于表示说话人将两个事物进行比较后所选择的结果。这一结果往往是出乎人们意料或不符合常理的。
* 译文：与……相比，还是……；与其……，还不如……

::: info :books: 例句

1. <grammer-content sentence='[国際化/こくさいか]という[点/てん]からは、[相撲界/すもうかい]での[外国人/がいこくじん]の[活躍/かつやく]は**むしろ**[喜/よろこ]ばしいと[考/かんが]えられるであろう。' trans='从国际化的角度来看，相扑界外国人的活跃反而是可喜的。' />
2. <grammer-content sentence='テレビが[故障/こしょう]したが、[修理/しゅうり]するより**むしろ**[新/あたら]しいのを[買/か]ったほうが[安/やす]い。' trans='电视坏了，与其修理还不如买新的便宜。' />
3. <grammer-content sentence='[自分/じぶん]の[好/す]きな[仕事/しごと]ができれば、[大企業/だいきぎょう]よりも**むしろ**[中小企業/ちゅうしょうきぎょう]で[働/はたら]きたいという[若者/わかもの]もいる。' trans='如果能做自己喜欢的工作，也有年轻人宁愿在中小企业工作而不是大企业。' />
4. <grammer-content sentence='お[酒/さけ]は[大勢/おおぜい]で[飲/の]むのもよいのだが、[私/わたし]は**むしろ**ひとりで[飲/の]むのが[好/す]きだ。' trans='酒大家一起喝也好，但我更喜欢一个人喝。' />

:::
