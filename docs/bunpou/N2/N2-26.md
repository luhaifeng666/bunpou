# ～もかまわず＜無視＞ <Badge type="danger" text="N2" />

* 意义：表示动作主体对该事物或现象存在熟视无睹或毫不介意。
* 接续：用词＋の／名词＋もかまわず
* 译文：不管……；不顾……

::: info :books: 例句

1. <grammer-content sentence='[人目/ひとめ]**もかまわず**[良/よ]くお[化粧/けしょう]できるなって[私/わたし]も[思/おも]うけど。' trans='不管别人的目光，居然能这样化妆，我也这么觉得。' />
2. <grammer-content sentence='[酔/よ]った[男/おとこ]が[大勢/おおぜい]の[乗客/じょうきゃく]がいるの**もかまわず**、[大/おお]きな[声/こえ]で[叫/さけ]んでいる。' trans='醉汉不管有很多乘客，大声叫喊。' />
3. <grammer-content sentence='お[母/かあ]さんは[疲/つか]れているの**もかまわず**、[早速/さっそく][食事/しょくじ]の[用意/ようい]を[始/はじ]めた。' trans='妈妈不顾疲劳，马上开始准备饭菜。' />
4. <grammer-content sentence='[彼女/かのじょ]は[雨/あめ]に[濡/ぬ]れるの**もかまわず**、[傘/かさ]も[差/さ]さずに[歩/ある]いている。' trans='她不顾被雨淋，不打伞走着。' />

:::
