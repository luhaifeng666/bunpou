# ～Vずにはいられない＜不由自主＞【第7课第1单元】 <Badge type="danger" text="N2" />

* 意义：表示不由自主地进行该动作或自然而然地出现该状态，不受人的意志控制。
* 接续：Vない＋ずにはいられない　する→せ＋ずにはいられない
* 译文：不由得……；忍不住……
* 补充：ずにはいられないなくなる＝变得不由自主

::: tip :bookmark: 説明

「ず」是否定助动词「ない」的连用形。「～ずにはいられない」表示"不……就忍不住"，强调自然产生的冲动或无法抑制的动作。一般用于书面语。

:::
