# からこそ＜凸顕原因＞ <Badge type="danger" text="N2" />

* 意义：表示原因。「こそ」起到凸显该原因的功能。
* 接续：简体句子＋からこそ
* 译文：正因为……

::: info :books: 例句

1. <grammer-content sentence='[彼女/かのじょ]**からすれば**、[自分/じぶん]が[会/あ]う[人/ひと]に[対/たい]して[恥/は]じらいも[敬意/けいい]を[感/かん]じる**からこそ**[化粧/けしょう]をするということだ。' trans='从她来看，正是因为对见面的人感到羞涩和敬意才化妆的。' />
2. <grammer-content sentence='[君/きみ]のことを[心配/しんぱい]した**からこそ**、そんな[話/はなし]をしたんだよ。' trans='正因为担心你，才说了那些话。' />
3. <grammer-content sentence='[難/むずか]しい**からこそ**、チャレンジする[価値/かち]があるのだ。' trans='正因为难，才有挑战的价值。' />
4. <grammer-content sentence='[自分/じぶん]たちが[決/き]めたルールだ**からこそ**、そのルールを[尊重/そんちょう]し、[守/まも]ろうとするのだ。' trans='正因为是自己决定的规则，才会尊重并遵守那个规则。' />

:::
