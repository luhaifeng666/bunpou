# ～だからといって＜転折＞ <Badge type="danger" text="N2" />

* 意义：表示转折关系。即尽管前面的句子所述的情况为事实，但不能以此为根据推测出相应的结果。
* 译文：尽管如此，然而……

::: tip :bookmark: 説明

句尾一般为否定的表达方式。

:::

::: info :books: 例句

1. <grammer-content sentence='[京都/きょうと]の[建物/たてもの]や[庭園/ていえん]は[確/たし]かに[素晴/すば]らしい、**だからといって**[伝統/でんとう]にばかり[頼/たよ]ってはいられない。' trans='京都的建筑和庭院确实很棒，但不能因此就一味地依赖传统。' />
2. <grammer-content sentence='[毎日/まいにち][忙/いそが]しい。しかし、**だからといって**、[好/す]きな[空手/からて]を[止/や]める[つもり]はない。' trans='每天很忙。但是，尽管如此，不打算放弃喜欢的空手道。' />
3. <grammer-content sentence='この[仕事/しごと]は[好/す]きではない。**だからといって**、[簡単/かんたん]に[止/や]めることはできない。' trans='这份工作不喜欢。尽管如此，也不能简单地辞职。' />
4. <grammer-content sentence='[家/いえ]に[帰/かえ]りたくない。**だからといって**、ほかに[帰/かえ]る[場所/ばしょ]もない。' trans='不想回家。尽管如此，也没有其他可回的地方。' />

:::
