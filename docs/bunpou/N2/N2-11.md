# ～だけでは＜否定的限定＞ <Badge type="danger" text="N2" />

* 意义：用于表示对限定进行否定。
* 接续：动词、名词＋だけでは
* 译文：只进行……（不能）……；单单凭着……（不能）……

::: tip :bookmark: 説明

后句常常为否定的表达。

:::

::: info :books: 例句

1. <grammer-content sentence='[説明書/せつめいしょ]を[読/よ]む**だけでは**わからない。やはり[一度/いちど]やってみたほうがいいかな。' trans='只读说明书不明白。还是做一次比较好。' />
2. <grammer-content sentence='こんなことをしてしまったら、「ごめんなさい」と[謝/あやま]る**だけでは**不[十分/じゅうぶん]だ。' trans='做了这样的事，只说"对不起"是不够的。' />
3. <grammer-content sentence='[人/ひと]に[教/おし]える[能力/のうりょく]というのは[学歴/がくれき]**だけでは**[判断/はんだん]できないと[思/おも]う。' trans='我认为教人的能力不能仅凭学历来判断。' />
4. <grammer-content sentence='これ**だけでは**たりないよ！' trans='只有这些不够啊！' />

:::
