# Vず〈动作的否定〉 <Badge type="danger" text="N2" />

- 意义：否定动作或变化的发生，在句中用于中顿或并列。
- 接续：Vない + ず（する → せず）
- 译文：不......

::: info :books: 说明

用于书面语或惯用的表达方式，口语中使用「Vなくて」或「Vないで」。

:::
