# N1をN2とする／として＜確定＞ <Badge type="danger" text="N2" />

* 意义：表示将N1看做或确定为N2。
* 接续：名词＋を＋名词＋とする／として＋动词／とした／とする＋名词
* 译文：以……为……；把……当做……

::: info :books: 例句

1. <grammer-content sentence='[年少者/ねんしょうしゃ]を[対象/たいしょう]**とした**[日本語教育/にほんごきょういく]に[関心/かんしん]があります。' trans='对以未成年人为对象的日语教育感兴趣。' />
2. <grammer-content sentence='[日本語能力試験/にほんごのうりょくしけん]N1[合格/ごうかく]を[目標/もくひょう]**としている**。' trans='以日语能力考试N1合格为目标。' />
3. <grammer-content sentence='[登山/とざん]を[趣味/しゅみ]**として**[楽/たの]しむ[人/ひと]が[増/ふ]えている。' trans='把登山当做乐趣的人在增加。' />
4. <grammer-content sentence='[自分/じぶん]のやりたいことを[仕事/しごと]**とする**のが[一番/いちばん]よいことだ。' trans='把自己想做的事情当做工作是最好的。' />

:::
