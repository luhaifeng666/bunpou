# ～V能动ものなら<条件・愿望> <Badge type="danger" text="N2" />

* 意义：接动词能动态，表示条件，多用于假设本身发生可能性极低的事件的发生。强调说话人的愿望、期待等心情。
* 接续：动词能动态+ものなら
* 译文：如果能……的话（那么）……

::: info :books: 例句

1. <grammer-content sentence='[金钱/きんせん]に[金色/きんいろ]になれる**ものなら**[世界中/せかいじゅう]の[美味/おい]しいものを[食/た]べてみたい。' trans='如果能成为有钱人，想吃遍全世界好吃的。' />
2. <grammer-content sentence='[寝/ね]られる**もん（口）なら**[一生/いっしょう][寝/ね]ていたい。' trans='如果能一直睡的话，想睡一辈子。' />
3. <grammer-content sentence='[手伝/てつだ]える**ものなら**[手伝/てつだ]って[挙/あ]げたいが、[不幸/ふこう]にも[手/て]が[空/あ]いていない。' trans='如果能帮忙的话想帮，但偏偏手头没空。' />
4. <grammer-content sentence='[昨日/きのう]のライブ、**[行/い]けるものなら**[行/い]きたかったが[大雨/おおあめ]で[行/い]けなかった。' trans='如果昨天的演唱会能去就好了，但下大雨去不了。' />
5. <grammer-content sentence='あの[辺/あた]りの[高級住宅街/こうきゅうじゅうたくがい]に**[住/す]めるものなら**[一度/いちど]は[住/す]んでみたい。' trans='如果能住在那一带高级住宅区的话，想住一次看看。' />
:::
