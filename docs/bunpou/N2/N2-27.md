# ～かのようだ＜印象・比喻＞ <Badge type="danger" text="N2" />

* 意义：表示从外表上观察得到该印象，但事实并非如此。
* 接续：动词、形容词的简体形式＋かのようだ；名词、二类形容词词干＋である＋かのようだ
* 译文：仿佛……；似乎……

::: tip :bookmark: 説明

这是一种比喻的说法，经常与「まるで、あたかも」等副词搭配使用。

:::

::: info :books: 例句

1. <grammer-content sentence='[当/とう]の[本人/ほんにん]は[周囲/しゅうい]の[人/ひと]がまったく[目/め]に[入/はい]らない**かのように**、[平気/へいき]な[顔/かお]で[化粧/けしょう]を[続/つづ]けている。' trans='本人仿佛完全看不到周围的人，满不在乎地继续化妆。' />
2. <grammer-content sentence='このところ[暑/あつ]い[日/ひ]が[続/つづ]いている。[夏/なつ]が[戻/もど]ってきた**かのようだ**。' trans='最近炎热的日子持续着，仿佛夏天回来了。' />
3. <grammer-content sentence='その[噂/うわさ]はまるで[事実/じじつ]である**かのように**インターネットで[広/ひろ]がった。' trans='那个传闻仿佛事实一样在网上扩散了。' />
4. <grammer-content sentence='[簡単/かんたん]なことをわざわざと[難/むずか]しい[言葉/ことば]を[使/つか]って、**あたかも**[自分/じぶん]の[能力/のうりょく]が[高/たか]い**かのようだ**。' trans='把简单的事故意用难懂的话来表达，仿佛自己的能力很高。' />

:::
