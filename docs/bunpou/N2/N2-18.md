# ～だけに＜比例した因果関係＞ <Badge type="danger" text="N2" />

* 意义：表示因果关系，意为从句表达的原因与主句表达的结果在程度上是相当的，是成正比的。
* 接续：名词／动词、形容词的连体形＋だけに
* 译文：正因为……

::: tip :bookmark: 説明

主句表达说话人的评价以及感觉、感情。

:::

::: info :books: 例句

1. <grammer-content sentence='[国技/こくぎ]**だけに**[相撲界/すもうかい]にはさまざまな[伝統/でんとう]やしきたりがある。' trans='正因为是国技，相扑界才有各种各样的传统和规矩。' />
2. <grammer-content sentence='[同/おな]じ[趣味/しゅみ]を[持/も]つ[仲間/なかま]**だけに**[話/はなし]もよく[通/つう]じる。' trans='正因为是志同道合的伙伴，所以很谈得来。' />
3. <grammer-content sentence='もうだめだと[諦/あきら]めていた**だけに**、[優勝/ゆうしょう]したときは、[涙/なみだ]が[出/で]るほど[嬉/うれ]しかった。' trans='正因为已经放弃了，所以获胜时高兴得流下了眼泪。' />
4. <grammer-content sentence='あの[人/ひと]は10[年/ねん]も[アメリカ/あめりか]にいた**だけに**[英語/えいご]が[上手/じょうず]だ。' trans='正因为在美国待了10年，所以英语很好。' />
5. <grammer-content sentence='この[機械/きかい]は[仕組/しく]みが[複雑/ふくざつ]な**だけに**、[修理/しゅうり]に[時間/じかん]がかかる。' trans='正因为这台机器结构复杂，所以修理需要时间。' />

:::
