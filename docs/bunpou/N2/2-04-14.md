# 〜と言える〈提出主张〉 <Badge type="danger" text="N2" />

- 意义：用于提出主张或做出判断。
- 接续：简体句子 + と言える；II类形容词词干 / 名词 + と言える
- 译文：可以说......；可以认为......

::: info :books: 说明

多用于书面语。

:::

::: info :books: 例句

1. <grammer-content sentence='[正/まさ]に[中世/ちゅうせい][日本/にほん]のファンタジーの[世界/せかい]であり、[現代/げんだい]の[漫画/まんが]の[原点/げんてん]と[言/い]える。' trans='可以说正是中世日本的幻想世界，是现代漫画的原点。' />
2. <grammer-content sentence='これらのデータは、[世界経済/せかいけいざい]の[状況/じょうきょう]を[反映/はんえい]していると[言/い]える。' trans='可以说这些数据反映了世界经济状况。' />
3. <grammer-content sentence='[近/ちか]くに[商店街/しょうてんがい]があって、[買物/かいもの]には[非常/ひじょう]に[便利/べんり]と[言/い]えるだろう。' trans='可以说附近有商店街，购物非常方便吧。' />
4. <grammer-content sentence='パソコンを[使/つか]わない[仕事/しごと]のほうが[今/いま]では[珍/めずら]しいと[言/い]えるかもしれません。' trans='可以说现在不用电脑的工作反而很少见。' />

:::
