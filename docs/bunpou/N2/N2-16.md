# ～やら～やら＜並列＞ <Badge type="danger" text="N2" />

* 意义：用于表示同类事物的并列。
* 接续：名词／动词、形容词的词典形＋やら
* 译文：……呀，……呀

::: info :books: 例句

1. <grammer-content sentence='[練習/れんしゅう]の[方法/ほうほう]**やら**[先輩/せんぱい]、[後輩/こうはい]の[関係/かんけい]**やら**で[戸惑/とまど]いました。' trans='练习的方法呀，前辈后辈的关系呀，让我不知所措。' />
2. <grammer-content sentence='[来週/らいしゅう]は[レポート]**やら**[試験/しけん]**やら**で[忙/いそが]しくなりそうだ。' trans='下周有报告呀，考试呀，好像会很忙。' />
3. <grammer-content sentence='[今日/きょう]は[洗濯/せんたく]**やら**[部屋/へや]の[片付/かたづ]け**やら**で、あっという[間/ま]に[半日/はんにち]が[過/す]ぎてしまった。' trans='今天有洗衣服呀，收拾房间呀，转眼间半天就过去了。' />
4. <grammer-content sentence='スピーチコンテストで3[位/い]に[入賞/にゅうしょう]したとき、[嬉/うれ]しい**やら**[悔/くや]しい**やら**[複雑/ふくざつ]な[気持/きも]ちだった。' trans='在演讲比赛中获得第三名时，又高兴又遗憾，心情很复杂。' />
5. <grammer-content sentence='みんなで[温泉/おんせん]に[行/い]って、[飲/の]む**やら**[歌/うた]う**やら**、[大騒/おおさわ]ぎした。' trans='大家一起去温泉，又喝又唱，大吵大闹。' />

:::
