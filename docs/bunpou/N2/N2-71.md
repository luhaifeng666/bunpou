# ～すら～（ない）<凸显极端事例> <Badge type="danger" text="N2" />

* 意义：举出极端的事例，表达其他更不用说的含义。书面语色彩浓厚，多与谓语的否定形式一起使用。
* 接续：名词+（に/で/と）+すら
* 译文：连……都……；甚至……

::: info :books: 例句

1. <grammer-content sentence='**[仕事/しごと]が[忙/いそが]しくて、[食事/しょくじ]の[時間/じかん]すら[nari/と]りに[取/と]れない**。' trans='工作忙得连吃饭的时间都抽不出来。' />
2. <grammer-content sentence='**[高熱/こうねつ]で[体/からだが[重/おも]く、[立/た]ち[上/あ]がる[nari/こと]すら[nari/と]れない**。' trans='发高烧身体沉重，连站起来都做不到。' />
3. <grammer-content sentence='**[誰/だれ]が[一番/いちばん]になるかは、[審査員/しんさいん]にすら[予想/よそう]が[nari/と]れない**。' trans='谁会是第一名，就连评审员都无法预测。' />
4. <grammer-content sentence='**[不況/ふきょう]のため、[高/こう][学歴/がくれき]の[nari/ひと]でさえ[nari/と]に[仕事/しごと]が[nari/と]，[見/み]つからない**。' trans='因为不景气，就连高学历的人都找不到工作。' />
:::
