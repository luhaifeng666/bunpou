# ～すら＜凸顕極端事例＞ <Badge type="danger" text="N2" />

* 意义：举出极端的事例，表达其他更不用说的含义。与「さえ」相比，书面语色彩浓厚，多与谓语的否定形式一起使用。
* 接续：名词＋（に／で／と）＋すら
* 译文：连……都……；甚至……

::: info :books: 例句

1. <grammer-content sentence='[日本人/にほんじん]にとって**すら**[簡単/かんたん]ではない[外来語/がいらいご]の[理解/りかい]・[使用/しよう]は、[日本語/にほんご][学習者/がくしゅうしゃ]にとってはさらに[困難/こんなん]を[極/きわ]めるものとなるのではないか。' trans='就连对日本人来说都不简单的外来语理解和使用，对日语学习者来说更是难上加难。' />
2. <grammer-content sentence='[仕事/しごと]が[忙/いそが]しくて、[食事/しょくじ]の[時間/じかん]**すら**[十分/じゅうぶん]に[取/と]れない。' trans='工作忙得连吃饭的时间都抽不出来。' />
3. <grammer-content sentence='[高熱/こうねつ]で[体/からだ]が[重/おも]く、[立/た]ち[上/あ]がること**すら**できない。' trans='发高烧身体沉重，连站起来都做不到。' />
4. <grammer-content sentence='[誰/だれ]が[一番/いちばん]になるかは、[審査員/しんさいん]に**すら**[予想/よそう]ができない。' trans='谁会是第一名，就连评审员都无法预测。' />
5. <grammer-content sentence='[不景気/ふけいき]のため、[高学歴/こうがくれき]の[人/ひと]で**すら**[仕事/しごと]が[見/み]つからない。' trans='因为不景气，就连高学历的人都找不到工作。' />

:::
