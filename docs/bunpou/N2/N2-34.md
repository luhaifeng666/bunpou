# Nを問わず＜無区別＞ <Badge type="danger" text="N2" />

* 意义：表示无论事物的性质如何，谓语所指的情况都是成立的。
* 接续：名词＋を問わず
* 译文：无论……；不管……

::: tip :bookmark: 説明

名词一般为包含多个选项含义的「年齢／季節／国籍／理由」等或包含对立含义的「男女／有無／昼夜／大小」等。

:::

::: info :books: 例句

1. <grammer-content sentence='[季節/きせつ]**を問わず**、いつでも[利用/りよう]できる。' trans='无论季节，随时都可以使用。' />
2. <grammer-content sentence='18[歳/さい]以上なら、[経験/けいけん]や[能力/のうりょく]**を問わず**[誰/だれ]でも[参加/さんか]できる。' trans='18岁以上，无论经验和能力，谁都可以参加。' />
3. <grammer-content sentence='この[本/ほん]は、[年齢/ねんれい]、[男女/だんじょ]**を問わず**、[大勢/おおぜい]の[人々/ひとびと]に[読/よ]まれている。' trans='这本书无论年龄、男女，被很多人阅读。' />
4. <grammer-content sentence='インターネットを[使/つか]えば、[場所/ばしょ]、[時間/じかん]**を問わず**、いろんな[人/ひと]と[会話/かいわ]ができる。' trans='使用互联网，无论地点、时间，可以和很多人对话。' />

:::
