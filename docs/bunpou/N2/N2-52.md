# ～といえば＜让步＞ <Badge type="danger" text="N2" />

* 意义：前后连接同一个词语，表示对结论的消极认可，意为虽不否认该事实，但是存在与预想相反的情况。
* 接续：名词／动词＋といえば
* 译文：要说……确实……；提到……

::: info :books: 例句

1. <grammer-content sentence='[便利/べんり]**といえば**[日本/にほん]ほど[便利/べんり]な[国/くに]はないだろう。' trans='要说便利，没有比日本更便利的国家吧。' />
2. <grammer-content sentence='[怖/こわ]い**といえば**[映画/えいが]で[見/み]たあの[幽霊/ゆuirei]が[頭/あたま]に[浮/う]かぶ。' trans='要说可怕，浮现出在电影里看到的幽灵。' />
3. <grammer-content sentence='[夏/なつ]**といえば**[海/うみ]だが、[今年/ことし]は[行/い]かなかった。' trans='提到夏天确实是海，但今年没去。' />
4. <grammer-content sentence='[日本語/にほんご]**といえば**[難/むずか]しいと[言/い]う[人/ひと]がいますが、[実際/じっさい]は[文法的/ぶんぽうてき]には[体系的/たいけいてき]に[整理/せいり]されています。' trans='有人说日语很难，但实际上语法上是有体系地整理过的。' />

:::
