# ～Nじゃあるまいし＜否定性原因＞ <Badge type="danger" text="N2" />

* 意义：表示否定性的原因、理由，主句一般为说话人的判断、反问、祈使、责备、推测等。
* 接续：名词＋じゃ（では）あるまいし
* 译文：又不是……，（所以）……

::: tip :bookmark: 説明

「まい」为否定推测/否定意志助动词，「Nじゃあるまいし」即「Nではないだろうし」，表示"因为并不是N，所以……"。

:::

::: info :books: 例句

1. <grammer-content sentence='[物々交換/ぶつぶつこうかん]**じゃあるまいし**。' trans='又不是以物易物。' />
2. <grammer-content sentence='[子供/こども]**じゃあるまいし**、[自分/じぶん]のことは[自分/じぶん]で[決/き]めてください。' trans='又不是小孩子，自己的事情请自己决定。' />
3. <grammer-content sentence='[神様/かみさま]**じゃあるまいし**、[事故/じこ]を[予知/よち]して[防/ふせ]ぐなんて[無理/むり]な[話/はなし]だ。' trans='又不是神，预知并阻止事故这种事根本不可能。' />
4. <grammer-content sentence='コンピューター**ではあるまいし**、そんなにたくさん[頭/あたま]に[入/はい]らないよ。' trans='又不是电脑，记不了那么多东西的。' />
5. <grammer-content sentence='そんなに[泣/な]かないでよ。[子供/こども]**じゃあるまいし**。' trans='别哭成那样嘛，又不是小孩子。' />

:::
