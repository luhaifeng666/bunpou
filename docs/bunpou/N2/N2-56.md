# ～Vてこそ＜凸显＞ <Badge type="danger" text="N2" />

* 意义：表示对动作、状态的凸显、强调，意为只有通过做某事或在某一状态下，才会有积极的意义或好的结果。
* 接续：Vてこそ
* 译文：只有……，才……；正因为……，才……

::: info :books: 例句

1. <grammer-content sentence='[伝/つた]えたいことがあって**こそ**の[言葉/ことば]、なのである。' trans='正因为有想要传达的事情，才有了语言的存在。' />
2. <grammer-content sentence='[外国語/がいこくご]は[声/こえ]を[出/だ]して**話してこそ**[上達/じょうたつ]するのである。' trans='外语只有大声说出来才能进步。' />
3. <grammer-content sentence='[団結/だんけつ]こそ[力/ちから]であり、**団結してこそ**[勝利/しょうり]することができる。' trans='团结就是力量，只有团结才能取得胜利。' />
4. <grammer-content sentence='みんなで[協力/きょうりょく]して**こそ**よい[仕事/しごと]ができる。' trans='只有大家一起协作，才能做好工作。' />
