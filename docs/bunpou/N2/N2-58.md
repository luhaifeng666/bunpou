# Vようがない／ようもない＜无法＞ <Badge type="danger" text="N2" />

* 意义：表示用任何方法、手段都无法完成某事。
* 接续：动词第一连用形＋ようがない／ようもない
* 译文：无法……

::: tip :bookmark: 説明

后面接名词时，有时用「VようのないN」的形式。惯用语：どうしようもない（无论如何都改变不了、为时已晚）。

:::

::: info :books: 例句

1. <grammer-content sentence='そしてスーッと[流/なが]れる[涙/なみだ]を[見/み]たとき、[私/わたし]は**いいようのない**[感動/かんどう]と[恐/おそ]ろしさを[覚/おぼ]えた。' trans='看到眼泪唰地流下来的时候，我感受到了难以言表的感动和恐惧。' />
2. <grammer-content sentence='メールアドレスも[電話番号/でんわばんごう]も[知/し]らないので、こちらから**連絡れんらくしようがありません**。' trans='邮箱和电话都不知道，所以这边无法联系。' />
3. <grammer-content sentence='[地震/じしん]のような[自然災害/しぜんさいがい]は[人間/にんげん]の[力/ちから]では**防ふせぎようがない**。' trans='地震这样的自然灾害，凭人类的力量是无法预防的。' />
4. <grammer-content sentence='[終/お]わったことは[後悔/こうかい]しても**どうしようもない**。' trans='已经结束的事情，后悔也无济于事。' />
:::
