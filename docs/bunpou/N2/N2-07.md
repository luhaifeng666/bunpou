# ～ということは～だ＜解釈・説明＞ <Badge type="danger" text="N2" />

* 意义：用于对某一事物或事项进行解释、说明，这种说明大多为说话人的推理、判断或评价。
* 接续：简体句子＋ということは＋（简体句子＋ということ）だ
* 译文：……就是……；……就是说……

::: tip :bookmark: 説明

在对话中也用于对说话人根据某一事实做出判断并向听话人确认。

:::

::: info :books: 例句

1. <grammer-content sentence='[自分/じぶん]で[自分/じぶん]の[時間割/じかんわり]を[作/つく]る**ということは**、その[作業/さぎょう]を[通/とお]じて、[自主性/じしゅせい]、[主体性/しゅたいせい]を[確立/かくりつ]していく、**ということでもある**。' trans='自己制定自己的课表，就是说通过这个过程来确立自主性和主体性。' />
2. <grammer-content sentence='[大人/おとな]になる**ということは**[自分/じぶん]の[行動/こうどう]に[責任/せきにん]を[持/も]つ**ということだ**。' trans='成为大人就是对自己的行为负责。' />
3. <grammer-content sentence='[地元/じもと]の[人/ひと]がよく[店/みせ]に[来/く]る**ということは**、[美味/おい]しい**でしょう**。' trans='当地人都常来这家店，就是说很好吃吧。' />
4. <grammer-content sentence='Ａ：[雨/あめ][降/ふ]ってるよ。Ｂ：え？**ということは**、[午後/ごご]の[試合/しあい]は[中止/ちゅうし]なのか。' trans='A：下雨了。B：诶？就是说下午的比赛要取消吗？' />

:::
