# ～ばかりか＜付加・递進＞ <Badge type="danger" text="N2" />

* 意义：表示附加、递进的关系。
* 接续：名词／动词、形容词的连体形＋ばかりか／ばかりでなく
* 译文：不止……；岂止……

::: tip :bookmark: 説明

此句式突出的是主句的内容。主句的句子成分（主要是主语）一般要使用「も・さえ・まで」等凸显助词呼应。

:::

::: info :books: 例句

1. <grammer-content sentence='[実際/じっさい]、[最近/さいきん]では[車内/しゃない]で[化粧/けしょう]をする[女性/じょせい]**ばかりか**、[床/ゆか]に[直接/ちょくせつ][座/すわ]り[込/こ]んでいる[若者/わかもの]や、カップめんを[食/た]べるもの**まで**[見/み]かけるようになった。' trans='实际上最近不仅是女性，甚至坐在地板上的年轻人和吃泡面的人都能看到了。' />
2. <grammer-content sentence='[最近/さいきん]では、[大学/だいがく]**ばかりか**[高校/こうこう]や[中学校/ちゅうがっこう]でも[海外/かいがい]の[学校/がっこう]との[交流/こうりゅう]が[盛/さか]んになっている。' trans='最近不仅是大学，高中和初中与海外学校的交流也变得活跃。' />
3. <grammer-content sentence='[学校/がっこう]に[行/い]けない**ばかりか**、[食事/しょくじ]も[満足/まんぞく]に[食/た]べることができない[子供/こども]がかなりいるようだ。' trans='似乎有不少孩子不仅不能上学，连饭都不能吃饱。' />
4. <grammer-content sentence='[彼/かれ]はアメリカ**ばかりか**ヨーロッパへの[留学経験/りゅうがくけいけん]**も**ある。' trans='他不仅有美国的留学经验，还有欧洲的。' />
