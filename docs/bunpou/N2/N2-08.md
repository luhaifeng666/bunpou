# ～とともに＜共同動作主体／同時＞ <Badge type="danger" text="N2" />

* 意义：①表示共同动作的主体；②表示两种状态同时存在。
* 接续：名词＋とともに／名词＋である／二类形容词词干（＋である）／动词、一类形容词的连体形＋とともに
* 译文：①与……一起；②在……的同时

::: info :books: 例句

1. <grammer-content sentence='[同/おな]じ[関心/かんしん]を[持/も]つ[先輩/せんぱい]や[学友/がくゆう]**とともに**、さまざまな[角度/かくど]から[学/まな]んでいくのである。' trans='与对同一课题感兴趣的前辈和学友一起，从各个角度进行学习。' />
2. <grammer-content sentence='[週末/しゅうまつ]は[家族/かぞく]**とともに**[過/す]ごしたい。' trans='周末想和家人一起度过。' />
3. <grammer-content sentence='[両親/りょうしん]**とともに**[初/はじ]めて[海外/かいがい]に[滞在/たいざい]したのはニュージーランドだった。' trans='第一次和父母一起在国外生活是在新西兰。' />
4. <grammer-content sentence='[地域/ちいき]の[住民/じゅうみん]は、[地域/ちいき]の[子供/こども]たちの[活動/かつどう]を[学校/がっこう]**とともに**[支援/しえん]している。' trans='当地居民与学校一起支持地区孩子们的活动。' />
5. <grammer-content sentence='[不安/ふあん]**とともに**[期待/きたい]があった。' trans='在不安的同时也有期待。' />
6. <grammer-content sentence='あの[人/ひと]は[学者/がくしゃ]である**とともに**、[教育者/きょういくしゃ]でもある。' trans='那个人既是学者，也是教育者。' />
7. <grammer-content sentence='[日本語/にほんご]を[勉強/べんきょう]する**とともに**[経済学/けいざいがく]の[授業/じゅぎょう]にも[出/で]ている。' trans='在学习日语的同时，也在上经济学的课。' />
8. <grammer-content sentence='[多/おお]くの[方々/かたがた]に[協力/きょうりょく]していただき、[嬉/うれ]しい**とともに**[感謝/かんしゃ]の[気持/きも]ちでいっぱいです。' trans='得到很多人的帮助，既高兴又充满感激之情。' />

:::
