# Nにしてみれば＜看法＞ <Badge type="danger" text="N2" />

* 意义：用来叙述由于不同的身份、立场而持有的不同的看法、判断。
* 接续：指人名词＋にしてみれば
* 译文：在……看来

::: info :books: 例句

1. <grammer-content sentence='[保守的/ほしゅてき]な[私/わたし]**にしてみれば**、[京都/きょうと]の[景観/けいかん]の[変化/へんか]は[残念/ざんねん]でならない。' trans='在保守的我看来，京都景观的变化令人遗憾不已。' />
2. <grammer-content sentence='[親/おや]**にしてみれば**いくつになっても[子/こ]は[子/こ]なのだ。' trans='在父母看来，无论多大孩子还是孩子。' />
3. <grammer-content sentence='[本人/ほんにん]**にしてみれば**[当/あた]たり[前/まえ]でも、ほかの[人/ひと]にはそうでないということがたくさんある。' trans='在本人看来理所当然的事，对其他人来说往往不是这样。' />
4. <grammer-content sentence='[日本人/にほんじん]にとっては[簡単/かんたん]なことかもしれないけれど、[外国人/がいこくじん]**にしてみれば**[大変/たいへん]なことでしょう。' trans='对日本人来说可能很简单的事，但在外国人看来是很困难的事吧。' />

:::
