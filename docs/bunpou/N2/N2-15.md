# Nに加えて＜递进・累加＞ <Badge type="danger" text="N2" />

* 意义：表示在某事物、行为的基础上增加了其他的事物、行为。
* 接续：名词或名词化的小句＋に加えて
* 译文：除了……还……；加之……

::: info :books: 例句

1. <grammer-content sentence='みなさんは[大学/だいがく]に[入学/にゅうがく]して、[英語/えいご]**に加えて**[新/あら]たな[外国語/がいこくご]を[習/なら]い[始/はじ]め、[世界/せかい]が[広/ひろ]がったことでしょう。' trans='大家进入大学后，除了英语还开始学习新的外语，世界应该变得更广阔了吧。' />
2. <grammer-content sentence='[日本語/にほんご]による[普通/ふつう]の[面接/めんせつ]**に加えて**、[英語/えいご]による[面接/めんせつ]をする[企業/きぎょう]が[増/ふ]えています。' trans='除了日语的普通面试，进行英语面试的企业也在增加。' />
3. <grammer-content sentence='お[祭/まつ]りは[見/み]る[楽/たの]しさ**に加えて**[元気/げんき]も[与/あた]えてくれます。' trans='祭典除了观赏的乐趣，还能给予元气。' />
4. <grammer-content sentence='[元気/げんき]なお[年寄/としよ]りが[増/ふ]えたのは、[医療技術/いりょうぎじゅつ]が[進歩/しんぽ]したの**に加えて**[個人/こじん]の[健康管理意識/けんこうかんりいしき]も[高/たか]まったからだろう。' trans='健康的老年人增加，除了医疗技术的进步，个人的健康管理意识也提高了。' />
