# Nのことだから＜人物の判断＞ <Badge type="danger" text="N2" />

* 意义：表示根据说话人所了解的该人物的性格或行为特征，可以推断出主句表达的结果。
* 接续：指人（有时为组织、场所的）名词＋のことだから
* 译文：因为是……的人，所以……

::: tip :bookmark: 説明

前接名词为说话人比较了解的人或组织等，主句句尾一般为「～だろう／と思う／はずだ／かもしれない」等，表达说话人的推测、判断。

:::

::: info :books: 例句

1. <grammer-content sentence='[三好/みよし]さん**のことだから**、また[寝坊/ねぼう]した**んじゃない**？' trans='因为是三好，又睡过头了吧？' />
2. <grammer-content sentence='[積極的/せっきょくてき]な[加藤/かとう]さん**のことだから**、やれる**かもしれない**。' trans='因为是积极的加藤，也许能做到。' />
3. <grammer-content sentence='[頑張/がんば]り[屋/や]の[彼/かれ]**のことだから**、きっと[合格/ごうかく]する**と思いますよ**。' trans='因为他是很努力的人，我觉得一定能合格。' />
4. <grammer-content sentence='[田舎/いなか]**のことだから**、[二人/ふたり]の[噂/うわさ]はすぐ[広/ひろ]まる**だろう**。' trans='因为是乡下，两个人的传闻很快就会传开吧。' />

:::
