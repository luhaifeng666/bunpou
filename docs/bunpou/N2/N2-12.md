# N1をN2に＜主観的に見做す＞ <Badge type="danger" text="N2" />

* 意义：表示将N1设定为或当做N2之意。N2多为「目標、機、きっかけ、目的、中心、モットー」等抽象名词。
* 接续：N1＋を＋N2＋に（＋して）
* 译文：以……为……

::: info :books: 例句

1. <grammer-content sentence='「テニスを[楽/たの]しもう」をモットー**に**、[何/なに]よりもテニスが[好/す]きな[人間/にんげん]が[集/あつ]まったクラブです。' trans='以"享受网球"为宗旨，聚集了一群最喜欢网球的人的俱乐部。' />
2. <grammer-content sentence='われわれ[空手部/からてぶ]は[地区大会/ちくたいかい][団体戦/だんたいせん][優勝/ゆうしょう]を[目標/もくひょう]**に**、[稽古/けいこ]に[取/と]り[組/く]んでいます。' trans='我们空手道部以地区大会团体赛优胜为目标，努力练习。' />
3. <grammer-content sentence='さあ、[大学入学/だいがくにゅうがく]を[機/き]**に**、[手話/しゅわ]を[始/はじ]めましょう。' trans='来吧，以进入大学为契机，开始学手语吧。' />
4. <grammer-content sentence='[彼女/かのじょ]は1[か月/かげつ]2キロを[目標/もくひょう]**に**、ダイエットしている。' trans='她以一个月2公斤为目标在减肥。' />
5. <grammer-content sentence='[設立/せつりつ]5[周年/しゅうねん]を[機/き]**に**、[社名/しゃめい]を[変更/へんこう]した。' trans='以成立5周年为契机，更改了公司名称。' />
6. <grammer-content sentence='[海外生活/かいがいせいかつ]の[経験/けいけん]を[生/い]かし、[元気/げんき]と[笑顔/えがお]をモットー**に**[頑張/がんば]りたい。' trans='想活用海外生活的经验，以元气和笑容为座右铭努力。' />

:::
