# Nに基づいて／に基づき＜基准＞ <Badge type="danger" text="N2" />

* 意义：表示实施动作、行为的基础、依据。
* 接续：名词＋に基づいて／に基づき
* 译文：基于……；依据……

::: tip :bookmark: 説明

后接名词时，使用「N1に基づくN2」或「N1に基づいたN2」的形式。

:::

::: info :books: 例句

1. <grammer-content sentence='[社会/しゃかい]の[一定/いってい]のルール**に基づいて**、それは[目的/もくてき][達成/たっせい]のための[有効/ゆうこう]な[手段/しゅだん]となるが……' trans='基于社会的一定规则，它成为达成目的的有效手段……' />
2. <grammer-content sentence='[収集/しゅうしゅう]したデータ**に基づいて**、[事故/じこ]の[原因/げんいん]を[分析/ぶんせき]する。' trans='基于收集的数据，分析事故原因。' />
3. <grammer-content sentence='この[施設/しせつ]は、[残/のこ]された[資料/しりょう]**に基づき**、[昔/むかし]の[様子/ようす]が[再現/さいげん]されている。' trans='这个设施根据留下的资料，再现了过去的样子。' />
4. <grammer-content sentence='[複雑/ふくざつ]な[状況/じょうきょう]のなかでも、[正/ただ]しい[情報/じょうほう]と[事実/じじつ]**に基づいた[判断/はんだん]**が必要だ。' trans='即使在复杂的情况下，也需要基于正确信息和事实进行判断。' />
:::
