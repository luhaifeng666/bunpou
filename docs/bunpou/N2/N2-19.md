# ～はもちろん～（も）＜代表性事物＞ <Badge type="danger" text="N2" />

* 意义：表示前项所指的代表性事物自不用说，另有同类其他事物也具有后项谓语所示的特征。
* 接续：用词＋の／名词＋はもちろん
* 译文：……就不必说，……（也）……

::: tip :bookmark: 説明

有时也用「～はもちろんのこと」。

:::

::: info :books: 例句

1. <grammer-content sentence='[相撲/すもう]の[世界/せかい]では、[力士/りきし]**はもちろん**[行司/ぎょうじ]や[呼/よ]び[出/だ]しなど、[取/と]り[組/く]みに[関係/かんけい]する[仕事/しごと]はすべて[男性/だんせい]が[行/おこな]っている。' trans='在相扑界，力士自不必说，行司和呼出等与比赛相关的工作都由男性担任。' />
2. <grammer-content sentence='[田中/たなか]さんは[英語/えいご]**はもちろん**、[中国語/ちゅうごくご]もフランス語も[話/はな]せる。' trans='田中英语自不必说，中文和法语也会说。' />
3. <grammer-content sentence='この[博物館/はくぶつかん]は[週末/しゅうまつ]**はもちろんのこと**、[平日/へいじつ]も[多/おお]くの[人/ひと]が[訪/おとず]れている。' trans='这个博物馆周末自不必说，平日也有很多人来参观。' />
4. <grammer-content sentence='このかばんは、[値段/ねだん]が[安/やす]いの**はもちろん**、デザインも[可愛/かわい]い。' trans='这个包价格便宜自不必说，设计也很可爱。' />
5. <grammer-content sentence='[綺麗/きれい]な[景色/けしき]を[見/み]るの**はもちろん**、[地元/じもと]の[料理/りょうり]を[味/あじ]わうのも[旅行/りょこう]の[楽/たの]しみの[一/ひと]つだ。' trans '欣赏美丽的景色自不必说，品尝当地美食也是旅行的乐趣之一。' />
