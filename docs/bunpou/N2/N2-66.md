# Nに応じて<根据> <Badge type="danger" text="N2" />

* 意义：表示根据该事物的性质、内容不同而采取相应的措施。后面一般接自然而然产生的结果。
* 接续：名词+に応じて
* 译文：随着……；根据……；应……

::: info :books: 例句

1. <grammer-content sentence='[必要/ひつよう]**に応じて**[この/この]二つを[使/つか]い[分/わ]ける[所/ところ]が[日本語/にほんご]の[ユニーク/ゆにーく]な[点/てん]だと[思/おも]います。' trans='根据需要区分使用这两种，我认为这是日语独特的地方。' />
2. <grammer-content sentence='[参加/さんか][人数/にんずう]**に応じて**[場所/ばしょ]を[変更/へんこう]する[可能/かのう]性が[ある/あります]。' trans='可能会根据参加人数更改场地。' />
3. <grammer-content sentence='お[客/きゃく]様の**[要求/ようきゅう]に応じて**[様々/さまざま]な[サービス/さーびす]を[提供/ていきょう]します。' trans='根据客户的要求提供各种服务。' />
4. <grammer-content sentence='あの[店/みせ]は**[季節/きせつ]に応じて**[メニュー/めにゅー]を[変/か]えている。这也是[人気/にんき]の[理由/りゆう]の[一/ひと]つだ。' trans='那家店根据季节变换菜单。这也是受欢迎的秘诀之一。' />
:::
