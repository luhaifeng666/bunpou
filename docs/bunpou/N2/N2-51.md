# ～くせに＜転折＞ <Badge type="danger" text="N2" />

* 意义：构成表示转折关系的从句，主句表示说话人的不满、责备、轻蔑等语气。
* 接续：名词＋の／动词、形容词的连体形＋くせに
* 译文：明明……，却……

::: tip :bookmark: 説明

从句与主句必须是同一主语。「くせに」带有责备、不满的语气，不能用于自己的情况。

:::

::: info :books: 例句

1. <grammer-content sentence='[彼/かれ]は[知/し]っている**くせに**、[教/おし]えてくれない。' trans='他明明知道，却不告诉我。' />
2. <grammer-content sentence='[子供/こども]の**くせに**、[大人/おとな]ぶる。' trans='明明是孩子，却装大人。' />
3. <grammer-content sentence='[彼女/かのじょ]は[お金/おかね]がない**くせに**、ブランド[品/ひん]ばかり[買/か]っている。' trans='她明明没钱，却只买名牌商品。' />
4. <grammer-content sentence='[自分/じぶん]は[何/なに]もしない**くせに**、[他人/たにん]に[文句/もんく]ばかり[言/い]う。' trans='自己什么都不做，却总是对别人发牢骚。' />

:::
