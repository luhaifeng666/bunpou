# Vるわけにはいかない＜不能＞ <Badge type="danger" text="N2" />

* 意义：表示由于某种理由，或受一般常识、社会观念及经验的制约，不能或无法进行该动词所指的动作。
* 接续：动词的词典形或否定形＋わけにはいかない
* 译文：不能……；不好……；不应该……；无法……

::: info :books: 例句

1. <grammer-content sentence='[日本人/にほんじん]の[入門者/にゅうもんしゃ]が[少/すく]なくなってきている[状況/じょうきょう]で、[今/いま]や、[外国人力士/がいこくじんりきし]を[抜/ぬ]きにして[相撲/すもう]を[話題/わだい]にする**わけにはいかない**。' trans='在日本入门者越来越少的情况下，现在不能抛开外国人力士来谈论相扑。' />
2. <grammer-content sentence='[大切/たいせつ]な[会議/かいぎ]ですから、[欠席/けっせき]する**わけにはいきません**。' trans='因为是重要的会议，所以不能缺席。' />
3. <grammer-content sentence='お[金/かね]が[必要/ひつよう]なので、アルバイトを[休/やす]む**わけにはいかない**。' trans='因为需要钱，所以不能请假。' />
4. <grammer-content sentence='[明日/あした][試験/しけん]があるので、[復習/ふくしゅう]しない**わけにはいかない**。' trans='因为明天有考试，所以不能不复習。' />
5. <grammer-content sentence='みんなに[期待/きたい]されているから、この[仕事/しごと]を[途中/とちゅう]で[止/や]める**わけにはいかない**。' trans='因为被大家期待着，所以不能中途放弃这个工作。' />
