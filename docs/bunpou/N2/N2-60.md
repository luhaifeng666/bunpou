# Nのところ（を）<对方所处的状态> <Badge type="danger" text="N2" />

* 意义：表示对方所处的状态，谓语部分多为表示感谢、道歉或请求的词语。
* 接续：名词+の/形容词的连体形+ところ（を）
* 译文：在您……的时候

::: tip :bookmark: 説明

名词、形容词前一般带前缀「お/ご」，以表示对对方的敬意。

:::

::: info :books: 例句

1. <grammer-content sentence='お[忙/いそが]しい**ところ（を）**ありがとうございます。' trans='在您百忙之中，非常感谢。' />
2. <grammer-content sentence='お[疲/つか]れの**ところを**[いろ/いろ]教えていただきありがとうございます。' trans='在您辛苦的时候承蒙您多方指教，非常感谢。' />
3. <grammer-content sentence='お[休/やす]みの**ところを**[恐れ入/おそれい]りますが、[乗車券/じょうしゃけん]を[拝見/はいけん]いたします。' trans='在您休息的时候打扰您不好意思，请让我看一下您的车票。' />
:::
