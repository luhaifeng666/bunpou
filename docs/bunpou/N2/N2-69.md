# ～にすぎない＜程度低＞ <Badge type="danger" text="N2" />

* 意义：表示说话人的价值判断，即认为该事物数量小、程度低，或动作、性质状态不过如此之意。
* 接续：名词／Ⅱ类形容词词干／动词、Ⅰ类形容词的连体形＋にすぎない
* 译文：不过……；仅仅……

::: info :books: 例句

1. <grammer-content sentence='[外来語/がいらいご]の[割合/わりあい]は5%[程度/ていど]**にすぎない**という[報告/ほうこく]もあるそうです。' trans='也有报告称外来语的比例不过5%左右。' />
2. <grammer-content sentence='[私/わたし]は[当/あ]たり[前/まえ]のことを[言/い]った**にすぎない**。' trans='我只是说了理所当然的话。' />
3. <grammer-content sentence='そんなことをするのは[時間/じかん]と[金/かね]の[無駄/むだ]**にすぎない**。' trans='做那种事情不过是浪费时间金钱。' />
4. <grammer-content sentence='あの[話/はなし]はあくまで[噂/うわさ]**にすぎず**、[実際/じっさい]のところはわからない。' trans='那只是传言，不知道实际情况。' />

:::
