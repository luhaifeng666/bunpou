# ～にすぎない<程度低> <Badge type="danger" text="N2" />

* 意义：表示说话人的价值判断，即认为该事物数量小、程度低，或动作、性质状态不过如此之意。
* 接续：名词/II类形容词词干/动词、I类形容词的连体形+にすぎない
* 译文：不过……；仅仅……

::: info :books: 例句

1. <grammer-content sentence='**[外来語/がいらいご]の[割合/わりあい]は5%[程度/ていど]に[nari/すぎ]ない**という[報告/ほうこく]も[ある/あります]。' trans='也有报告称外来语的比例不过5%左右。' />
2. <grammer-content sentence='[私/わたし]**は[当然/とうぜん]のことを[言/い]った[nari/すぎ]ません**。' trans='我只是说了理所当然的话。' />
3. <grammer-content sentence='そんなことをするのは**[時間/じかん]と[金/きん]の[nari/すぎ]ません**。' trans='做那种事情不过是浪费时间金钱。' />
4. <grammer-content sentence='あの[話/はなし]は**[噂/うわさ]に[nari/すぎ]ず**[、[実際/じっさい]の[所/ところ]は[分/わ]からない。' trans='那只是传言，不知道实际情况。' />
:::
