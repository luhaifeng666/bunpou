# Vる一方（で）／その一方（で）＜もう一方情況＞ <Badge type="danger" text="N2" />

* 意义：用于表示对一个事物的两个方面进行对比说明。表示除了前面叙述的情况以外，同时还存在着另外一种情况。
* 接续：动词的词典形＋一方（で）；句子＋その一方（で）＋句子
* 译文：一方面……，而另一方面……

::: info :books: 例句

1. <grammer-content sentence='[彼/かれ]は[仕事/しごと]に[集中/しゅうちゅう]する**一方（で）**、[家族/かぞく]の[看護/かんご]もしている。' trans='他一方面专注于工作，同时也照顾家人。' />
2. <grammer-content sentence='**その一方**[相撲/すもう]が[柔道/じゅうどう]と[同/おな]じように[国際化/こくさいか]していると[言/い]う[見方/みかた]もあります。' trans='另一方面也有人认为相扑和柔道一样正在国际化。' />
3. <grammer-content sentence='[僕/ぼく]は[一所懸命/いっしょうけんめい][仕事/しごと]をとする**一方**、[休日/きゅうじつ]には[思/おも]い[切/き]り[遊/あそ]んでいる。' trans='我一方面拼命工作，休息日则尽情地玩。' />
4. <grammer-content sentence='よく[勉強/べんきょう]する**一方で**、[趣味/しゅみ]も[楽/たの]しむ[学生/がくせい]が[増/ふ]えている。' trans='一方面努力学习、一方面享受兴趣的学生在增加。' />
5. <grammer-content sentence='[人々/ひとびと]の[生活/せいかつ]は[便利/べんり]になった。**その一方で**[環境/かんきょう]が[悪化/あっか]している。' trans='人们的生活变得便利了，另一方面环境也在恶化。' />

:::
