# ～とともに<变化> <Badge type="danger" text="N2" />

* 意义：表示一方发生变化，另一方也随之发生变化。书面语。
* 接续：表示变化的动作性名词/动词的辞典形+とともに
* 译文：随着……；……的同时

::: info :books: 例句

1. <grammer-content sentence='**[辛/つら]かった[記憶/きおく]は、[時間/じかん]**ともに**[忘/わす]られて、[楽/たの]しい[思い出/おもいで]だけが[残/のこ]されるものだ。' trans='痛苦的记忆会随着时间被遗忘，只留下快乐的回忆。' />
2. <grammer-content sentence='**[グローバル化/ぐろーはるか]の[進展/しんてん]や[new/あたら]しい[技術/ぎじゅつ]の[開発/かいはつ]**とともに**[次/つぎつぎに]new[言葉/ことば]が[現/あらわ]れている。' trans='随着全球化的进展和新技术的开发，新词语不断出现。' />
3. <grammer-content sentence='**[経済/けいざい]の[発展/はってん]**とともに**[国民/こくみん]の[生活/せいかつ][様式/ようしき]も[変/か]わってきた。' trans='随着经济的发展，国民的生活方式也发生了变化。' />
4. <grammer-content sentence='**[社会/人/しゃかいじん]になる**とともに**[一人/ひとり][暮/く]らしが[始/はじ]まった**。' trans='成为社会人的同时，开始了独自生活。' />
:::
