# ～Vつつ<同时进行> <Badge type="danger" text="N2" />

* 意义：表示动作与后句动词所指称的动作为同时关系。
* 接续：动词第一连用形+つつ
* 译文：……的同时，……

::: tip :bookmark: 説明

一般用于书面语，相当于口语的「ながら」。

:::
