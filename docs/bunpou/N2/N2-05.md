# ～といっても＜補足説明＞ <Badge type="danger" text="N2" />

* 意义：表示转折，意为实际情况达不到对方想象的程度，主句用来对从句所指的事项进行修正、补充、限定性的说明。
* 接续：名词／动词、形容词的简体形式＋といっても
* 译文：虽说……，但是（实际上）……

::: tip :bookmark: 説明

也可以与前面的句子分开，独立使用。

:::

::: info :books: 例句

1. <grammer-content sentence='[劉/りゅう]：「[三好学/みよしがく]」。[王/おう]：えっ、いいお[名前/なまえ]ですね。[三好/みよし]：**といっても**、スポーツしかできない「[一好/いっこう]」で……。' trans='刘："三好学"。王：啊，好名字啊。三好：虽说如此，但我是只会运动的"一好"……' />
2. <grammer-content sentence='9[月/がつ]**といっても**、まだまだ[暑/あつ]い[日/ひ]が[続/つづ]いている。' trans='虽说到了9月，但炎热的日子还在继续。' />
3. <grammer-content sentence='よく[行/い]く**といっても**、1[か月/かげつ]に[一度/いちど][程度/ていど]だけど。' trans='虽说经常去，但也就一个月一次左右。' />
4. <grammer-content sentence='[安/やす]い**といっても**、[学生/がくせい]にとってはかなりの[負担/ふたん]だろう。' trans='虽说便宜，但对学生来说也是不小的负担吧。' />
5. <grammer-content sentence='[合格/ごうかく]した**といっても**、まだ1[次試験/じしけん]を[通過/つうか]しただけです。' trans='虽说合格了，但也只是通过了第一轮考试。' />

:::
