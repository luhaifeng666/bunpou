# Vてみせる＜演示／決心＞ <Badge type="danger" text="N2" />

* 意义：①用于向对方演示或示范某一动作；②用于表达说话人一定要做到某事的决心。
* 接续：Vて＋みせる
* 译文：①做给……看；②一定要……给你看

::: tip :bookmark: 説明

「見せる」本身有"给看、让X看"的意思。

表达决心时，多含有"说我不行，偏偏要做给你们看看的"的语气。

:::

::: info :books: 例句

3. <grammer-content sentence='[歌/うた]がお[上手/じょうず]だそうですね。[一度/いちど][歌/うた]って**みせて**ください。' trans='听说您唱歌很好听。请唱给我听听。' />
