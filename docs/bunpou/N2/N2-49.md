# ～のみならず＜範囲＞ <Badge type="danger" text="N2" />

* 意义：表示事物存在的范围不限于此。
* 接续：名词／动词、Ⅰ类形容词的连体形／Ⅱ类形容词词干＋のみならず
* 译文：不仅……

::: info :books: 例句

1. <grammer-content sentence='この[本/ほん]は[子供/こども]**のみならず**、[大人/おとな]にも[人気/にんき]がある。' trans='这本书不仅孩子，大人也喜欢。' />
2. <grammer-content sentence='[彼/かれ]は[英語/えいご]**のみならず**、[日本語/にほんご]も[話/はな]せる。' trans='他不仅英语，日语也能说。' />
3. <grammer-content sentence='[経済的/けいざいてき]**のみならず**、[社会的/しゃかいてき]な[問題/もんだい]でもある。' trans='不仅是经济问题，也是社会问题。' />
4. <grammer-content sentence='[面白/おもしろ]い**のみならず**、[勉強/べんきょう]にもなる。' trans='不仅有趣，还能学到东西。' />

:::
