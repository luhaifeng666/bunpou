# Nに応じて＜根据＞ <Badge type="danger" text="N2" />

* 意义：表示根据该事物的性质、内容的不同而采取相应的措施。
* 接续：名词＋に応じて
* 译文：随着……；根据……

::: tip :bookmark: 説明

后面一般接自然而然产生的结果。

:::

::: info :books: 例句

1. <grammer-content sentence='[状況/じょうきょう]**に応じて**、[対応/たいおう]を[変/か]える。' trans='根据情况改变应对方式。' />
2. <grammer-content sentence='[人数/にんずう]**に応じて**、[部屋/へや]を[用意/ようい]する。' trans='根据人数准备房间。' />
3. <grammer-content sentence='[客/きゃく]の[注文/ちゅうもん]**に応じて**、[料理/りょうり]を[作/つく]る。' trans='根据客人的订单做菜。' />
4. <grammer-content sentence='[需要/じゅよう]**に応じて**、[生産/せいさん][量/りょう]を[調整/ちょうせい]する。' trans='根据需求调整生产量。' />
