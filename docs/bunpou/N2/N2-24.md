# ～際＜時点＞ <Badge type="danger" text="N2" />

* 意义：用于表示动作行为的时间。
* 接续：名词＋の／动词的连体形＋際
* 译文：……时；……的时候

::: tip :bookmark: 説明

用于书面语或正式场合。有时也用「～際は」「～際に（は）」「Nに際して」「Nに際するN」。

:::
