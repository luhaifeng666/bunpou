# Vつつある＜持続性的変化＞ <Badge type="danger" text="N2" />

* 意义：表示状态的持续变化。
* 接续：动词的第一连用形＋つつある
* 译文：正在……

::: info :books: 例句

1. <grammer-content sentence='[状況/じょうきょう]は[改善/かいぜん]しつつある。' trans='状况正在改善。' />
2. <grammer-content sentence='[日本/にほん]の[人口/じんこう]は[減少/げんしょう]しつつある。' trans='日本的人口正在减少。' />
3. <grammer-content sentence='[技術/ぎじゅつ]は[進歩/しんぽ]しつつある。' trans='技术正在进步。' />
4. <grammer-content sentence='[問題/もんだい]は[解決/かいけつ]に[向/む]かいつつある。' trans='问题正在向解决方向发展。' />

:::
