# Vる一方だ＜不断増強の勢い＞ <Badge type="danger" text="N2" />

* 意义：表示变化的势头在不断增强，一时难以止住。多用于叙述消极的事情。
* 接续：动作性名词＋の＋一方だ；动词的辞典形＋一方だ
* 译文：不断……

::: info :books: 例句

1. <grammer-content sentence='[外来語/がいらいご]が[増/ふ]える**一方である**[以上/いじょう]、[学習者/がくしゅうしゃ]は[積極的/せっきょくてき]に[向/む]き[合/あ]わなければならない。' trans='既然外来语不断增加，学习者就必须积极面对。' />
2. <grammer-content sentence='[兄/あに]は[結婚/けっこん]してから、[太/ふと]る**一方だ**。' trans='哥哥结婚后不断发胖。' />
3. <grammer-content sentence='このごろは、[食料/しょくりょう]の[値段/ねだん]が[上/あ]がる**一方で**、[困/こま]っている。' trans='最近食品价格不断上涨，真让人头疼。' />
4. <grammer-content sentence='AIの[進歩/しんぽ]で、この[業界/ぎょうかい]は[今後/こんご]は[衰退/すいたい]の**一方だろう**。' trans='随着AI的进步，这个行业今后恐怕会不断衰退。' />

:::
