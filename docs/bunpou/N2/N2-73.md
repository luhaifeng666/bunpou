# ～Vる/Vた一方だ<不断增强的势头> <Badge type="danger" text="N2" />

* 意义：表示变化的势头在不断增强，一时难以止住。多用于叙述消极的事情。
* 接续：动作性名词+の+一方だ／动词的辞典形+一方だ
* 译文：不断……

::: info :books: 例句

1. <grammer-content sentence='**[結婚/けっこん]し[nari/なく]、[子/こども]も[産/う]まない Such[nari/若/わか]者が**[増/ふ]える[nari/一方だ]**。' trans='不结婚也不生孩子，这样的年轻人不断增加。' />
2. <grammer-content sentence='**[外来語/がいらいご]が**[増/ふ]える[nari/一方だ]**[以上/いじょう]、[学習者/がくしゅうしゃ]は[積極的/せっきょくてき]に[向/む]き[nari/合わ]なければならない。' trans='既然外来语不断增加，学习者就必须积极面对。' />
3. <grammer-content sentence='[兄/あに]は**[結婚/けっこん]してから**[太/ふと]る[nari/一方だ]**。' trans='哥哥结婚后不断发胖。' />
