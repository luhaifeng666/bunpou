# Nに限らず＜非限定＞ <Badge type="danger" text="N2" />

* 意义：表达后面所述的内容不限于名词所指称的事物。一般用于书面语。
* 接续：名词＋に限らず
* 译文：不仅……；不限于……

::: info :books: 例句

1. <grammer-content sentence='[経験者/けいけんしゃ]**に限らず**、[初心者/しょしんしゃ]も[大歓迎/だいかんげい]！！' trans='不仅欢迎有经验的人，初学者也大受欢迎！！' />
2. <grammer-content sentence='[勉強/べんきょう]**に限らず**、どんなことでも[努力/どりょく]が[大切/たいせつ]だ。' trans='不仅学习，任何事情努力都很重要。' />
3. <grammer-content sentence='[高校生/こうこうせい]**に限らず**、[誰/だれ]でも[参加/さんか]できます。' trans='不仅高中生，谁都可以参加。' />
4. <grammer-content sentence='[少子化/しょうしか]は[高校野球/こうこうやきゅう]**に限らず**、スポーツ[界/かい]にとって[深刻/しんこく]な[問題/もんだい]だ。' trans='少子化不仅限于高中棒球，对整个体育界都是严重的问题。' />

:::
