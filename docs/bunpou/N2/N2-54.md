# ～といえば＜让步＞ <Badge type="danger" text="N2" />

* 意义：前后连接同一个词语，表示对结论的消极认可，意为虽不否认该事实，但是存在与预想相反的情况。
* 接续：同一个词语＋といえば＋同一个词语
* 译文：要说……确实是……，（但是）……

::: info :books: 例句

1. <grammer-content sentence='[そこ/そこ]に[身/み]を[置/お]く[わけ/わけ][当/あ]たり[前/まえ]**といえば[当/あ]たり[前/まえ]**のことかもしれない。' trans='因为置身于那种环境当中，要说理所当然大概也的确是理所当然的吧。' />
2. <grammer-content sentence='[田舎/いなか]**といえば[田舎/いなか]**だけど、とても[暮/く]らしやすいよ。' trans='要说乡下确实是乡下，但是非常宜居。' />
3. <grammer-content sentence='[簡単/かんたん]**といえば[簡単/かんたん]**だが、[慣/な]れないと[時間/じかん]がかかる。' trans='要说简单确实简单，但不习惯的话很花时间。' />
4. <grammer-content sentence='もう11[月/がつ]に[入/はい]っているので、[寒/さむ]いのは[当/あ]たり[前/まえ]**といえば[当/あ]たり[前/まえ]**だね。' trans='已经11月了，要说冷是理所当然的。' />
5. <grammer-content sentence='[意味/いみ]が[分/わ]かる**といえば[分/わ]かる**のだが、もう[少/すこ]し[表現/ひょうげん]に[工夫/くふう]をして[欲/ほ]しい。' trans='要说意思能懂确实能懂，但希望表达上再下点功夫。' />
:::
