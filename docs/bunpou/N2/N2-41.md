# A／Vてならない＜看法＞ <Badge type="danger" text="N2" />

* 意义：表示说话人难以控制的、强烈的心理状态。
* 接续：AⅠくて／AⅡで／Vて＋ならない
* 译文：非常……；~得不得了

::: tip :bookmark: 説明

前接表达感情或思考义的动词或形容词。

:::

::: info :books: 例句

1. <grammer-content sentence='[保守的/ほしゅてき]な[私/わたし]にしてみれば、[京都/きょうと]の[景観/けいかん]の[変化/へんか]は**残念でならない**。' trans='在保守的我看来，京都景观的变化令人遗憾不已。' />
2. <grammer-content sentence='いちばん[仲/なか]がよかった[友達/ともだち]が[留学/りゅうがく]してしまい、**寂しくてならない**。' trans='关系最好的朋友留学了，寂寞得不得了。' />
3. <grammer-content sentence='[今日/きょう]の[出来事/できごと]を[誰/だれ]かに**話したくてならない**。' trans='今天发生的事想跟谁说想得不得了。' />
4. <grammer-content sentence='この[歌/うた]を[聞/き]くと、[大学時代/だいがくじだい]のことが**懐かしく思い出されてならない**。' trans='听到这首歌，大学时代的事令人怀念不已。' />

:::
