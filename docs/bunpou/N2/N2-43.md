# ～といってよい＜評価＞ <Badge type="danger" text="N2" />

* 意义：表示说话人对所述事物或人进行说明、判断或评论。
* 接续：简体句子＋といってよい
* 译文：可以说……

::: tip :bookmark: 説明

多用于书面语。同义的句式还有「～といってもいい」「～ということができる」。

:::

::: info :books: 例句

1. <grammer-content sentence='この[研究/けんきゅう]は[成功/せいこう]したと**いってよい**。' trans='这个研究可以说成功了。' />
2. <grammer-content sentence='[彼/かれ]はこの[分野/ぶんや]の[専門家/せんもんか]と**いってよい**。' trans='他可以说是这个领域的专家。' />
3. <grammer-content sentence='[日本/にほん]は[経済/けいざい][大国/たいこく]と**いってもいい**。' trans='日本可以说是经济大国。' />
4. <grammer-content sentence='この[本/ほん]は[古典/こてん]の[名作/めいさく]と**いうことができる**。' trans='这本书可以说是古典名作。' />

:::
