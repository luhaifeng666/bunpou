# Vるまでもない＜没有必要＞ <Badge type="danger" text="N2" />

* 意义：表示没有必要做某事。
* 接续：动词词典形＋までもない
* 译文：不必……

::: tip :bookmark: 説明

「言うまでもない」是惯用词，表示"当然，自不必说"。

:::

::: info :books: 例句

1. <grammer-content sentence='**言うまでもなく**、[平安京/へいあんきょう]は[大唐時代/だいとうじだい]の[長安/ちょうあん]の[都/みやこ]にならってつくられた[町/まち]である。' trans='自不必说，平安京是模仿大唐时代的长安都建立起来的城市。' />
2. <grammer-content sentence='メールで[連絡/れんらく]すればいい。わざわざあって[話/はな]す**までもない**。' trans='用邮件联系就好。不必特意见面说。' />
3. <grammer-content sentence='ネットで[調/しら]べればすぐわかるのだから、[先生/せんせい]に[聞/き]く**までもない**。' trans='网上查一下马上就明白，不必问老师。' />
4. <grammer-content sentence='[説明/せつめい]する**までもない**だろうが、[現在/げんざい]、[地球規模/ちきゅうきぼ]の[問題/もんだい]となっているのが[地球温暖化/ちきゅうおんだんか]だ。' trans='不必说明吧，现在成为全球规模问题的是地球变暖。' />

:::
