# すら＜凸顕極端事例＞ <Badge type="danger" text="N2" />

* 意义：举出极端的事例，表达其他更不用说的含义。
* 接续：名词＋すら
* 译文：连……都……；甚至……

::: tip :bookmark: 説明

比较负面，更多带上说话人惊讶或者批评的语气。

:::

::: info :books: 例句

1. <grammer-content sentence='[彼/かれ]は[自分/じぶん]の[名前/なまえ]**すら**[書/か]けない。' trans='他连自己的名字都不会写。' />
2. <grammer-content sentence='[小学生/しょうがくせい]**すら**[知/し]っているような[簡単/かんたん]なことだ。' trans='这是连小学生都知道的简单事。' />
3. <grammer-content sentence='[親/おや]**すら**[知/し]らなかった。' trans='连父母都不知道。' />
4. <grammer-content sentence='[夢/ゆめ]**すら**[見/み]なかった。' trans='连做梦都没想过。' />

:::
