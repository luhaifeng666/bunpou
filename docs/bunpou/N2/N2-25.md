# ～ほどだ／～ほど～＜程度＞ <Badge type="danger" text="N2" />

* 意义：用于举出具体事例说明非同一般的程度。
* 接续：动词、形容词的连体形＋ほど（だ）
* 译文：甚至……；甚至于……

::: info :books: 例句

1. <grammer-content sentence='その[巨体/きょたい]と[愛嬌/あいきょう]のあるしぐさを[含/ふく]めて[人気力士/にんきりきし]となり、40[歳/さい]で[引退/いんたい]したあともテレビ・コマーシャルの[人気者/にんきもの]になった**ほどである**。' trans='包括那庞大的身躯和可爱的举止在内，他成为了人气力士，40岁引退后甚至成为了电视广告的宠儿。' />
2. <grammer-content sentence='[青海湖/せいかいこ]の[美/うつく]しさは[言葉/ことば]では[表現/ひょうげん]できない**ほどだ**。' trans='青海湖的美甚至无法用语言表达。' />
3. <grammer-content sentence='[咳/せき]がひどくて[夜/よる]も[眠/ねむ]れない**ほどだった**ので、[病院/びょういん]に[行/い]った。' trans='咳嗽严重到甚至晚上睡不着，所以去了医院。' />
4. <grammer-content sentence='[死/し]ぬ**ほど**[頑張/がんば]ったのに、[試験/しけん]に[落/お]ちてしまった。' trans='甚至拼命努力了，却还是考试落榜了。' />
5. <grammer-content sentence='[映画/えいが]の[最後/さいご]のシーンは[素晴/すば]らしくて、[涙/なみだ]が[出/で]る**ほど**[感動/かんどう]した。' trans='电影最后的场景太精彩了，感动得甚至流下了眼泪。' />

:::
