# ～まる／める＜程度の発展＞ <Badge type="danger" text="N2" />

* 意义：表示程度的不断发展。
* 接续：Ⅰ类形容词词干＋まる（自动词）／める（他动词）
* 译文：（使）增强、提高、加深、减弱

::: tip :bookmark: 説明

这类常用的成对自他动词有：
- 強まる／強める
- 高まる／高める
- 深まる／深める
- 弱まる／弱める
- 薄まる／薄める
- 固まる／固める
- 早まる／早める
- 広まる／広める

`～まる／～める`一般表抽象事物的程度变化，注意其与`AⅠくなる／AⅠくする`的不同。

:::

::: info :books: 例句

1. <grammer-content sentence='[年々/ねんねん][企業/きぎょう]の[面接重視/めんせつじゅうし]の[傾向/けいこう]が**強まっている**。' trans='企业重视面试的倾向逐年增强。' />
2. <grammer-content sentence='[最近/さいきん]、あの[動画/どうが]はネット[上/じょう]で[人気/にんき]が**高まっている**。' trans='最近，那个视频在网上人气越来越高。' />
3. <grammer-content sentence='[夏/なつ]になれば、ウイルス[影響/えいきょう]が**弱まる**だろう。' trans='到了夏天，病毒的影响应该会减弱吧。' />
4. <grammer-content sentence='[交流/こうりゅう][活動/かつどう]を[通/とお]して、[両国/りょうこく]の[学生/がくせい]はお[互/たが]いの[文化/ぶんか]への[理解/りかい]を**深めた**。' trans='通过交流活动，两国学生加深了对彼此文化的理解。' />
5. <grammer-content sentence='[台風/たいふう]の[影響/えいきょう]で、[野菜/やさい]の[値段/ねだん]が（×**高まっている**／○[高/たか]くなっている）。' trans='受台风影响，蔬菜价格上涨了。（具体价格变化用`高くなる`，抽象程度变化用`高まる`）' />
