# ～Vるところを／Vているところを／Vたところを<逆接> <Badge type="danger" text="N2" />

* 意义：表示正在做某动作或行为的时候，发生了对其不利或意外阻断的情况。
* 接续：Vる/Vている/Vた+ところ（を）
* 译文：正在……的时候，（却）……

::: tip :bookmark: 説明

「ところ」表示场景、时间点，主句描述对其不利或意外的情况。另有「Nのところ（を）」表示对方所处的状态，谓语为感谢、道歉或请求。

:::
