# ～ように＜铺垫＞ <Badge type="danger" text="N2" />

* 意义：用于对后面的说明进行铺垫，表示前句所陈述的事实与后句所要陈述的内容是一致的。
* 接续：接在动词的连体形或"动作性名词＋の"
* 译文：正如……一样；如同……一样

::: info :books: 例句

1. <grammer-content sentence='[京都/きょうと]の[町/まち]は、[一木一草/いちぼくいっそう]にも[歴史/れきし]があると[言/い]われる**ように**、いつ、どこでも[新/あたら]しい[発見/はっけん]がある。' trans='京都的街道，正如人们说的一草一木都有历史一样，无论何时何地都有新发现。' />
2. <grammer-content sentence='ご[存知/ぞんじ]の**ように**、[中国/ちゅうごく]は[国土/こくど]が[広/ひろ]く、[地方/ちほう]によって[言葉/ことば]や[習慣/しゅうかん]が[大/おお]きく[異/こと]なる。' trans='正如您所知，中国国土辽阔，地区不同语言和习惯差异很大。' />
3. <grammer-content sentence='[先/さき]ほど[先生/せんせい]がおっしゃっていた**ように**、バランスのいい[食事/しょくじ]はとても[大事/だいじ]です。' trans='正如刚才老师所说的，均衡的饮食很重要。' />
4. <grammer-content sentence='やはり[新聞/しんぶん]に[書/か]いてある**ように**、この二つの[事件/じけん]には[関連/かんれん]があるのだろうか。' trans='果然正如报纸上写的，这两起事件有关联吗？' />

:::
