# Vることはない＜無必要＞ <Badge type="danger" text="N2" />

* 意义：表示说话人认为没有必要进行该动作。
* 接续：动词的词典形＋ことはない
* 译文：不必……

::: info :books: 例句

1. <grammer-content sentence='[確/たし]かに[風景/ふうけい]に[対/たい]してなら[恥/は]じらいを[感/かん]じる**ことはない**わけだ。' trans='确实对风景的话不必感到羞涩。' />
2. <grammer-content sentence='[電話/でんわ]で[申/もう]し[込/こ]んでもいいのだから、わざわざ[行/い]く**ことはない**。' trans='电话申请也可以，不必特意去。' />
3. <grammer-content sentence='お[金/かね]のことなら[心配/しんぱい]する**ことはない**よ。' trans='钱的事不必担心。' />
4. <grammer-content sentence='[困/こま]ったことがあったらいつでも[相談/そうだん]に[乗/の]るから、[一人/ひとり]で[悩/なや]む**ことはない**。' trans='如果有困难随时可以找我商量，不必一个人烦恼。' />

:::
