# ～わりに（は）＜比例不均衡＞ <Badge type="danger" text="N2" />

* 意义：表示转折，意为以从句的状态或程度为标准进行衡量的话，主句所示的实际情况与预测的情况不符。
* 接续：名词＋の／动词、形容词的连体形＋わりに（は）
* 译文：虽然……却……；尽管……但……

::: tip :bookmark: 説明

一般表示说话人的主观评价。多用于口语。

由「～わりに（は）」连接的主句和从句，表达的是在数量、程度等方面存在落差、不成比例的含义，不构成比例关系时不能使用「わりに（は）」。此外，主句不表示一次性事件。

:::

::: info :books: 例句

2. <grammer-content sentence='このマンションは[駅/えき]に[近/ちか]い**わりには**[安/やす]い。' trans='这个公寓离车站很近，却很便宜。' />
3. <grammer-content sentence='[一生懸命/いっしょうけんめい][勉強/べんきょう]した**わりには**、[成績/せいせき]があがらなかった。' trans='虽然拼命学习了，但成绩却没有提高。' />
4. <grammer-content sentence='[新幹線/しんかんせん]でいけるので、[距離/きょり]の**わりには**、[往復/おうふく]が[楽/らく]である。' trans='因为可以坐新干线去，距离虽远，往返却很轻松。' />
5. <grammer-content sentence='あの[島/しま]は[景色/けしき]がきれいな**わりには**、[観光客/かんこうきゃく]があまりいない。' trans='那个岛虽然景色很美，但游客不多。' />
