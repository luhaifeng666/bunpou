# Nまでして／Vてまで＜極端の程度＞ <Badge type="danger" text="N2" />

* 意义：表示极端的行为或状况，意思是为了达到某一目的而选择非同一般的手段，或付出了非同寻常的努力。
* 接续：动作性名词＋までして；Vて＋まで
* 译文：甚至……；以至于……；不惜……

::: info :books: 例句

1. <grammer-content sentence='[鑑真/がんじん]って、[何度/なんど]も[遭難/そうなん]しそうになって、[失明/しつめい]**までして**[日本/にほん]にたどり[着/つ]かれたのよね。' trans='鉴真多次遇难，甚至失明才到达了日本。' />
2. <grammer-content sentence='そんな[厳/きび]しい[練習/れんしゅう]**までして**[賞/しょう]を[取/と]ろうとは[思/おも]いません。' trans='我不打算通过那么严格的练习来获奖。' />
3. <grammer-content sentence='[授業/じゅぎょう]をサボって**まで**[行/い]く[必要/ひつよう]はないじゃないか。' trans='没必要甚至逃课去。' />
4. <grammer-content sentence='[試験/しけん]に[合格/ごうかく]するため、[睡眠時間/すいみんじかん]を[減/へ]らして**まで**[勉強/べんきょう]している。' trans='为了考试合格，甚至减少睡眠时间来学习。' />

:::
