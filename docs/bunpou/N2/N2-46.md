# ～とともに＜変化＞ <Badge type="danger" text="N2" />

* 意义：表示一方发生变化，另一方也随之发生变化。
* 接续：动词词典形／名词＋とともに
* 译文：随着……

::: tip :bookmark: 説明

书面语。

:::

::: info :books: 例句

1. <grammer-content sentence='[年齢/ねんれい]を[重/かさ]ねる**とともに**、[考/かんが]え[方/かた]も[変/か]わった。' trans='随着年龄增长，想法也改变了。' />
2. <grammer-content sentence='[技術/ぎじゅつ]の[進歩/しんぽ]**とともに**、[生活/せいかつ]が[便利/べんり]になった。' trans='随着技术进步，生活变得方便了。' />
3. <grammer-content sentence='[経済/けいざい]の[発展/はってん]**とともに**、[環境問題/かんきょうもんだい]も[深刻/しんこく]になった。' trans='随着经济发展，环境问题也变得严重了。' />
4. <grammer-content sentence='[季節/きせつ]が[変/か]わる**とともに**、[服装/ふくそう]も[変/か]わる。' trans='随着季节变化，服装也改变。' />

:::
