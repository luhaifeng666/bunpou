# Ｎ１とＮ２とどちらが～＜选择＞

* 意义：表示选择性疑问，询问N1、N2二者中哪一个更具有谓语所示的特征；
* 接续：名词 + と + 名词 + と（では）＋どちら＋（のほう）が；
* 译文：N1和N2，哪个更...

::: tip :bookmark: 补充

「では」「のほう」可省。回答一般用`「N1/N2のほうが～」(N1/N2更...)`或者`「（N1もN2も）どちらも（同じくらい～）(N1和N2都...)」`

:::
