# Nのようだ/です

Nのようだ 主要有以下几种用法。

## 1. Nのようだ/です <比喻>
  
- 意义：表示明喻,将某事物比作「ようだ」前面的名词，常与副词 `「まるで」(完全，全然，简直)` 搭配使用；
- 接续：
  - 名词 + のようだ
  - 名词 + のような + 名词
  - 名词 + のように + 动词/ 形容词
- 译文：像 …… ; 就像……一样 ;宛如 ……
  
::: info :books: 例句
  
1. <grammer-content sentence='[香山公園/こうざんこうえん]の[頂上/ちょうじょう]から[見/み]た景色けしきは、まるで[絵/え]の**ようでした**。' trans='从香山公园山顶看到的景色，简直像画一样美。' />
2. <grammer-content sentence='[妹/いもうと]は[髪/かみ]が[短/みじか]くて、**まるで[男の子/おとこのこ]のようです**。' trans='妹妹头发很短，简直像个男孩子。' />
3. <grammer-content sentence='[豆腐料理/とうふりょうり]ですが、[肉/にく]の**ような[味/あじ]**がします。' trans='虽然是豆腐做的菜，却有肉的味道。' />
  
:::

## 2. Nのようだ/です <例举>

- 意义：将 `「ような/ように」前面的名词` 作为例子 , 表示某事物具有与其相同的特征；
- 接续：
  - 名词 + のような + 名词
  - 名词 + のように + 动词/ 形容词
- 译文：例如……; 同……一样 ;像 ……
  
::: info :books: 例句
  
1. <grammer-content sentence='ご[存/じぞんじ]**のように**、[中日/ちゅうにち]は[一衣帯水/いちいたいすい]の[隣国/りんごく]である。' trans='正如您所知，中国和日本是一衣带水的邻邦。' />
2. <grammer-content sentence='[孫/そん]さん**のように**[上手/じょうず]に[日本語/にほんご]が[話せる/はなせる][人/ひと]はなかなかいません。' trans='很少有人能像孙先生那样流利地说日语。' />
3. <grammer-content sentence='[私/わたし]は[東京/とうきょう]**のような**[大都会/だいとかい]に[住み/すみ]たいです。' trans='我想住在东京这样的大都市。' />
  
:::
