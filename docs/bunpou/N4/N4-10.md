# に＜主体＞ <Badge type="warning" text="N4" />

* 意义：表示能力的主体。
* 接续：指人名词＋に＋表能力的动词
* 译文：（某人）能……

::: tip :bookmark: 説明

后接动词的否定形式、需要提示主题时，常用`には`，含有类同的意义时，常用`にも`。

:::

::: info :books: 例句

1. <grammer-content sentence='[私/わたし]にできるかな……。' trans='我能行吗……' />
2. <grammer-content sentence='みんなに[分/わ]かるようにゆっくり、はっきり[話/はな]してください。' trans='为了让大家都能听懂，请慢慢、清楚地说。' />
3. <grammer-content sentence='そのバッグがほしいけれど、[高/たか]くて[学生/がくせい]のわたしに[買/か]えるような[値段/ねだん]ではない。' trans='那个包虽然很想要，但是太贵了，不是学生身份的我买得起的价格。' />
4. <grammer-content sentence='こんな[難/むずか]しい[文章/ぶんしょう]は、二年生**には**[翻訳/ほんやく]できない。' trans='这么难的文章，二年级学生翻译不了。' />
5. <grammer-content sentence='この[問題/もんだい]、[私/わたし]**には**[答/こた]えられそうにありません。' trans='这个问题，我好像答不上来。' />
6. <grammer-content sentence='[操作/そうさ]が[簡単/かんたん]だから、お[年寄/としより]**にも**[使/つか]えるでしょう。' trans='因为操作简单，老年人也能使用吧。' />
7. <grammer-content sentence='この[漢字/かんじ]は[専門家/せんもんか]**にも**[読/よ]めないらしい。' trans='这个汉字好像连专家也读不出来。' />

:::
