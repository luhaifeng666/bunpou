# ～んじゃない＜禁止＞ <Badge type="warning" text="N4" />

* 意义：表示禁止、不许对方做某事，常在父母对孩子说话等场景中出现。
* 接续：Vる＋んじゃない
* 译文：不要……；不许……

::: tip :bookmark: 説明

读降调，表示禁止、不许对方做某事。

:::

::: info :books: 例句

1. <grammer-content sentence='[父/ちち]「[走/はし]り[回/まわ]る**んじゃない**。[本/ほん]でも[読/よ]んで、[少/すこ]し[静/しず]かにしていなさい。」' trans='爸爸："不要到处跑。看看书，安静一会儿。"' />
2. <grammer-content sentence='[兄/あに]「[電車/でんしゃ]の[中/なか]で[大声/おおごえ]で[話/はな]す**んじゃない**。」' trans='哥哥："不要在电车里大声说话。"' />
3. <grammer-content sentence='[食/た]べ[物/もの]の[好/す]き[嫌/きら]いを[言/い]う**んじゃありません**よ。' trans='不要挑食。' />

:::
