# AⅠくする/AⅡにする＜状態・態度＞ <Badge type="warning" text="N4" />

* 意义：`AⅠくする/AⅡにする`除了可以表示使某物发生状态变化的含义外，还可以表示动作主体保持着某种状态或表现出某种态度的含义。
* 接续：形容词的第一连用形＋する
* 译文：保持……状态；表现得……

::: info :books: 例句

1. <grammer-content sentence='[元気/げんき]にしてた？' trans='你好吗？' />
2. <grammer-content sentence='お[母さん/かあさん]はいつもきれいにしている。' trans='妈妈总是打扮得很漂亮。' />
3. <grammer-content sentence='[仕事/しごと]や[家事/かじ]などで[毎日/まいにち][忙しく/いそがしく]していると、[勉強/べんきょう]の[時間/じかん]がなかなか[取/と]れない。' trans='如果每天因为工作和家务而忙碌的话，很难抽出学习的时间。' />
4. <grammer-content sentence='みんなの[未来/みらい]のために、[地球/ちきゅう]にやさしくしましょう。' trans='为了大家的未来，让我们善待地球吧。' />
5. <grammer-content sentence='[周/まわ]りのみなさんがいつも[親切/しんせつ]にしてくださることに[感謝/かんしゃ]しています。' trans='感谢周围的人总是对我很亲切。' />

:::
