# Vるといい＜提案＞ <Badge type="warning" text="N4" />

* 意义：用于给对方提建议。
* 接续：动词的词典形＋といい
* 译文：……比较好；最好是……

::: warning :warning: 注意

对上级或年长者一般不用该句式。

:::

::: info :books: 例句

1. <grammer-content sentence='A.N.さんももっと[学生生活/がくせいせいかつ]を[楽/たの]しむことを[考/かんが]えてみる**といい**んじゃないでしょうか。' trans='A.N.你也应该想一想如何更快乐地度过大学生活吧。' />
2. <grammer-content sentence='[日本語/にほんご]の[辞書/じしょ]はこれを[使/つか]う**といい**です。' trans='日语词典用这本比较好。' />
3. <grammer-content sentence='[風邪/かぜ]ならこの[薬/くすり]を[飲/の]む**といい**ですよ。' trans='如果是感冒的话，喝这个药比较好。' />
4. <grammer-content sentence='[疲/つか]れたでしょう。[少/すこ]し[休/やす]む**といい**ですよ。' trans='累了吧。休息一下比较好。' />

:::
