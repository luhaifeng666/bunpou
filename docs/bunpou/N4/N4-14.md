# Nのところ＜処所化＞ <Badge type="warning" text="N4" />

* 意义：表示存在和活动的处所。
* 接续：表示人、物的名词＋のところ
* 译文：……这里；……的地方

::: info :books: 例句

1. <grammer-content sentence='じゃあ、[書/か]き[終/お]わったら、[私/わたし]の**ところ**へ[持/も]ってきてください。' trans='那写完之后，请拿到我这里来。' />
2. <grammer-content sentence='[週末/しゅうまつ]になると、[両親/りょうしん]の**ところ**に[帰/かえ]る。' trans='一到周末，就回父母那里。' />
3. <grammer-content sentence='[前/まえ]にお[借/か]りした[本/ほん]を[先生/せんせい]の**ところ**に[返/かえ]しに[行/い]きました。' trans='之前借的书，我去老师那里还了。' />
4. <grammer-content sentence='[李/り]さんはドアの**ところ**に[立/た]っています。' trans='小李站在门那里。' />
5. <grammer-content sentence='[図書館/としょかん]に[入/はい]ったら、[高橋/たかはし]さんはまずパソコンの**ところ**に[行/い]って[検索/けんさく]してみた。' trans='进了图书馆之后，高桥首先走到电脑那里搜索了一下。' />

:::
