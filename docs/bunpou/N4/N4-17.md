# ～だけ＜限定／程度＞ <Badge type="warning" text="N4" />

* 意义：表示限定所述的事物。
* 接续：名词／动词、形容词的连体形＋だけ
* 译文：只……；仅仅……

::: tip :bookmark: 説明

惯用搭配：`できるだけ`（极限程度）、`Vたいだけ`、`ほしいだけ`、`好きなだけ`等。

「～だけで」放在句子中间表示中顿，「～だけだ／だけです」放在句子末尾结句。

:::

::: info :books: 例句

1. <grammer-content sentence='ちょっとのぞく**だけ**……。' trans='只是稍微看看……' />
2. <grammer-content sentence='[彼/かれ]は[文句/もんく]を[言/い]う**だけで**、[何/なに]も[手伝/てつだ]ってくれない。' trans='他只是发牢骚，什么都不帮忙。' />
3. <grammer-content sentence='あのみせの[料理/りょうり]は[高/たか]い**だけで**[全然/ぜんぜん][美味/おい]しくない。' trans='那家店的菜只是贵，一点都不好吃。' />
4. <grammer-content sentence='[彼/かれ]はコミュニケーションが[嫌/きら]いなんじゃなく、[苦手/にがて]な**だけ**だと[思/おも]う。' trans='我觉得他不是讨厌交流，只是不擅长而已。' />
5. <grammer-content sentence='[明日/あした]は**できるだけ**[早/はや]く[来/き]てください。' trans='明天请尽可能早点来。' />
6. <grammer-content sentence='[食/た]べられる**だけ**[食/た]べた。' trans='能吃多少就吃多少。' />
7. <grammer-content sentence='[日曜日/にちようび]は[寝/ね]たい**だけ**[寝/ね]ることができる。' trans='周日想睡多久就能睡多久。' />
8. <grammer-content sentence='[好/す]きなものを[好/す]きな**だけ**[食/た]べなさい。' trans='随你喜欢去吃吧。' />

:::
