# Vてくる＜物品・情報の移動＞ <Badge type="warning" text="N4" />

* 意义：表示向着说话人（一方）进行物品或信息的移动。
* 接续：前接表示物品移动或信息传递的「送る」「届ける」「郵送する」「伝える」等他动词和「届く」「（電話が）かかる」等自动词，以及指向特定对象的、表示语言行为的「言う」「相談する」「頼む」「依頼する」等动词。当接受者是第一人称时，一般省略。
* 译文：……来；……来了

1. <grammer-content sentence='しばらく[電話/でんわ]してこなかったから、[心配/しんぱい]してたのよ。' trans='你好长时间没有来电话，我有点担心。' />
2. <grammer-content sentence='[小学校/しょうがっこう]の[先生/せんせい]から[母/はは]に[電話/でんわ]がかかってきた。' trans='小学老师给妈妈打来了电话。' />
3. <grammer-content sentence='[注文/ちゅうもん]した[本/ほん]はもう[送/おく]ってきました。' trans='订购的书已经寄来了。' />
