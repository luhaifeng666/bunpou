# 動詞の禁止形 <Badge type="warning" text="N4" />

* 意义：用于要求、命令对方不要做某一行为、动作。
* 接续：动词词典形＋な
* 译文：不要……；别……

::: tip :bookmark: 説明

动词的禁止形也可以说就是动词简体命令形的否定形式，`Vるな`与动词的简体命令形一样多为男性使用。

:::

::: info :books: 例句

1. <grammer-content sentence='お[父さん/とうさん]も「[頑張/がんば]れ。あきらめる**な**。」って[言/い]ってるわよ。' trans='爸爸也在说"加油，别放弃"。' />
2. <grammer-content sentence='[先生/せんせい]に[遅刻/ちこく]する**な**と[怒/おこ]られた。' trans='被老师训斥说不要迟到。' />
3. <grammer-content sentence='そんなつまらないことを[人/ひと]に[頼/たの]む**な**。' trans='不要把那种无聊的事情拜托给别人。' />
4. <grammer-content sentence='[大丈夫/だいじょうぶ]だから、もう[泣/な]く**な**よ。' trans='没事的，别再哭了。' />

:::
