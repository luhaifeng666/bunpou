# ばかり＜限定＞ <Badge type="warning" text="N4" />

* 意义：表示事物的数量多或动作、变化的次数多。
* 接续：名词＋（に／で／と）＋ばかり
* 译文：光……；净……；总是……

::: tip :bookmark: 説明

经常带有不满、抱怨的语气。口语中表强调时用`ばっかり`。

`だけ`和`ばかり`都表达限定，`Aだけ`意为只有A，不存在其他事物；`Aばかり`意为A的数量或与A相关的次数多。

:::

::: info :books: 例句

1. <grammer-content sentence='[日本/にほん]に[来/き]てもう4[年/ねん]になるわりには、[失敗/しっぱい]**ばかり**しています。' trans='来日本已经4年了，却总是失败。' />
2. <grammer-content sentence='ゲーム**ばかり**していないで、ちゃんと[勉強/べんきょう]しなさい。' trans='不要光玩游戏，好好学习。' />
3. <grammer-content sentence='[交流会/こうりゅうかい]で[隣/となり]の[人/ひと]と**ばかり**[話/はな]していた。' trans='在交流会上光跟旁边的人说话。' />
4. <grammer-content sentence='[今日/きょう]の[食堂/しょくどう]のメニューは[甘/あま]い[物/もの]**ばかりで**、カロリーが[気/き]になる。' trans='今天食堂的菜单净是甜食，让人担心卡路里。' />
5. <grammer-content sentence='お[酒/さけ]{○だけ　×ばかり}[買/か]った。' trans='只买了酒（即没有买其他东西）。' />
6. <grammer-content sentence='いつもお[酒/さけ]{？？だけ　○ばかり}[飲/の]んでいる。' trans='总是喝酒（即喝酒的次数多）。' />
