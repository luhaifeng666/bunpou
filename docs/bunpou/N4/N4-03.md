# 動詞の命令形と命令文 <Badge type="warning" text="N4" />

* 意义：用于发号施令，表示命令。
* 译文：（做）……！

::: tip :bookmark: 動詞の命令形の活用

**Ⅰ類動詞**：詞尾ウ段 → エ段
- 買う → 買え
- 送る → 送れ

**Ⅱ類動詞**：去掉词尾「る」，加上「ろ」（口语）或「よ」（书面语）
- 出る → 出ろ／出よ
- 信じる → 信じろ／信ぜよ

**Ⅲ類動詞**：
- 来る（くる） → 来い（こい）
- する → しろ（口语）或せよ（书面语）
- 勉強する → 勉強しろ／勉強せよ

:::

::: warning :warning: 使用说明

简体命令形使用范围比较窄，常用于发号施令（如军官对士兵、警察对犯人）、交通标志等，多为男性使用。一般性的命令则多使用`Vなさい`。

`せよ`多用于试题的导语。

除了动词的命令形之外，动词的"て形"也可以用于上对下或关系亲密的人之间，表示命令的语气。

:::
