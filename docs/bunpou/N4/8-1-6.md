# Nをもとに（して）〈题材、基础〉 <Badge type="tip" text="N4" />

* 意义：表示以某一事物为题材或基础进行言语行为或创作活动。
* 译文： `以……为（素材、基础）`。

::: tip :bookmark: 说明

1. 接续：名词 + もとに（して）
2. 表示以某物为基础进行创作、表达等。

:::

::: info :books: 例句

1. <grammer-content sentence="[最近/さいきん]の[話題/わだい]をもとに[話/はな]すのはどうですか。" trans='主要讲最近的话题怎么样？' />
2. <grammer-content sentence="[小説/しょうせつ]をもとにして[映画/えいが]を[作/つく]った。" trans='以小说为基础拍了电影。' />
3. <grammer-content sentence="[聞/き]いた[話/はなし]をもとにして[小説/しょうせつ]を[書/か]きました。" trans='根据听到的故事写了小说。' />
4. <grammer-content sentence="[調査/ちょうさ]の[結果/けっか]をもとに対策を[考/かんが]えよう。" trans='让我们根据调查结果来考虑对策。' />

:::
