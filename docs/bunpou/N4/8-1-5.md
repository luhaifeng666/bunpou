# Vておく〈提前准备〉口语：とく／どく <Badge type="tip" text="N4" />

* 意义：表示该动作是为后面要做的事情提前做好准备。也可以表示短时间内保持某种状态。
* 译文： `事先（做）······; 预先（做）······`。

::: tip :bookmark: 说明

1. 接续：动词 + ておく
2. 口语中常简化为「とく」（动词原形+とく）
3. 表示为将来可能发生的动作或情况做准备。

:::

::: info :books: 例句

1. <grammer-content sentence="じゃ、いろいろ[体験/たいけん]しておいたほうがいいですね。" trans='那么，体验各种各样的事情比较好哦。' />
2. <grammer-content sentence="[明日/あした]は[忙/いそが]しい[1日/いちにち]になるから、[今晩/こんばん]しっかり[寝/ね]ておこう。" trans='明天会是忙碌的一天，今晚好好睡吧。' />
3. <grammer-content sentence="その[資料/しりょう]はあとで[見/み]ますから、[机/つくえ]の[上/うえ]に[置/お]いておいてください。" trans='那些资料稍后要看，请放在桌子上。' />
4. <grammer-content sentence="[李/り]:[黒板/こくばん]を[消/け]しましょうか。" trans='李：擦一下黑板好吗？' />
5. <grammer-content sentence="先生/せんせい]:そのまま［保持原样］にしておいてください。" trans='老师：请保持原样。' />

:::

::: tip :bookmark: 说明

「ておいた」与「てある」有时表示的是基本相同的事实，但二者叙述的着眼点各异。

:::

2. <grammer-content sentence="[会議/かいぎ]の[資料/しりょう]は[コピー/コピー]してあります。" trans='会议资料已经复印好了（强调复印后的状态）。' />
