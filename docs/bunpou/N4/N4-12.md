# V（ら）れる＜尊他＞ <Badge type="warning" text="N4" />

* 意义：描述动作的尊他语，表示对动作主体的敬意。
* 接续：动词词干＋（ら）れる，与被动态形式相同
* 译文：（您）……

::: tip :bookmark: 活用形

**1类动词**：書く → 書かれる、読む → 読まれる

**2类动词**：借りる → 借りられる、教える → 教えられる

**3类动词**：来る → 来られる、する → される

:::

::: warning :warning: 注意

「V（られる）」可以表示敬语、被动态、自动态、能动态，在句子中使用时究竟是哪一种功能，要依靠句子结构或上下文来判断。

:::

::: info :books: 例句

1. <grammer-content sentence='[昨日/きのう]、[胡/こ][先生/せんせい]が**来られて**（[来/き]て）、[教/おし]えてくださいました。' trans='昨天胡老师来了，告诉了我。' />
2. <grammer-content sentence='あの[方/かた]は、[東西/とうざい][大学/だいがく]で[中国/ちゅうごく][史学/しがく]を**専攻されました**。' trans='那位先生在东西大学专攻中国史学。' />
3. <grammer-content sentence='この[本/ほん]は、[遠藤/えんどう][先生/せんせい]が**書かれた**[本/ほん]です。' trans='这本书是远藤老师写的书。' />

:::
