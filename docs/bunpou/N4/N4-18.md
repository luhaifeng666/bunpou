# Vばいい＜提案＞ <Badge type="warning" text="N4" />

* 意义：表示说话人的建议。
* 接续：动词的条件形＋いい
* 译文：……就行；……就好

::: tip :bookmark: 説明

口语中也用「Vたらいい」。

:::

::: info :books: 例句

1. <grammer-content sentence='[自分/じぶん]の[好/す]きな[力士/りきし]を[応援/おうえん]すれ**ばいい**じゃない。' trans='支持自己喜欢的力士不就行了。' />
2. <grammer-content sentence='お[土産/みやげ]は[空港/くうこう]で[買/か]え**ばいい**よ。' trans='礼物在机场买就行。' />
3. <grammer-content sentence='[泣/な]きたいときはたくさん[泣/な]け**ばいい**。' trans='想哭的时候就尽情哭吧。' />
4. <grammer-content sentence='[日本語/にほんご]はどのように[勉強/べんきょう]すれ**ばいい**か[分/わ]からない。' trans='不知道该怎么学习日语。' />
5. <grammer-content sentence='[明日/あした][早/はや]いから、[今日/きょう]はもう[寝/ね]たら**いい**よ。' trans='明天要早起，今天早点睡吧。' />

:::
