# ～なら＜話題＞ <Badge type="warning" text="N4" />

* 意义：用于凸显主题，主要是就对方提出的话题或问题做出回答、提出请求或者加以说明、建议。
* 接续：动词／Ⅰ类形容词的连体形＋なら　名词／Ⅱ类形容词词干＋なら
* 译文：要说……；要是……；就……来说

::: warning :warning: 注意

与第7课第1单元学过的表示条件的「なら(ば)」的用法不同，表示主题的「なら」之后一般不添加「ば」。

:::
