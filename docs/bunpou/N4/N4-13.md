# Vていただけませんか＜客気に依頼＞ <Badge type="warning" text="N4" />

* 意义：表示客气地请求对方为自己做某事。
* 接续：Vて＋いただけませんか
* 译文：能否……；可不可以……

::: tip :bookmark: 説明

用日语请求时通常使用授受动词来表示说话人的收益，这样的表达方式比较礼貌。

:::

::: info :books: 例句

1. <grammer-content sentence='[私/わたし]は（[相手/あいて]に）（[作文/さくぶん]を）[一度/いちど]**見ていただけませんか**。' trans='能不能请您帮我看一下（作文）呢？' />
2. <grammer-content sentence='[明日/あした]、こちらに**来ていただけませんか**。' trans='明天能不能请您来这里一趟？' />
3. <grammer-content sentence='これを**翻訳していただけませんか**。' trans='能不能请您翻译这个？' />
4. <grammer-content sentence='[私/わたし]は（あなたに）その[本/ほん]を**貸していただけませんか**。' trans='能不能请您把那本书借给我？' />

:::
