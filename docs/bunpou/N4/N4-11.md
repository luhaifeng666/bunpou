# お／ごVです＜尊他＞ <Badge type="warning" text="N4" />

* 意义：描述动作的尊他语，表示对动作主体的敬意。
* 接续：お＋和语动词第一连用形＋です；ご＋汉语动词的词干＋です
* 译文：（您）正在……；（您）要……

::: info :books: 例句

1. <grammer-content sentence='[遠藤/えんどう][先生/せんせい]！[今/いま]、お[帰/かえ]り**です**か。' trans='远藤老师！您现在要回去吗？' />
2. <grammer-content sentence='お[客様/きゃくさま]、[何/なに]かお[探/さが]し**です**か。' trans='客人，您在找什么吗？' />
3. <grammer-content sentence='[課長/かちょう]、[山田/やまだ][様/さま]がお[待/ま]ち**です**。' trans='科长，山田先生在等您。' />
4. <grammer-content sentence='[佐藤/さとう][部長/ぶちょう]は[明日/あした]からご[旅行/りょこう]**で**、3[週間/しゅうかん]いらっしゃらないそう**です**。' trans='听说佐藤部长从明天开始要去旅行，三周不在。' />
5. <grammer-content sentence='シンポジウムには[遠藤/えんどう][先生/せんせい]もご[出席/しゅっせき]**でした**。' trans='研讨会上远藤老师也出席了。' />

:::
