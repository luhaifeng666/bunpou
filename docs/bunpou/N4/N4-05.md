# N₁はN₂ほど～ない＜比較＞ <Badge type="warning" text="N4" />

* 意义：表示经过比较后，认为N1在程度上不及N2。
* 接续：名词1＋は＋名词2＋ほど＋形容词/动词的否定形式
* 译文：……不如……；……没有……；……不比……

::: tip :bookmark: 説明

第一册第14课第2单元学过的表示比较的句式`N₁はN₂より～`多用于肯定的形式，而表示否定的意思时一般常用`N₁はN₂ほど～ない`这种句式，其中「ほど」表示比较的基准。

:::

::: info :books: 例句

1. <grammer-content sentence='[私/わたし]は[美穂/みほ]**ほど**うまく[話/はな]せないし、まだまだ……。' trans='我不如美穂说得那么好，还差得远呢……' />
2. <grammer-content sentence='[今年/ことし]の[冬/ふゆ]も[寒/さむ]いです。でも、[今年/ことし]は[去年/きょねん]**ほど**[寒/さむ]くありません。' trans='今年冬天也很冷。但是，今年没有去年那么冷。' />
3. <grammer-content sentence='[李/り]さんも[日本語/にほんご]が[上手/じょうず]だが、[王/おう]さん**ほど**[上手/じょうず]ではない。' trans='小李日语也很好，但没有小王那么好。' />
4. <grammer-content sentence='[基本的/きほんてき]に[男性/だんせい]は[女性/じょせい]**ほど**[家事/かじ]がうまくできないと[思/おも]われていた。' trans='一般认为男性不如女性擅长家务。' />

:::
