# Vてほしい／Vないでほしい＜对他人的希望＞ <Badge type="warning" text="N4" />

* 意义：`Vてほしい`表示说话人希望对方或他人为自己做某事。`Vないでほしい`表示说话人希望、要求对方或他人不做某事。
* 接续：动词て形＋ほしい／动词ない形＋で＋ほしい
* 译文：希望……／希望不要……

::: tip :bookmark: 説明

`Vてほしい`与`てもらいたい`一样，都表示强烈地希望对方进行该动作。动词的主体用格助词`に`表示。

`てほしい`和`ないでほしい`还可表示说话人希望某事或某现象发生或不发生，这时谓语动词均为非自主的自动词，动词所涉及的主体一般用格助词`が`表示。

:::
