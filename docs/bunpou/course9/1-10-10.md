# ～でしょう＜确认＞

* 意义：向对方确认或征得对方同意；
* 接续：
  * 简体句子 + でしょう；
  * 名词／Ⅱ 类形容词词干＋でしょう；
* 译文：...吧。

::: tip :bookmark: 说明

「でしょう」在`表示确认`的意思时，要读`升调`。一般`不用于与上级或长辈的对话中`;

「～んでしょう」是「～んです」的`推测形式`，表示说话人就已知信息向对方确认。

:::
