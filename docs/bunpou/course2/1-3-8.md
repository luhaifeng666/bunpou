# N1 や N2 など＜并列＞

* 意义：用于列举两个或两个以上的事物。`「など」`　有时可以省略。
* 译文： `......等等；......之类`

::: tip :bookmark: 与 と 在用法上的区别

该语法用于 `部分列举`, 而 [`と`](../../auxiliary/to.md) 用于完全列举。

:::
