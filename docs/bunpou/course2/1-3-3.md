# まで＜终点＞

* 意义：表示终点。
* 接续：名词 + まで。
* 译文： `到......;......为止`

::: tip :bookmark: 说明

与 [から](./1-3-2.md#から＜起点＞) 一样，除了可以表示 `时间的结束` 外，还可以接在 `处所、顺序` 的名词后面，用于表示 `处所、顺序` 的终点。

:::

3. <grammer-content id='1-3-3-2' sentence="[私/わたし]の[授業/じゅぎょう]は４[時/じ]５０[分/ぷん]**まで**です。" trans='我的课4点50分结束。' />

:::
